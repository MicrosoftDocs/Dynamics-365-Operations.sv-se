# <a name="set-up-an-appointment-for-a-load"></a>Skapa en tidpunkt för en beläggning

[!include[task guide banner](../../includes/task-guide-banner.md)]

I den här proceduren visas hur du ställer in och planerar en lastkajstidpunkt för en beläggning. Detta görs normalt av en transportkoordinator. Du kan använda den här proceduren i demonstrationsdataföretaget USMF.


## <a name="select-the-load"></a>Markera beläggningen
1. Gå till Transporthantering > Planering > Workbench för lastplanering.
2. Avmarkera kryssrutan Clear the Hide shipped and received.
3. Välj den beläggning som har statusen Shipped i listan.
4. Klicka på Transportation.
5. Klicka på Appointment scheduling.

## <a name="create-an-appointment"></a>Skapa en tidpunkt
1. Klicka på Ny.
2. Ange eller välj ett värde i fältet Appointment rule.
3. Klicka på Spara.
4. Klicka på Update status.
5. Klicka på Bekräfta.
6. Klicka på Spara.
7. Stäng sidan.

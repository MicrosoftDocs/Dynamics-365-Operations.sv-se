# <a name="set-up-main-account-categories"></a>Ställ in kategorier för huvudkonto

[!include[task guide banner](../../includes/task-guide-banner.md)]

Huvudkontokategorier används för standardrapporterna i ekonomisk rapportering och i Power BI. DU kan byta namn på men inte ta bort huvudkontokategorier som skapats som standard. Ytterligare kontokategorier kan skapas och användas för rapporterings- och analysändamål. I den här uppgiften används demonstrationsföretaget USMF.


## <a name="create-a-main-account-category"></a>Skapa en huvudkontokategori
1. Gå till huvudboken > kontoplan > Konton > Main konto kategorier.
2. Klicka på Ny.
3. Ange ett unikt namn i fältet Huvudkontokategori.
4. I fältet Beskrivning anger du en beskrivning av huvudkontokategorin.
5. I fältet Huvudkontotyp väljer du den huvudkontotyp som ska länkas till kategorin.

## <a name="link-main-accounts-to-account-category"></a>Länka huvudkonton till kontokategori
1. Klicka på Länka huvudkonton.
2. I listan väljer du de huvudkonton som ska tilldelas till huvudkontokategorin.
    * När du tilldelar huvudkonton till en huvudkontokategori läggs kontosaldona till när kategorin används för ekonomisk rapportering och analys.  
3. Markera eller avmarkera alternativet Länkad för att välja huvudkontona.
4. Klicka på OK.
5. Klicka på Ja.

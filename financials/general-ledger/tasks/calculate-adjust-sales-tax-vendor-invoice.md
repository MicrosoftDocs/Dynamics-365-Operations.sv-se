[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="21578-104">Om det ursprungliga källdokumentet visar andra momsbelopp än dem som beräknas kan du justera dessa belopp, innan du bokför dem.</span><span class="sxs-lookup"><span data-stu-id="21578-104">If the original source document displays different tax amounts as calculated, you can adjust those amounts before posting.</span></span> <span data-ttu-id="21578-105">I den här uppgiften används demonstrationsföretaget DEMF.</span><span class="sxs-lookup"><span data-stu-id="21578-105">This task uses the DEMF demo company.</span></span>

2. <span data-ttu-id="21578-107">Klicka på Ny.</span><span class="sxs-lookup"><span data-stu-id="21578-107">Click New.</span></span>
3. <span data-ttu-id="21578-108">Markera vald rad i listan.</span><span class="sxs-lookup"><span data-stu-id="21578-108">In the list, mark the selected row.</span></span>

6. <span data-ttu-id="21578-111">Klicka på Rader.</span><span class="sxs-lookup"><span data-stu-id="21578-111">Click Lines.</span></span>
7. <span data-ttu-id="21578-112">Markera vald rad i listan.</span><span class="sxs-lookup"><span data-stu-id="21578-112">In the list, mark the selected row.</span></span>
8. <span data-ttu-id="21578-113">Ange önskade värden i fältet Konto.</span><span class="sxs-lookup"><span data-stu-id="21578-113">In the Account field, specify the desired values.</span></span>

12. <span data-ttu-id="21578-117">Klicka på Moms.</span><span class="sxs-lookup"><span data-stu-id="21578-117">Click Sales tax.</span></span>

15. <span data-ttu-id="21578-120">Klicka på Spara.</span><span class="sxs-lookup"><span data-stu-id="21578-120">Click Save.</span></span>
16. <span data-ttu-id="21578-121">Klicka på Moms.</span><span class="sxs-lookup"><span data-stu-id="21578-121">Click Sales tax.</span></span>

20. <span data-ttu-id="21578-125">Klicka på Spara.</span><span class="sxs-lookup"><span data-stu-id="21578-125">Click Save.</span></span>

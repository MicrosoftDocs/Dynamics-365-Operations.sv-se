# <a name="manage-email-templates"></a>Hantera e-postmallar

[!include[task guide banner](../../includes/task-guide-banner.md)]

Du kan överföra information från organisationens databas till bokmärkena i ett nytt dokument och använda den i mallarna som hjälper dig att kommunicera på ett effektivt sätt med sökande och kandidater. Om du vill göra detta skapar du en mall som innehåller standardtext och några bokmärken där systemdata ska infogas. Du kan till exempel infoga adress- och kontaktinformation för en sökande i ett Microsoft Word-dokument som du kan använda när du kommunicerar med den sökande. Det demonstrationsdataföretag som används för att skapa den här proceduren är USMF.


## <a name="select-which-bookmarks-to-use-in-your-email-templates"></a>Välj vilka bokmärken som ska användas i e-postmallarna
1. Gå till Ansökningsbokmärken.
2. Hitta och markera önskad korrespondensåtgärd.
3. Klicka på Redigera.
4. Hitta och markera önskad post i listan.
    * Markera de fält som du vill kunna använda i en e-postmeddelandemall för den valda korrespondensåtgärden och flytta dem till bokmärkesfälten.  
5. Stäng sidan.

## <a name="create-an-email-template"></a>Skapa en e-postmall
1. Gå till Personal Rekrytering > Kommunikations > E-postmallar för ansökningar.
2. Klicka på Ny.
3. Välj "Intervju" i fältet Korrespondensåtgärd.
    * Välj den korrespondensåtgärd som innehåller bokmärkena som ska användas för den här typen av e-postkommunikationen.  
4. Skriv ett värde i fältet E-postmall.
5. Skriv ett värde i fältet Ämne.
6. Skriv ett värde i fältet Text.
7. Hitta och markera önskat bokmärkesfält.
8. Fortsätt skriva ditt e-postmeddelande och infoga bokmärkesfälten där de behövs.
    * Fortsätt skriva ditt e-postmeddelande för att infoga bokmärkesfälten där så önskas.  
9. Klicka på Spara.

# <a name="introduction-to-dynamics-365-for-operations"></a>Introduktion till Dynamics 365 för operationer

[!include[banner](includes/banner.md)]

Microsoft Dynamics 365 för operationer är Microsofts affärsprogram för företag. Det gör att snabbt fatta smartare beslut via ett intelligent användargränssnitt som ger åtkomst till information i realtid och data. Tack vare beprövade affärslogik att konstruera affärsprocesserna snabbare kan företag transformeringen och innnovate. Den ger företag möjlighet att växa i sin egen takt och arbeta nästan var som helst, när som helst, så att de kan skalas globalt så att de passar företagets behov verksamheten. 

Den här användarhandboken enklare Läs Dynamics 365 för operationer som passar ditt företag.

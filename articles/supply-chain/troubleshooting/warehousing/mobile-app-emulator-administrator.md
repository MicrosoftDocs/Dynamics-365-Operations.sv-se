# <a name="can-only-use-administrator-role-to-access-the-mobile-device-app-emulator"></a>Kan bara använda administratörsrollen för åtkomst till appemulatorn för mobilenhet

## <a name="symptoms"></a>Symtom

När du ställer in lagerställen i Supply Chain Management kan du inte använda någon roll förutom administratörsrollen för åtkomst till appemulatorn för mobila enheter.

## <a name="resolution"></a>Lösning

Appemulatorn för den mobila enheten är inställd på att fungera endast med administratörskontot. För alla test- och liveprocess-ändamål rekommenderar vi att du använder själva mobilappen för distributionslagerhantering.

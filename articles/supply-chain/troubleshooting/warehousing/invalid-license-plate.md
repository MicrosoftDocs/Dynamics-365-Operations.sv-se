# <a name="invalid-license-plate-error-when-scanning-license-plate-id-in-the-mobile-app"></a>Felet Ogiltigt ID-nummer vid skanning av ID-nummer i mobilappen

## <a name="symptoms"></a>Symtom

När du skannar ett ID-nummer i Warehouse Management-mobilappen kan du få följande felmeddelande:

> Ogiltigt ID-nummer.

## <a name="resolution"></a>Lösning

Kontrollera att det finns ett ID-nummer i tabellen med ID-nummer och att artiklarna och kvantiteterna på ID-numret är tillgängliga (dvs. de spärras inte).

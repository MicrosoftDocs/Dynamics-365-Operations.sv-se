# <a name="packing-station-doesnt-show-product-notes"></a>Packstationen visar inte produktanteckningar

KB-nummer: 4614615

## <a name="symptoms"></a>Symtom

Följesedlar visas inte i förpackningsformuläret när förpackningsanvisningarna läggs till som en bifogad fil i en produktmall eller en produktvariant.

## <a name="resolution"></a>Upplösning

Systemet beter sig som det är utformat.

Den aktuella logiken för att visa följesedlar i förpackningsformuläret kräver att anteckningarna associeras med leveranserna.

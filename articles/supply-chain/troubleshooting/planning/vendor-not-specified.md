# <a name="vendor-isnt-specified-when-planned-orders-are-firmed"></a>Leverantörsleverantör som inte anges när planerade order bekräftas

Felkod: SYS23532

## <a name="symptoms"></a>Symtom

När du har bekräftat planerade order visas följande felmeddelande:

> Ingen leverantör har angetts

## <a name="resolution"></a>Lösning

Om du vill ange en leverantör, följ dessa steg.

1. Öppna den planerade order som saknar en leverantör.
1. På snabbfliken **Planerad leverans** i fältet **leverantör** välj leverantör.

# <a name="resource-name-isnt-updated-after-changing-a-worker-name-in-the-address-book"></a>Resursnamn uppdateras inte efter ändring av en arbetares namn i adressboken

## <a name="symptoms"></a>Symtom

Om du ändrar ett arbetarnamn i den globala adressboken uppdateras inte det matchande resursnamnet i resursgruppmall

## <a name="resolution"></a>Lösning

Det matchande resursnamnet uppdateras för närvarande inte automatiskt. För att åtgärda problemet måste du uppdatera resursnamnet manuellt.

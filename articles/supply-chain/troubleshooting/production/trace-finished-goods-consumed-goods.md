# <a name="cant-trace-serial-numbers-of-finished-goods-against-those-of-consumed-goods"></a>Det går inte att spåra serienummer på färdiga varor mot förbrukade varor

## <a name="symptoms"></a>Symtom

Du kan inte spåra serienumren för färdiga varor mot serienummer för materialen som en produktionsorder använder för att tillverka dessa färdiga varor.

## <a name="workaround"></a>Lösning

Lösningen är att skapa produktionsorder för kvantiteten 1.

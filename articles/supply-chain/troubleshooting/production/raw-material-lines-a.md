# <a name="users-can-unpick-material-lines-for-products-that-have-been-reported-as-finished"></a>Användare kan ta bort materialrader för produkter som har rapporterats som färdiga

KB-nummer: 4614721

## <a name="symptoms"></a>Symtom

Systemet tillåter användare att ta bort råmaterialrader för produkter som har rapporterats som färdiga.

## <a name="resolution"></a>Upplösning

Systemet beter sig som det är utformat. Råvaruförbrukningen kan justeras tills produktionsordern når statusen *Avslutad*.

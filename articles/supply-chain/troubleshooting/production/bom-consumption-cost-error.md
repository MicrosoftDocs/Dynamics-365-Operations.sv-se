# <a name="cant-end-a-production-order-because-of-a-bom-consumptioncost-value-error"></a>Det går inte att avsluta en tillverkningsorder på grund av värdefel i strukturlistans consumptionCost

## <a name="symptoms"></a>Symtom

När du försöker avsluta en produktionsorder visas följande felmeddelande:

> Beräkning av strukturlistans consumptionCost värde måste vara negativt vid utleverans från lager.

## <a name="resolution"></a>Lösning

Det här problemet har åtgärdats i version 10.0.15.

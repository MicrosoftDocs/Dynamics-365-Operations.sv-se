# <a name="inventory-aging-report-discrepancies"></a>Avvikelser för lagerföråldringsrapport

## <a name="symptoms"></a>Symtom

**Åldersfördelningsrapport för lager** visar olika värden när den visas på olika lagringsdimensioner (t.ex. webbplats eller lagerställe).

## <a name="resolution"></a>Lösning

Mer information om rapportlogik finns i rapport över [exempel på lagerföråldringsrapport och logik](/dynamics365/supply-chain/cost-management/inventory-aging-report.md).

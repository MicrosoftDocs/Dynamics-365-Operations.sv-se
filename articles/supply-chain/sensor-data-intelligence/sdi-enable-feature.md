# <a name="turn-on-sensor-data-intelligence-for-your-system"></a>Aktivera Sensor Data Intelligence för ditt system

[!include [banner](../includes/banner.md)]
[!INCLUDE [preview-banner](../includes/preview-banner.md)]

Innan du kan använda Sensor Data Intelligence måste den aktiveras i ditt system.

1. Gå till **Systemadministration \> Arbetsytor \> Funktionshantering**. (Mer information finns i [Översikt för funktionshantering](../../fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview.md).)
1. På fliken **Alla**, använda fältet **Filter** för att söka efter funktionen som är namngiven *Sensor Data Intelligence*.
1. Om funktionen *Sensor Data Intelligence* är aktiverat på ditt system, välj det i listan och välj sedan **Inaktivera** för att inaktivera den. Du kan inte använda den här äldre versionen av funktionen tillsammans med den nya förhandsgranskningsversionen som du aktiverar i nästa steg.
1. Använd fliken **Filter**, använda fältet Filter för att söka efter funktionen som är namngiven *(Förhandsversion) Sensor Data Intelligence*.
1. Välj funktion i listan och välj sedan **Aktivera nu** för att aktivera den.

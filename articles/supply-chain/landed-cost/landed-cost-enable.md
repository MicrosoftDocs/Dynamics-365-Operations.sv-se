# <a name="turn-on-the-landed-cost-module-for-your-system"></a>Aktivera modulen Hemtagningskostnad för ditt system

[!include [banner](../includes/banner.md)]

Innan du kan använda modulen Hemtagningskostnad den aktiveras i ditt system. Administratörer kan använda inställningarna [funktionshantering](../../fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview.md) för att kontrollera funktionens status och aktivera den. I arbetsytan **utgiftshantering** anges den här funktionen på följande sätt:

- **Modul:** *Transporthantering*
- **Funktionsnamn:** *Hemtagningskostnad*

# <a name="planned-orders-simplified"></a>Förenklade planerade order

[!include [banner](../../includes/banner.md)]

I denna artikel beskrivs hur du kan visa, hantera och bekräfta planerade order på sidan **Förenklade planerade order**.

Sidan **Förenklade planerade order** är en förenklad version av sidan **Planerade order**. Det ger färre alternativ men går snabbare att arbeta med när allt du behöver göra är att granska, godkänna och bekräfta dina planerade order.

## <a name="turn-on-this-feature-for-your-system"></a>Aktivera funktionen i systemet

Om ditt system inte redan innehåller de funktioner som beskrivs i denna artikel går du till [funktionshantering](../../../fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview.md) och aktiverar funktionen *Förenklade planerade order*.

## <a name="view-manage-and-firm-planned-orders"></a>Visa, hantera och bekräfta planerade order

Sidan **Förenklade planerade order** har främst utformats för att bekräfta funktionerna. I rapporten visas alla planerade order som har genererats för den valda planen, på samma sätt som sidan **Planerade order** gör. Därför kan den användas som ett optimerat alternativ för att visa resultaten från en huvudplaneringskörning.

Om du vill öppna sidan **Förenklade planerade order**, gå till **Huvudplanering \> Huvudplanering \> Förenklade planerade order**. Du kan använda den här sidan för att utföra följande uppgifter:

- Visa alla planerade order för en specifik plan.
- Skapa nya planerade order.
- Radera en eller flera planerade order.
- Bekräfta en eller flera planerade order.
- Ändra den planerade orderstatusen för enstaka eller flera planerade order.
- Sömlöst öppna planerade orderdetaljer för en enskild planerad order för att visa fullständiga detaljer för ordern.
- Skapa anpassade vyer med hjälp av sparade vyer.

Andra scenarier, som delning, stöds av sidan **Planerade order**.

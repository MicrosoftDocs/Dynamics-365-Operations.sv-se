# <a name="master-plans-overview"></a><span data-ttu-id="38db4-103">Huvudplaner – översikt</span><span class="sxs-lookup"><span data-stu-id="38db4-103">Master plans overview</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="38db4-104">Använd olika huvudplaner för att hantera ditt företags dagliga verksamhet, simulera olika planeringsstrategier som du vill övervaka, och implementera en företagspolicy, till exempel en policy för intern prestanda eller kundvård.</span><span class="sxs-lookup"><span data-stu-id="38db4-104">Use various master plans to support your company's daily working operations, simulate different planning strategies that you want to monitor, and implement a company policy, such as a policy about internal performance or customer satisfaction.</span></span> 

<span data-ttu-id="38db4-105">Du kan konfigurera huvudplaner på sidan **Huvudplaner**.</span><span class="sxs-lookup"><span data-stu-id="38db4-105">You can configure master plans on the **Master plans** page.</span></span>

<span data-ttu-id="38db4-106">Det finns två typer av planer:</span><span class="sxs-lookup"><span data-stu-id="38db4-106">There are two types of plans:</span></span>
-   <span data-ttu-id="38db4-107">**Statisk plan** – Beräkningen av huvudplaneringen använder aktuella data för att generera en nettobehovsplan.</span><span class="sxs-lookup"><span data-stu-id="38db4-107">**Static plan** – The master planning calculation uses the current data to generate a net requirements plan.</span></span> <span data-ttu-id="38db4-108">Den här planen förändras inte förrän nästa gång du kör huvudplanering eller manuellt ändra planen.</span><span class="sxs-lookup"><span data-stu-id="38db4-108">This plan remains unchanged until the next time that you run master planning or manually change the plan.</span></span> <span data-ttu-id="38db4-109">Det är en verksamhetsplan som olika anställda, till exempel en inköpare eller produktionsplanerare, kan använda för att basera sina beslut på och utföra sina dagliga aktiviteter.</span><span class="sxs-lookup"><span data-stu-id="38db4-109">This is an operating plan that various company personnel, such as a purchaser or production planner, can use to base their decisions on and perform their daily activities.</span></span>
-   <span data-ttu-id="38db4-110">**Dynamisk plan** – Den här planen börjar med samma nettobehovsplan som genererades av huvudplaneringen.</span><span class="sxs-lookup"><span data-stu-id="38db4-110">**Dynamic plan** – This plan starts with the same net requirements plan that was generated by master planning.</span></span> <span data-ttu-id="38db4-111">Du kan dock uppdatera den dynamiska planen varje gång som huvuddata ändras.</span><span class="sxs-lookup"><span data-stu-id="38db4-111">However, you can update the dynamic plan every time that the master data changes.</span></span> <span data-ttu-id="38db4-112">Det kan till exempel vara när du skapar en ny försäljningsorder.</span><span class="sxs-lookup"><span data-stu-id="38db4-112">This could be when you create a new sales order, for example.</span></span> <span data-ttu-id="38db4-113">Det blir då lättare att övervaka ändringar av order och artikeltillgänglighet utan att ändra den statiska planen som andra använder i sina arbetsprocesser.</span><span class="sxs-lookup"><span data-stu-id="38db4-113">This enables you to monitor the changing order network and item availability without disturbing the static plan that others are using for their work processes.</span></span>

<span data-ttu-id="38db4-114">Ett företag kan välja att arbeta med bara en dynamisk plan eller det kan använda både en statisk och dynamisk plan.</span><span class="sxs-lookup"><span data-stu-id="38db4-114">A company may choose to work with just a dynamic plan or it may use both static and dynamic plans.</span></span> <span data-ttu-id="38db4-115">Du kan dessutom konfigurera en huvudplan så att den återspeglar en viss strategi eller fokuserar på ett problem.</span><span class="sxs-lookup"><span data-stu-id="38db4-115">In addition, you can configure any master plan to reflect a particular strategy or address an issue.</span></span> <span data-ttu-id="38db4-116">Exempel är som följer:</span><span class="sxs-lookup"><span data-stu-id="38db4-116">Examples are as follows:</span></span>
-   <span data-ttu-id="38db4-117">Ställa in högre lagernivåer för att förhindra brist.</span><span class="sxs-lookup"><span data-stu-id="38db4-117">Set higher inventory levels to guarantee against stockouts.</span></span>
-   <span data-ttu-id="38db4-118">Ställa in längre säkerhetsmarginaler för att skydda sig mot opålitliga leverantörer.</span><span class="sxs-lookup"><span data-stu-id="38db4-118">Set longer safety margins to protect against unreliable vendors.</span></span>

<span data-ttu-id="38db4-119">Du kan också ställa in den dynamiska planen så att den uppdateras med den nya behovsplanen varje gång du kör huvudplanering.</span><span class="sxs-lookup"><span data-stu-id="38db4-119">You can also set up the starting dynamic plan so that it is updated with the new requirements plan every time that you run master planning.</span></span> <span data-ttu-id="38db4-120">Du kan ange inställningarna på sidan **Huvudplaneringsparametrar**.</span><span class="sxs-lookup"><span data-stu-id="38db4-120">You can specify these settings on the **Master planning parameters** page.</span></span>

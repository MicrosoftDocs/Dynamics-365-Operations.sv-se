# <a name="copy-a-formula"></a>Kopiera en formel

[!include [task guide banner](../../includes/task-guide-banner.md)]

Den här proceduren fokuserar på att skapa en formel som innehåller samma ingredienser som en befintlig formel, men med mindre skillnader. Om du vill skapa formelraderna kan du använda kopieringsfunktionen för att kopiera en befintlig formel som har innehåller de flesta av de ingredienser som du behöver. Du kan sedan göra nödvändiga ändringar på enskilda rader i den nya versionen. Genom att använda kopieringsfunktionen behöver du inte skapa flera formler som är nästan identiska. Det demonstrationsdataföretag som används för att skapa den här uppgiften är USP2.


## <a name="create-a-formula"></a>Skapa ett recept
1. Gå till Produktinformationshantering > Strukturlistor och formler > Formler.
2. Klicka på Ny.
3. Skriv ett värde i fältet Formel.
4. Skriv ett värde i fältet Namn.
    * Skriv ett beskrivande namn för formeln.  
5. Öppna sökningen genom att klicka på listruteknappen i fältet Plats.
6. Klicka på länken på den valda raden i listan.
7. Öppna sökningen genom att klicka på den nedrullningsbara knappen i fältet Artikelgrupp.
8. Hitta och markera önskad post i listan.
9. Klicka på länken på den valda raden i listan.
10. Klicka på Spara.

## <a name="copy-formula-lines"></a>Kopiera formelrader
1. Klicka på Formel i åtgärdsfönstret.
2. Klicka på Kopiera.
3. Öppna sökningen genom att klicka på den nedrullningsbara knappen i fältet Artikelnummer.
4. Klicka på länken på den valda raden i listan.
5. Öppna sökningen genom att klicka på den nedrullningsbara knappen i fältet Formelversion.
6. Klicka på länken på den valda raden i listan.
7. Klicka på OK.

## <a name="adjust-copied-formula-lines"></a>Justera kopierade formelrader
1. Markera vald rad i listan.
2. Klicka på Ta bort.
3. Klicka på Ja.

## <a name="approve-formula"></a>Godkänn recept
1. Klicka på Formel i åtgärdsfönstret.
2. Klicka på Godkänn recept.
3. Öppna sökningen genom att klicka på den nedrullningsbara knappen i fältet Godkänt av.
4. Klicka på länken på den valda raden i listan.
5. Klicka på Välj.
6. Klicka på OK.

# <a name="create-a-formula-by-copying-from-an-existing-formula"></a>Skapa en formel genom att kopiera från en befintlig formel

[!include [banner](../../includes/banner.md)]

I den här proceduren visas hur du skapar snabbt en ny formel genom att kopiera från en befintlig formel. Du kan använda demonstrationsdataföretagen USMF, USP2 eller USPI om du vill skapa den här proceduren.


## <a name="create-a-new-formula"></a>Skapa en ny formel
1. Gå till Formler.
2. Klicka på Ny.
3. Skriv ett värde i fältet Formel.
4. Skriv ett värde i fältet Namn.
5. Ange eller välj ett värde i fältet Plats.
6. Ange eller välj ett värde i fältet Artikelgrupp.
7. Klicka på Spara.

## <a name="copy-from-an-existing-formula"></a>Kopiera från en befintlig formel
1. Klicka på Kopiera.
2. Ange eller välj ett värde i fältet Artikelnummer.
3. I fältet Formelversion, ange eller välj ett värde.
4. Klicka på OK.

## <a name="revise-the-copied-formula"></a>Ändra den kopierade formeln
1. Hitta och markera önskad post i listan.
2. Klicka på Ta bort.
3. Klicka på Ja.

## <a name="approve-formula"></a>Godkänn recept
1. Klicka på Godkänn recept.
2. Ange eller välj ett värde i fältet Godkänd.
3. Klicka på OK.
4. Stäng sidan.

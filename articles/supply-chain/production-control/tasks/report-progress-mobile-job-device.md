# <a name="report-progress-on-a-mobile-job-device"></a>Rapportera framsteg på en mobil jobbenhet

[!include [task guide banner](../../includes/task-guide-banner.md)]

I den här proceduren visas hur du startar och rapporterar utvecklingen för ett produktionsjobb i registreringsformuläret för jobbenheten.



För att kunna köra den här proceduren måste du ha rollen systemadministratör eller maskinoperatör associerad med användarkontot.

1. Gå till Produktionskontroll > Tillverkningskörning > Jobbkortenhet.
2. Ange ID-brickan för en arbetare i fältet WorkerTextField. I demonstrationsdata för USMF, skriv ”123” för Christina Portra.
3. Klicka på Logga in.
4. Klicka på knappen Filter.
5. Markera eller avmarkera kryssrutan Tillämpa konfigureringsfilter. Om du anger ett filter kan du använda produktionsenhet 110 i USMF.
6. I fältet Produktionsenhet, välj resursgruppen för de produktionsjobb arbetaren kan arbeta med.
7. Klicka på länken på den valda raden i listan.
8. Klicka på OK.
9. Klicka på knappen Starta jobb.
10. Klicka på OK.
11. Klicka på knappen Rapportförlopp.
12. Klicka på OK.
13. Klicka på knappen Nästa jobb.
14. Klicka på de tilldelade för att se en översikt över alla knappar för produktionsjobb.
15. Stäng sidan.
16. Klicka på knappen Avbrott.
17. Hitta och markera önskad post i listan.
18. Klicka på OK.
19. Klicka på knappen Lämnar.
20. Välj för att logga ut.
21. Klicka på OK.
22. I fältet WorkerTextField, logga in igen. Du kan välja arbetare ”123” i USMF-demonstrationsdata.
23. Klicka på Logga in.
24. Klicka på Stoppa rast.
25. Klicka på knappen Aktivitet.
26. Klicka på Avbryt.
27. Klicka på knappen Lämnar.
28. Välj för att stämpla ut.
29. Klicka på OK.
30. Välj en orsak till varför du stämplar ut tidigt.

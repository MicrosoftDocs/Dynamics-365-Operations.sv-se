<span data-ttu-id="6d92d-113">En modell för produktkonfiguration kan formuleras som ett [begränsningsbaserad problem (CSP)](http://aima.cs.berkeley.edu/2nd-ed/newchap05.pdf).</span><span class="sxs-lookup"><span data-stu-id="6d92d-113">A product configuration model can be formulated as a [constraint satisfaction problem (CSP)](http://aima.cs.berkeley.edu/2nd-ed/newchap05.pdf).</span></span> <span data-ttu-id="6d92d-114">Microsoft Solver Foundation (MSF) erbjuder två typer av problemlösarstrategier för att lösa de CSPer som kan användas från produktkonfigurationsmodeller.</span><span class="sxs-lookup"><span data-stu-id="6d92d-114">Microsoft Solver Foundation (MSF) provides two types of solver strategies to solve the CSPs that can be used from product configuration models.</span></span> <span data-ttu-id="6d92d-115">Dessa problemlösarstrategier är beroende av [heuristik](https://techterms.com/definition/heuristic), som används för att bestämma den ordning som variabler av CSPer beaktas när problem löses.</span><span class="sxs-lookup"><span data-stu-id="6d92d-115">These solver strategies rely on [heuristics](https://techterms.com/definition/heuristic), which are used to determine the order that the variables of the CSPs are considered in when the problem is being solved.</span></span> <span data-ttu-id="6d92d-116">Heuristik kan påverka prestanda betydligt när ett problem eller en problemklass löses.</span><span class="sxs-lookup"><span data-stu-id="6d92d-116">Heuristics can significantly affect performance when a problem or class of problems is being solved.</span></span>

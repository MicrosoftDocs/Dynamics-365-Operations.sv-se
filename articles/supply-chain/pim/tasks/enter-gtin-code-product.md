# <a name="enter-the-gtin-code-for-a-product"></a>Ange GTIN-koden för en produkt

[!include [banner](../../includes/banner.md)]

GTIN-koder tilldelas av organisationen GS1. I det här exemplet ska du ange en kod för produktnumret M0001 i demonstrationdataföretaget USMF.

1. Klicka på Underhåll av frisläppt produkt.
2. Klicka på Frisläppta produkter.
3. Expandera faktaboxen för produktbilden.
4. Hitta och markera önskad post i listan.
5. Klicka på Hantera inventering i åtgärdsfönstret.
6. Klicka på GTIN-koder.
7. Klicka på Ny.
8. Markera vald rad i listan.
9. Ange eller välj ett värde i fältet Enhet.
10. Välj ett alternativ i fältet för GTIN-kod.
11. Ange ett värde i fältet GTIN.
12. Stäng sidan.

# <a name="create-product-model-components"></a>Skapa produktmodellkomponenter

[!include [task guide banner](../../includes/task-guide-banner.md)]

Begränsningsbaserade produktkonfigurationsmodeller använder komponenter som byggstenar för att kunna bygga upp produktmodellstrukturen. Komponenterna måste skapas innan produktmodellen kan byggas. Det demonstrationsdataföretag som används för att skapa den här proceduren är USMF.

1. Klicka på Definition av produktvariantmodell.
2. Klicka på Modeller för produktkonfiguration.
3. Klicka på Modell i åtgärdsfönstret.
4. Klicka på Komponenter.
5. Klicka på Ny.
6. Skriv ett värde i fältet Namn.
7. Ange ett värde i fältet Beskrivning.
8. Stäng sidan.

# <a name="create-a-bar-code-for-a-product"></a><span data-ttu-id="0f8bd-103">Skapa en streckkod för en produkt</span><span class="sxs-lookup"><span data-stu-id="0f8bd-103">Create a bar code for a product</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="0f8bd-104">I det här avsnittet visas hur du skapar en streckkod manuellt med hjälp av artikelnumret M0001 som exempel.</span><span class="sxs-lookup"><span data-stu-id="0f8bd-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="0f8bd-105">Det demonstrationsdataföretag som används för att skapa den här proceduren är USMF.</span><span class="sxs-lookup"><span data-stu-id="0f8bd-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="0f8bd-106">Välj **Underhåll av frisläppt produkt** på startsidan.</span><span class="sxs-lookup"><span data-stu-id="0f8bd-106">Select **Released product maintenance** on the homepage.</span></span>

6. <span data-ttu-id="0f8bd-112">Välj **Ny**.</span><span class="sxs-lookup"><span data-stu-id="0f8bd-112">Select **New**.</span></span>
7. <span data-ttu-id="0f8bd-113">Markera den markerade raden som skapas i listan nedan.</span><span class="sxs-lookup"><span data-stu-id="0f8bd-113">Mark the selected row that is created in the list below.</span></span>

10. <span data-ttu-id="0f8bd-116">I fältet **Streckkod** anger du ett värde.</span><span class="sxs-lookup"><span data-stu-id="0f8bd-116">In the **Bar code** field, type a value.</span></span>

13. <span data-ttu-id="0f8bd-119">Stäng sidan.</span><span class="sxs-lookup"><span data-stu-id="0f8bd-119">Close the page.</span></span>

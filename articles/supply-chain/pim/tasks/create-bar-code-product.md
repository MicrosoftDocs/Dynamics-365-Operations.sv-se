[!include [banner](../../includes/banner.md)]

1. <span data-ttu-id="c48ed-106">Välj **Underhåll av frisläppt produkt** på startsidan.</span><span class="sxs-lookup"><span data-stu-id="c48ed-106">Select **Released product maintenance** on the homepage.</span></span>

6. <span data-ttu-id="c48ed-112">Välj **Ny**.</span><span class="sxs-lookup"><span data-stu-id="c48ed-112">Select **New**.</span></span>
7. <span data-ttu-id="c48ed-113">Markera den markerade raden som skapas i listan nedan.</span><span class="sxs-lookup"><span data-stu-id="c48ed-113">Mark the selected row that is created in the list below.</span></span>

10. <span data-ttu-id="c48ed-116">I fältet **Streckkod** anger du ett värde.</span><span class="sxs-lookup"><span data-stu-id="c48ed-116">In the **Bar code** field, type a value.</span></span>

13. <span data-ttu-id="c48ed-119">Stäng sidan.</span><span class="sxs-lookup"><span data-stu-id="c48ed-119">Close the page.</span></span>

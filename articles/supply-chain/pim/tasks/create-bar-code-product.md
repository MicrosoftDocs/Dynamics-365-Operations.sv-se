# <a name="create-a-bar-code-for-a-product"></a><span data-ttu-id="93f19-103">Skapa en streckkod för en produkt</span><span class="sxs-lookup"><span data-stu-id="93f19-103">Create a bar code for a product</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="93f19-104">I det här avsnittet visas hur du skapar en streckkod manuellt med hjälp av artikelnumret M0001 som exempel.</span><span class="sxs-lookup"><span data-stu-id="93f19-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="93f19-105">Det demonstrationsdataföretag som används för att skapa den här proceduren är USMF.</span><span class="sxs-lookup"><span data-stu-id="93f19-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="93f19-106">Välj **Underhåll av frisläppt produkt** på startsidan.</span><span class="sxs-lookup"><span data-stu-id="93f19-106">Select **Released product maintenance** on the homepage.</span></span>
2. <span data-ttu-id="93f19-107">Gå till **produkter > frisläppta produkter** under avsnittet **länkar**.</span><span class="sxs-lookup"><span data-stu-id="93f19-107">Go to **Products > Released products** under the **Links** section.</span></span>
3. <span data-ttu-id="93f19-108">Hitta och markera önskad post i listan.</span><span class="sxs-lookup"><span data-stu-id="93f19-108">In the list, find and select the desired record.</span></span> <span data-ttu-id="93f19-109">För detta exempel, välj artikelnummer **M0001**.</span><span class="sxs-lookup"><span data-stu-id="93f19-109">For this example, select item number **M0001**.</span></span>
4. <span data-ttu-id="93f19-110">I åtgärdsfönstret klicka på **Hantera inventering**.</span><span class="sxs-lookup"><span data-stu-id="93f19-110">On the Action Pane, select **Manage inventory**.</span></span>
5. <span data-ttu-id="93f19-111">Välj **streckkoder**.</span><span class="sxs-lookup"><span data-stu-id="93f19-111">Select **Bar codes**.</span></span>
6. <span data-ttu-id="93f19-112">Välj **Ny**.</span><span class="sxs-lookup"><span data-stu-id="93f19-112">Select **New**.</span></span>
7. <span data-ttu-id="93f19-113">Markera den markerade raden som skapas i listan nedan.</span><span class="sxs-lookup"><span data-stu-id="93f19-113">Mark the selected row that is created in the list below.</span></span>
8. <span data-ttu-id="93f19-114">I fältet **Streckkodsinställningar** anger du eller väljer ett värde.</span><span class="sxs-lookup"><span data-stu-id="93f19-114">In the **Barcode setup** field, enter or select a value.</span></span>
9. <span data-ttu-id="93f19-115">I fältet **Barkod** anger du eller väljer ett värde.</span><span class="sxs-lookup"><span data-stu-id="93f19-115">In the **Bar code** field, enter or select a value.</span></span>
10. <span data-ttu-id="93f19-116">I fältet **Streckkod** anger du ett värde.</span><span class="sxs-lookup"><span data-stu-id="93f19-116">In the **Bar code** field, type a value.</span></span>  
11. <span data-ttu-id="93f19-117">Ange ett nummer i fältet **Kvantitet**.</span><span class="sxs-lookup"><span data-stu-id="93f19-117">In the **Quantity** field, enter a number.</span></span>
12. <span data-ttu-id="93f19-118">Välj **Spara**.</span><span class="sxs-lookup"><span data-stu-id="93f19-118">Select **Save**.</span></span>
13. <span data-ttu-id="93f19-119">Stäng sidan.</span><span class="sxs-lookup"><span data-stu-id="93f19-119">Close the page.</span></span>

# <a name="create-batch-attributes-for-a-product"></a>Skapa batchattribut för en produkt

[!include [banner](../../includes/banner.md)]

I den här proceduren visas hur du skapar ett batchattribut, tilldelar standardvärdeintervall och inkluderar attributet i en grupp. Det demonstrationsdataföretag som används för att skapa den här proceduren är företaget USP2.

1. Gå till Lagerhantering > Inställningar > Batch > Batchattribut.
2. Klicka på Ny.
3. Skriv ett värde i fältet Attribut.
4. Ange ett värde i fältet Beskrivning.
5. Välj "Del" i fältet Attributtyp.
    * I den här proceduren används deltypen för att aktivera decimalvärden. Du kan välja andra attributtyper. Om du väljer uppräkningstypen måste du ange värden i uppräkningslista, innan du kan ange ett värde i målfältet.  
6. Ange ett nummer i fältet Minimum.
7. Ange ett nummer i fältet Maximum.
8. Välj ett nummer i fältet Stegvis.
9. Skriv ett värde i fältet Mål.
10. Klicka på Spara.
11. Stäng sidan.
12. Gå till Lagerhantering > Inställningar > Batch > Batchattributgrupper.
13. Klicka på Ny.
14. Skriv ett värde i fältet Attributgrupp.
15. Ange ett värde i fältet Beskrivning.
16. Klicka på Spara.
17. Klicka på Gruppattribut.
18. Klicka på Ny.
19. Öppna sökningen genom att klicka på den nedrullningsbara knappen i fältet Attribut.
20. Hitta och markera önskad post i listan.
21. Klicka på länken på den valda raden i listan.
    * Ett attribut kan inkluderas i någon av grupperna.  
22. Klicka på Spara.
23. Stäng sidan.

# <a name="set-up-project-resource-information"></a>Ställ in information om projektresurs

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

I den här proceduren visas hur du skapar en projektresurs. I den här uppgiften används datauppsättningen USMF.

1. Klicka på Projekthantering och redovisning > Projektresurser > Resurslista.
2. Klicka på Ny resurs.
3. I fältet Arbetare, ange eller välj ett värde.
4. Ange eller välj ett värde i fältet Kalender.
5. Ange eller välj ett värde i fältet Periodkod.
6. Klicka på Skapa.
7. Klicka på Självkostnader.
8. Klicka på Självkostnad - timme.
9. Klicka på Ny.
10. Ange ett datum i fältet Giltighetsdatum.
11. Ange ett värde i fältet Cost price.
12. Klicka på Spara.
13. Stäng sidan.
14. Klicka på Försäljningspriser.
15. Klicka på Försäljningspris - timme.
16. Klicka på Ny.
17. Ange ett datum i fältet Giltighetsdatum.
18. Ange ett nummer i fältet Pris.
19. Välj ett alternativ i fältet Gäller för.
20. Klicka på Spara.
21. Stäng sidan.

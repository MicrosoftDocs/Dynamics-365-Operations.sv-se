# <a name="create-a-location-profile"></a><span data-ttu-id="99b50-103">Skapa en platsprofil</span><span class="sxs-lookup"><span data-stu-id="99b50-103">Create a location profile</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="99b50-104">Varje plats på lagerstället måste ha en platsprofil associerad som beskriver egenskaperna för platsen, till exempel om blandade artiklar tillåts för platsen.</span><span class="sxs-lookup"><span data-stu-id="99b50-104">Every location in the warehouse needs to have a location profile associated with it that describes the properties of the location, for example, whether the location allows mixed items.</span></span> <span data-ttu-id="99b50-105">I den här proceduren skapar vi en profil för en plats som inte kräver kontroll av registreringsskylt.</span><span class="sxs-lookup"><span data-stu-id="99b50-105">In this procedure we’ll create a profile for a location that doesn’t require license plate control.</span></span> <span data-ttu-id="99b50-106">Vi ska aktivera blandade artiklar och blandad lagerstatus, samt tillåta rullande inventering.</span><span class="sxs-lookup"><span data-stu-id="99b50-106">We’ll enable mixed items, and mixed inventory statuses, and allow cycle counting.</span></span> <span data-ttu-id="99b50-107">Du kan använda den här proceduren i demonstrationsdataföretaget USMF.</span><span class="sxs-lookup"><span data-stu-id="99b50-107">You can use this procedure in the USMF demo data company.</span></span>

1. <span data-ttu-id="99b50-108">Klicka på Ny.</span><span class="sxs-lookup"><span data-stu-id="99b50-108">Click New.</span></span>
2. <span data-ttu-id="99b50-109">Skriv ett värde i fältet Platsprofil-ID.</span><span class="sxs-lookup"><span data-stu-id="99b50-109">In the Location profile ID field, type a value.</span></span>
3. <span data-ttu-id="99b50-110">Skriv ett värde i fältet Namn.</span><span class="sxs-lookup"><span data-stu-id="99b50-110">In the Name field, type a value.</span></span>
4. <span data-ttu-id="99b50-111">Ange eller välj ett värde i fältet Platsformat.</span><span class="sxs-lookup"><span data-stu-id="99b50-111">In the Location format field, enter or select a value.</span></span>
5. <span data-ttu-id="99b50-112">Ange eller välj ett värde i fältet Platstyp.</span><span class="sxs-lookup"><span data-stu-id="99b50-112">In the Location type field, enter or select a value.</span></span>
6. <span data-ttu-id="99b50-113">Ange eller välj ett värde i fältet Dock management profile ID.</span><span class="sxs-lookup"><span data-stu-id="99b50-113">In the Dock management profile ID field, enter or select a value.</span></span>
7. <span data-ttu-id="99b50-114">Välj Ja i fältet Tillåt blandade artiklar.</span><span class="sxs-lookup"><span data-stu-id="99b50-114">Select Yes in the Allow mixed items field.</span></span>
8. <span data-ttu-id="99b50-115">Välj Ja i fältet Tillåt blandade lagerstatusar.</span><span class="sxs-lookup"><span data-stu-id="99b50-115">Select Yes in the Allow mixed  inventory statuses field.</span></span>
9. <span data-ttu-id="99b50-116">Välj Yes i fältet Allow cycle counting.</span><span class="sxs-lookup"><span data-stu-id="99b50-116">Select Yes in the Allow cycle counting field.</span></span>
10. <span data-ttu-id="99b50-117">Klicka på Spara.</span><span class="sxs-lookup"><span data-stu-id="99b50-117">Click Save.</span></span>
11. <span data-ttu-id="99b50-118">Gå till Lagerstyrning > Inställningar > Lagerställe > Platsprofiler.</span><span class="sxs-lookup"><span data-stu-id="99b50-118">Go to Warehouse management > Setup > Warehouse > Location profiles.</span></span>

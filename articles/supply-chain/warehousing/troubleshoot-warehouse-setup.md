# <a name="troubleshoot-warehouse-setup"></a>Felsök inställning för distributionslager

[!include [banner](../includes/banner.md)]

I det här avsnittet beskrivs hur du åtgärdar vanliga problem som kan uppstå när du ställer in lagerställen i Microsoft Dynamics 365 Supply Chain Management.

## <a name="i-cant-use-any-role-except-administrator-to-access-the-mobile-device-app-emulator"></a>Jag kan inte använda någon roll förutom administratör för att få åtkomst till appemulatorn för den mobila enheten.

### <a name="issue-description"></a>Problembeskrivning

Du kan inte använda någon roll förutom administratör för att få åtkomst till appemulatorn för den mobila enheten.

### <a name="issue-resolution"></a>Problemlösning

Appemulatorn för den mobila enheten är inställd på att fungera endast med administratörskontot. För alla test- och liveprocess-ändamål rekommenderar vi att du använder själva distributionslagerappen.

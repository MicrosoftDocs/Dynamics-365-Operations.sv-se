# <a name="view-current-wip-status-on-a-production-order"></a>Visa aktuell PIA-status för en produktionsorder

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

Den här proceduren visar hur du visar PIA-utdrag på en produktionsorder. Det demonstrationsdataföretag som används för att skapa den här proceduren är USMF. Den här proceduren är avsedd för kostnadscontrollern.

1. Klicka på Kostnadsredovisning.
2. Klicka på Produktionsorder.
3. Använd snabbfiltret för att filtrera på fältet Produktion med värdet ”p000153”.
4. Klicka på Hantera kostnader i åtgärdsfönstret.
5. Klicka på PIA-utdrag för produktion.
6. Ange datumet till "2012-12-01" i fältet Från datum.
7. Ange datumet till "2012-12-31" i fältet Till datum.

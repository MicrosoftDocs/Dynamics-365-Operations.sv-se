# <a name="view-cost-entries-for-a-cost-object"></a>Visa kostnadsposter för ett kostnadsobjekt

[!include [task guide banner](../../includes/task-guide-banner.md)]

I den här proceduren visas hur du visar kostnadsposter för ett kostnadsobjekt. Det demonstrationsdataföretag som används för att skapa den här proceduren är USMF. Den här proceduren är avsedd för kostnadscontrollern.

1. Klicka på Kostnadsredovisning.
2. Klicka på Frisläppta produkter.
3. Använd snabbfiltret för att söka efter poster. Filtrera till exempel i fältet Artikelnummer med värdet m0004.
4. Klicka på Hantera kostnader i åtgärdsfönstret.
5. Klicka på Kostnadsobjekt
6. Klicka på Kostnadsposter.
7. Använd snabbfiltret för att filtrera fältet Nummer med värdet p000031.
    * Om kostnadsposterna är tomma ställer du in fråndatumet på 31 januari 2012 och tilldatumet på 31 december 2012.

# <a name="set-up-a-transportation-tender"></a>Ställ in ett transportanbud

[!include [banner](../../includes/banner.md)]

I den här proceduren visas hur du skapar ett anbud för transport. Detta görs normalt av en transportkoordinator. Du kan använda den här proceduren i demonstrationsdataföretaget USMF.


## <a name="select-a-route"></a>Välj ett flöde
1. Gå till Transporthantering > Planering > Workbench för lastplanering.
2. Avmarkera kryssrutan Clear the Hide shipped and received.
3. Välj raden med last-ID 00006.
4. Klicka på Värdering och rutt.
5. Klicka på Rotes.

## <a name="create-the-transportation-tender"></a>Skapa ett transportanbud
1. Klicka på Transportation tänders.
2. Klicka på Ny.
3. Expandera avsnittet Allmänt.
4. Ange ett nummer i fältet Restituerad retas.
5. Klicka på Spara.
6. Klicka på Update status.
7. Klicka på Skicka.
8. Välj ett flöde.

# <a name="work-order-report"></a>Arbetsorderrapport


[!include [banner](../../includes/banner.md)]

[!include [banner](../../includes/preview-banner.md)]


Du kan skapa en arbetsorderrapport som visar detaljerad information om en arbetsorder. Du kan välja en eller flera arbetsorder som ska visas i rapporten.

1. Klicka på **Tillgångshantering** > **Rapporter** > **Arbetsorder** > **Arbetsorderrapport**.

2. Välj "Ja"/"Nej" på växlingsknapparna och fyll i fälten efter behov för att bestämma vilka detaljer som ska ingå i rapporten.  
  a. I avsnittet **Utskriftsinställningar** kan du välja om du vill inkludera bilagor från den relaterade typen av underhållsjobb i utskriften.

3. På snabbfliken **Poster som ska ingå** kan du filtrera innehållet i rapporten efter **arbetsorder.**

4. Om det behövs kan du skapa en rapport för generering av arbetsorderrapporter som ett batchjobb genom att fylla i fälten på snabbfliken **Kör i bakgrunden**.

5. Klicka på **OK** för att generera rapporten.

Nedan visas ett exempel på hur parametrar kan ställas in och den relaterade arbetsorderrapporten.

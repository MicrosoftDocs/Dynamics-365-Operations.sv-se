# <a name="work-order-report"></a>Arbetsorderrapport

[!include [banner](../../includes/banner.md)]


Du kan skapa en arbetsorderrapport som visar detaljerad information om en eller flera arbetsorder som du valt.

1. Välj **Tillgångshantering** > **Rapporter** > **Arbetsorder** > **Arbetsorderrapporter**.

2. Välj "Ja"/"Nej" på växlingsknapparna efter behov för att bestämma vilka detaljer som ska ingå i rapporten.  
    - I avsnittet **Utskriftsinställningar** kan du välja om du vill inkludera bilagor från den relaterade typen av underhållsjobb.

3. På snabbfliken **Poster som ska ingå** kan du filtrera innehållet i rapporten efter **arbetsorder.**

4. På snabbfliken **kör i bakgrunden** kan du ställa in generering av arbetsorderrapport som ett batchjobb, efter behov.

5. Klicka på **OK** för att generera rapporten.

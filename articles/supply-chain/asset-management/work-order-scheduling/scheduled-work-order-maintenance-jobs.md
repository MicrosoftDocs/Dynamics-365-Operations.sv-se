1. <span data-ttu-id="ac6f9-108">Klicka på **Tillgångshantering** > **Allmänt** > **Arbetsorder** > **Schemalagda arbetsorder för underhållsjobb**.</span><span class="sxs-lookup"><span data-stu-id="ac6f9-108">Click **Asset management** > **Common** > **Work orders** > **Scheduled work order maintenance jobs**.</span></span> <span data-ttu-id="ac6f9-109">En lista över alla arbetsorder som har ställts in på arbetsorderns livscykeltillstånd "schemalagd" eller "pågår" visas.</span><span class="sxs-lookup"><span data-stu-id="ac6f9-109">You see a list of all work orders set to work order lifecycle state "Scheduled" or "In progress".</span></span>

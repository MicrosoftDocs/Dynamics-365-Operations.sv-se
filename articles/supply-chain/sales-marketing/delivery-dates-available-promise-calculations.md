<span data-ttu-id="1da47-107">Orderlöfte beräknar de tidigaste transport- och inleveransdatumen och baseras på metoden för kontroll av leveransdatum och transportdagar.</span><span class="sxs-lookup"><span data-stu-id="1da47-107">Order promising calculates the earliest ship and receipt dates, and is based on the delivery date control method and transport days.</span></span> <span data-ttu-id="1da47-108">Du kan välja mellan fyra metoder för kontroll av leveransdatum:</span><span class="sxs-lookup"><span data-stu-id="1da47-108">You can select among four delivery date control methods:</span></span>

-   <span data-ttu-id="1da47-109">**Produktionstid för försäljning** – Produktionstiden för försäljning är tiden mellan det att försäljningsordern skapas och artiklarna levereras.</span><span class="sxs-lookup"><span data-stu-id="1da47-109">**Sales lead time** – Sales lead time is the time between creation of the sales order and shipment of the items.</span></span> <span data-ttu-id="1da47-110">Beräknat för leveransdatum baseras på ett standardantal dagar och beaktar inte lagertillgänglighet, känd efterfrågan eller planerad tillgång.</span><span class="sxs-lookup"><span data-stu-id="1da47-110">The delivery date calculation is based on a default number of days, and does not consider stock availability, known demand, or planned supply.</span></span>

## <a name="atp-calculations"></a><span data-ttu-id="1da47-116">ATP-beräkningar</span><span class="sxs-lookup"><span data-stu-id="1da47-116">ATP calculations</span></span>
<span data-ttu-id="1da47-117">ATP-kvantiteten beräknas med metoden "ackumulerad ATP med framförhållning".</span><span class="sxs-lookup"><span data-stu-id="1da47-117">The ATP quantity is calculated by using the “cumulative ATP with look-ahead” method.</span></span> <span data-ttu-id="1da47-118">Den största fördelen med ATP-beräkningsmetoden är att den förmår hantera fall där summan av utleveranser bland inleveranser överstiger den senaste inleveransen (till exempel när en kvantitet från en tidigare inleverans måste användas för att kunna uppfylla ett behov).</span><span class="sxs-lookup"><span data-stu-id="1da47-118">The main advantage of this ATP calculation method is that it can handle cases where the sum of issues among receipts is more than the latest receipt (for example, when a quantity from an earlier receipt must be used to meet a requirement).</span></span> <span data-ttu-id="1da47-119">Beräkningsmetoden "ackumulerad ATP med framförhållning" omfattar alla utleveranser tills den ackumulerade kvantiteten att inleverera är större än den ackumulerade kvantiteten att utleverera.</span><span class="sxs-lookup"><span data-stu-id="1da47-119">The “cumulative ATP with look-ahead” calculation method includes all issues until the cumulative quantity to receive exceeds the cumulative quantity to issue.</span></span> <span data-ttu-id="1da47-120">Därför utvärderar den här ATP-beräkningsmetoden om en del av kvantiteten från en tidigare period kan användas i en senare period.</span><span class="sxs-lookup"><span data-stu-id="1da47-120">Therefore, this ATP calculation method evaluates whether some of the quantity from an earlier period can be used in a later period.</span></span>  

<span data-ttu-id="1da47-121">ATP-kvantiteten är det obekräftade lagersaldot under den första perioden.</span><span class="sxs-lookup"><span data-stu-id="1da47-121">The ATP quantity is the uncommitted inventory balance in the first period.</span></span> <span data-ttu-id="1da47-122">Vanligtvis beräknas den för varje period som en inleverans schemaläggs.</span><span class="sxs-lookup"><span data-stu-id="1da47-122">Typically, it's calculated for each period in which a receipt is scheduled.</span></span> <span data-ttu-id="1da47-123">Programmet beräknar ATP-perioden i dagar och beräknar det aktuella datumet som det första datumet för ATP-kvantiteten.</span><span class="sxs-lookup"><span data-stu-id="1da47-123">The program calculates the ATP period in days and calculates the current date as the first date for the ATP quantity.</span></span> <span data-ttu-id="1da47-124">Under den första perioden omfattar ATP lagerbehållningen minus kundorder som har förfallit.</span><span class="sxs-lookup"><span data-stu-id="1da47-124">In the first period, ATP includes on-hand inventory minus customer orders that are due and overdue.</span></span>  

<span data-ttu-id="1da47-125">ATP beräknas med följande formel:</span><span class="sxs-lookup"><span data-stu-id="1da47-125">ATP is calculated by using the following formula:</span></span>  

<span data-ttu-id="1da47-126">ATP = ATP för föregående period + inleveranser för aktuell period – utleveranser för aktuell period – nettosaldo för varje framtida period till och med den period då summan av inleveranser för alla framtida perioder (inklusive den framtida perioden) överskrider summan av inleveranserna (inklusive den framtida perioden).</span><span class="sxs-lookup"><span data-stu-id="1da47-126">ATP = ATP for the previous period + Receipts for the current period – Issues for the current period – Net issue quantity for each future period until the period when the sum of receipts for all future periods, up to and including the future period, exceeds the sum of issues up to and including the future period.</span></span>  

<span data-ttu-id="1da47-127">Observera att beräkningen av ATP inte innehåller information om utgångsdatum och efter den tidsgräns för ATP som systemet förväntar sig när någon kvantitet kan utlovas.</span><span class="sxs-lookup"><span data-stu-id="1da47-127">Notice that the ATP calculation does not include information around expiry date and beyond the ATP time fence that the system expects when any quantity can be promised.</span></span>

<span data-ttu-id="1da47-128">När det inte finns några fler utleveranser eller inleveranser att ta hänsyn till är kvantiteten tillgängligt att lova för de följande datumen densamma som den senast beräknade kvantiteten tillgängligt att lova.</span><span class="sxs-lookup"><span data-stu-id="1da47-128">When there are no more issues or receipts to consider, the ATP quantity for the following dates is the same as the latest calculated ATP quantity.</span></span>  

<span data-ttu-id="1da47-129">Om inte alla dimensionerna som används för en artikel är givna när ATP-kontrollen slutförs, kan de fortfarande anges i utleveransen och inleveranserna.</span><span class="sxs-lookup"><span data-stu-id="1da47-129">If not all the dimensions that are used for an item are given when the ATP check is completed, they can still be specified on the issue and receipts.</span></span> <span data-ttu-id="1da47-130">I beräkningen av tillgängligt att lova måste i så fall inleveranser och utleveranser aggregeras till de befintliga dimensionerna, för att antalet inleverans- och utleveransrader som används i beräkningen av tillgängligt att lova ska minska.</span><span class="sxs-lookup"><span data-stu-id="1da47-130">In this case, in the ATP calculation, the receipts and issues must be aggregated to the existing dimensions to reduce the number of receipt and issue lines that are used in the ATP calculation.</span></span>  

<span data-ttu-id="1da47-131">ATP-kvantiteten som visas är alltid större än eller lika med 0 (noll).</span><span class="sxs-lookup"><span data-stu-id="1da47-131">The ATP quantity that is shown is always greater than or equal to 0 (zero).</span></span> <span data-ttu-id="1da47-132">Om beräkningen returnerar en negativ ATP-kvantitet (till exempel kvantitet som lovades tidigare överskrider den tillgängliga kvantiteten) anges kvantiteten anges automatiskt till 0.</span><span class="sxs-lookup"><span data-stu-id="1da47-132">If the calculation returns a negative ATP quantity (for example, if the quantity that was previously promised exceeds the available quantity), the quantity is automatically set to 0.</span></span>

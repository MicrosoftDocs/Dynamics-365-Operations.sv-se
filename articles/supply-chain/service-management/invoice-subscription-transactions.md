# <a name="invoice-subscription-transactions"></a>Fakturera abonnemangstransaktioner 

[!include [banner](../includes/banner.md)]



1.  Klicka på **Servicehantering** \> **Vanligt** \> **Serviceabonnemang** \> **Alla serviceabonnemang**.

2.  Välj ett serviceabonnemang och klicka på **Redigera** eller klicka på **Abonnemang** om du vill skapa ett abonnemang.

3.  Klicka på **analysera**\>**avgiftstransaktioner**.

4.  Välj avgiftstransaktionen -transaktionerna som du vill fakturera och klicka sedan på **Funktioner** \> **Fakturaförslag** för att öppna formuläret **Fakturaförslag**.

5.  Klicka på **Skapa faktura** och ange lämpliga parametrar i formuläret **Skapa faktura**

6.  Klicka på **OK** för att skapa fakturaförslagen i formuläret **Fakturaförslag**.

7.  Kontrollera innehållet på fakturan, klickar på **Bokför faktura** för att öppna formuläret **Bokför faktura** när du vill bokföra fakturan. Du kan också skriva ut fakturan från formuläret **Bokför faktura**.

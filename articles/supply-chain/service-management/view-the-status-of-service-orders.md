# <a name="view-the-status-of-service-orders"></a>Visa status för serviceorder 

[!include [banner](../includes/banner.md)]


Du kan använda formuläret **utförseltavla** för att visa status för serviceaktiviteter i serviceorder.

## <a name="more-information-about-service-orders"></a>Mer information om serviceorder

Formuläret ger även verktyg för att spåra flödet av serviceorder och innehåller mer information som kan hjälpa dig att utvärdera hur viktig en serviceorder är, t.ex. vilket serviceavtal och vilket servicenivåavtal som den är associerad med.

För att du snabbt ska kunna bedöma hur viktig en serviceorder är, stöder Gantt-diagrammet användningen av färger. Färger som används för att särskilja prioriteten för ett servicebesök. Som standard betyder rött hög prioritet, gult normal och grönt låg prioritet.


> [!TIP]
> <P>Du kan ändra färgerna på fliken <STRONG>utförsel</STRONG> i formuläret <STRONG>Servicehanteringsparametrar</STRONG>.</P>

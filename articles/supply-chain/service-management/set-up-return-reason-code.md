# <a name="set-up-return-reason-codes"></a>Ställa in returorsakskoder   

[!include [banner](../includes/banner.md)]


## <a name="set-up-return-reason-code-groups"></a>Ställa in grupper med returorsakskoder 

1.  Klicka på noden **Försäljning och marknadsföring** \> **Inställningar** \> **Försäljningsorder** \> **Returer** \> **Grupper av returorsakskoder**.

2.  Tryck på CTRL+N för att skapa en rad för en returorsakskodgrupp.

3.  I fältet **Kodgrupp för returorsaker** skriv ett ID för gruppen.

4.  Skriv en kort beskrivning av hur gruppen ska användas i fältet **beskrivning**.

## <a name="set-up-return-reason-codes"></a>Ställa in returorsakskoder 

Den här proceduren gör det enklare att ställa in orsakskoder som du kan använda för att ange varför en produkt har returnerats av kunden.

1.  Klicka på noden **Försäljning och marknadsföring** \> **Inställningar** \> **Försäljningsorder** \> **Returer** \> **Returorsakskoder**.

2.  Tryck på CTRL+N om du vill skapa en rad för en ny returorsakskod.

3.  Ange en text för att identifiera den här koden fältet **Returorsakskod**.

4.  Ange en text för att beskriva returorsakskoden i fältet **beskrivning**.

5.  Använd listrutan **Kodgrupp för returorsaker** för att lägga till koden i en returorsakskodgrupp.

# <a name="delete-a-return-order"></a>Ta bort en returorder 

[!include [banner](../includes/banner.md)]


Använd formuläret **Ta bort returorder** när du vill ta bort flera returorder.

1.  Klicka på **försäljning och marknadsföring**\>**Periodisk**\>**Rensa**\>**ta bort returorder i**.

2.  Klicka på **Välj** för att öppna formuläret **Försäljningsuppdatering**.

3.  Ange kriterier och klicka på **OK** när du vill återgå till formuläret **Ta bort returorder**. Returorder som har skapats i juridiska personen visas, baserat på de angivna kriterierna.

4.  Markera returorderraden som du vill ta bort och klicka sedan på **OK**.

5.  Klicka på **Ja** för att radera returneraordrarna.

## <a name="credit-subscription-transactions"></a><span data-ttu-id="f48de-104">Kreditera abonnemangstransaktioner</span><span class="sxs-lookup"><span data-stu-id="f48de-104">Credit subscription transactions</span></span>

1.  <span data-ttu-id="f48de-105">Klicka på **Servicehantering** \> **Vanligt** \> **Serviceabonnemang** \> **Alla serviceabonnemang**.</span><span class="sxs-lookup"><span data-stu-id="f48de-105">Click **Service management** \> **Common** \> **Service subscriptions** \> **All service subscriptions**.</span></span>

2.  <span data-ttu-id="f48de-106">Välj det abonnemang som är kopplat till abonnemangstransaktionen som du vill skapa en kreditfaktura för.</span><span class="sxs-lookup"><span data-stu-id="f48de-106">Select the subscription attached to the subscription transaction for which you want to create a credit note.</span></span>

3.  <span data-ttu-id="f48de-107">Välj fliken **Analys** och klicka sedan på knappen **Avgiftstransaktioner** på åtgärdsfönstret.</span><span class="sxs-lookup"><span data-stu-id="f48de-107">Select the **Analyze** tab, and then click the **Fee transactions** button on the Action Pane.</span></span>

4.  <span data-ttu-id="f48de-108">Från formuläret **Avgiftstrasnsaktioner**, välj det abonnemang som är kopplat till abonnemangstransaktionen som du vill skapa en kreditfaktura för.</span><span class="sxs-lookup"><span data-stu-id="f48de-108">From the **Fee transactions** form, select the transaction for which you want to create a credit note.</span></span>

5.  <span data-ttu-id="f48de-109">Klicka på **Funnktioner** \> **Välj för kreditfaktura**.</span><span class="sxs-lookup"><span data-stu-id="f48de-109">Click **Functions** \> **Select for credit note**.</span></span>

6.  <span data-ttu-id="f48de-110">Från formuläret **Välj för kreditfaktura** markerar du den transaktion du vill kreditera och klicka sedan på **OK**.</span><span class="sxs-lookup"><span data-stu-id="f48de-110">From the **Select for credit note** form, select the transaction that you want to credit and then click **OK**.</span></span>


> [!NOTE]
> <P><span data-ttu-id="f48de-111">När du skapar kreditfakturan, se till att du väljer <STRONG>kreditfakturor</STRONG>.</span><span class="sxs-lookup"><span data-stu-id="f48de-111">When you create the credit note, make sure that you select <STRONG>Credit notes</STRONG>.</span></span> <span data-ttu-id="f48de-112">Den finns i listan <STRONG>faktureringsmetod</STRONG> i dialogrutan <STRONG>Skapa faktura</STRONG>.</span><span class="sxs-lookup"><span data-stu-id="f48de-112">This is found in the <STRONG>Invoicing method</STRONG> list in the <STRONG>Create invoice</STRONG> dialog box.</span></span></P>

<span data-ttu-id="f48de-113">Om fältet **Återför periodisering vid kreditering** i formuläret **Servicehanteringsparametrar** anges till **Manuell** måste du återföra alla transaktioner med upplupna intäkter var för sig innan du skapar ett kreditfakturaförslag för transaktionen.</span><span class="sxs-lookup"><span data-stu-id="f48de-113">If the **Reverse accruals on crediting** field in the **Service management parameters** form is set to **Manual**, you have to reverse each accrued revenue transaction individually before you create a credit note proposal for the transaction.</span></span>

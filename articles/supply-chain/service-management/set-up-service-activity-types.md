# <a name="set-up-service-activity-types"></a>Ställa in serviceaktivitetstyper 

[!include [banner](../includes/banner.md)]


1.  Klicka på **Försäljning och marknadsföring**\>**inställningar**\>**aktiviteter**\>**aktivitetstyper**.
    
    –eller–
    
    Klicka på **Organisationsadministration**\>**inställningar**\>**aktiviteter**\>**aktivitetstyper**.

2.  Tryck på **Aktivitetstyper**, tryck på CTRL + N för att lägga till en ny rad för aktivitetstyp.

3.  Ange ett namn på aktivitetstypen i fältet **Typ**.

4.  Ange en beskrivning av aktivitetstypen i fältet **Beskrivning**.

5.  Stäng formuläret så att ändringarna sparas.

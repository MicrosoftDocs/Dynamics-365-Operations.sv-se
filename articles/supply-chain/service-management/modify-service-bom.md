# <a name="modify-a-service-bom"></a>Ändra en servicestrukturlista 

[!include [banner](../includes/banner.md)]


Du kan registrera historiken för ett element inom en servicestrukturlista. Varje gång, som du uppdaterar en strukturlisterad skapas en historikrad i fönstret **Historik**. Historikraden visar aktuell status för strukturlisteraden.

## <a name="update-a-service-bom-element"></a>Uppdatera ett strukturlisteelement

1.  Klicka på **servicehantering** \> **allmänt** \> **serviceavtal** \> **serviceavtal**.

2.  Klicka på **Redigera** för att öppna detaljformuläret **Serviceavtal**.

3.  På **åtgärdsfönstret**, klickar du på **Serviceobjekt** för att öppna formuläret **Serviceobjekt**.

4.  Markera objektet som du vill uppdatera en strukturlisterad för och klicka på **Designer**.

5.  I formuläret **Designer** välj strukturlisteraden du vill uppdatera och klicka sedan på **redigera strukturlisterad**.
    
    > [!NOTE]
    > <P>På fliken <STRONG>Inställning</STRONG> markera kryssrutan <STRONG>Redigera vid tillägg</STRONG> om du vill att formuläret <STRONG>Redigera strukturlisterad</STRONG> ska  öppnas när du drar en rad till servicestrukturlistan.</P>

6.  I fältet **Kvantitet** anger du kvantiteten.

7.  Markera kryssrutan **Skapa serviceorderrad** om du vill skapa en serviceorderrad för ersättningsartikeln, som sedan kan faktureras.

8.  Klicka på **OK** om du vill stänga formuläret.

## <a name="delete-a-service-bom-line"></a>Ta bort en rad från en servicestrukturlista

1.  Klicka på **servicehantering** \> **allmänt** \> **serviceavtal** \> **serviceavtal**.

2.  Klicka på **Redigera** för att öppna detaljformuläret **Serviceavtal**.

3.  På **åtgärdsfönstret**, klickar du på **Serviceobjekt** för att öppna formuläret **Serviceobjekt**.

4.  Markera objektet som du vill ta bort en strukturlisterad från och klicka sedan på **Designer**.

5.  I formuläret **Designer** välj strukturlisteraden du vill ta bort och klicka sedan på **Ta bort strukturlisterad**.

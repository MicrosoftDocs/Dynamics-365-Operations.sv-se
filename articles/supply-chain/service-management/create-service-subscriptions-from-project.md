# <a name="create-service-subscriptions-from-a-project"></a>Skapa serviceprenumerationer från ett projekt    

[!include [banner](../includes/banner.md)]


Du kan skapa ett serviceabonnemang från ett projekt i **projekthantering och redovisning**.

1.  Klicka på **Projekthantering och redovisning** \> **Vanligt** \> **Projekt** \> **Alla projekt**. Öppna projektet som du vill skapa ett abonnemang för.

2.  I formuläret **projekt** i **åtgärdsfönstret** på fliken **hantera** klickar du på **Service**\>**serviceabonnemang**.

3.  I formuläret **abonnemang** i fältet **serviceabonnemang** anger du ett unikt identifikationsnummer. Projektnumret anges automatiskt i abonnemanget för det valda projektet.

4.  Ange ytterligare information i formuläret.

Nu kan du skapa abonnemangstransaktioner för abonnemanget.


> [!NOTE]
> <P>Om du har skapat en nummersekvens för abonnemangsgruppen, genereras ett abonnemangs-ID automatiskt för abonnemanget när du anger abonnemangsgruppen.</P>

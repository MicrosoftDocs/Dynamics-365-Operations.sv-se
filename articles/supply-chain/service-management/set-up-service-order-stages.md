# <a name="set-up-service-order-stages"></a>Ställ in serviceorderfaser 

[!include [banner](../includes/banner.md)]


1.  Klicka på **servicehantering**\>**inställningar**\>**serviceorder**\>**Servicefaser**.

2.  Skapa en ny post genom att trycka på CTRL+N.

3.  I fälten **Servicefas** och **Beskrivning**, ange ett servicefas-ID och beskrivning.

4.  Välj lämpliga parametrar för fasen.

5.  Välj överordnad fas för den aktuella fasen eller lämna fältet **Överordnad** blankt om fasen finns i den ursprungliga fasen i inställningarna för fas.


> [!NOTE]
> <P>Du kan inte ändra fältet <STRONG>Överordnad</STRONG> efter att du sparat fasen. Istället kan du ta bort posten och skapa om den med ett annat urval i fältet <STRONG>Överordnad</STRONG>.</P>
> <P>Du kan bara skapa en fas med ett tomt fält för <STRONG>Överordnad</STRONG>. Det innebär att endast en ursprunglig fas tillåts.</P>

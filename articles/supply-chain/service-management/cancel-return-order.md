# <a name="cancel-a-return-order"></a>Annullera en returorder 

[!include [banner](../includes/banner.md)]


Du kan annullera en returorder i stället för att ta bort den, om du vill spara en post för den planerade returen.


> [!NOTE]
> <P>Det är enbart returorder med statusen <STRONG>Skapad</STRONG> som kan annulleras. Returorder med status <STRONG>Öppen</STRONG> eller <STRONG>Stängd</STRONG> kan inte annulleras.</P>


1.  Klicka på **Försäljning och marknadsföring** \> **Returorder** \> **Alla returorder** \> **Alla försäljningsorder**.
    
    –eller–
    
    Klicka på **Försäljning och marknadsföring** \> **Returorder** \> **Alla returorder** \> **Skapa returorder**.

2.  Välj returordern att annullera.

3.  Klciak på **Avbryt order**.

# <a name="set-up-template-to-represent-objects-at-customer-site"></a><span data-ttu-id="15d56-103">Ställ in mall för att representera objekt hos kunden</span><span class="sxs-lookup"><span data-stu-id="15d56-103">Set up template to represent objects at customer site</span></span> 

[!include [banner](../includes/banner.md)]


<span data-ttu-id="15d56-104">Använd den här proceduren för att ställa in en mall som ska representera objekt hos en kund.</span><span class="sxs-lookup"><span data-stu-id="15d56-104">Use this procedure to set up a template that represents objects at a customer site.</span></span>

1.  <span data-ttu-id="15d56-105">Klicka på **servicehantering** \> **allmänt** \> **serviceavtal** \> **serviceavtal**.</span><span class="sxs-lookup"><span data-stu-id="15d56-105">Click **Service management** \> **Common** \> **Service agreements** \> **Service agreements**.</span></span>

2.  <span data-ttu-id="15d56-106">Dubbelklicka på ett serviceavtal eller klicka på **Serviceavtal** om du vill skapa ett nytt serviceavtal.</span><span class="sxs-lookup"><span data-stu-id="15d56-106">Double-click a service agreement, or click **Service agreement** to create a new service agreement.</span></span>

3.  <span data-ttu-id="15d56-107">Klicka på fliken **Inställningar**.</span><span class="sxs-lookup"><span data-stu-id="15d56-107">Click the **Setup** tab.</span></span>

4.  <span data-ttu-id="15d56-108">Om du vill koppla en mallstrukturlista till serviceavtalet klickar du på **Serviceobjekt**.</span><span class="sxs-lookup"><span data-stu-id="15d56-108">Click **Service objects** to attach a template bill of materials (BOM) to the service agreement.</span></span>

5.  <span data-ttu-id="15d56-109">I formuläret **Serviceobjekt** markerar du serviceobjektet och klickar sedan på **funktioner**\>**Bifoga strukturlistemall**.</span><span class="sxs-lookup"><span data-stu-id="15d56-109">In the **Service objects** form, select a service object, and then click **Functions** \> **Attach template BOM**.</span></span>

6.  <span data-ttu-id="15d56-110">Välj en mallstrukturlista i fältet **Välj strukturlistemall** i formuläret **Mallstrukturlista** och sedan klicka på **OK**.</span><span class="sxs-lookup"><span data-stu-id="15d56-110">In the **Select template BOM** form, select a template BOM in the **Template BOM** field, and then click **OK**.</span></span>

7.  <span data-ttu-id="15d56-111">I formuläret **Serviceobjekt**, klicka på **Designer** om du vill redigera raderna i mallstrukturlistan så att strukturlistan visar de objekt du vill använda.</span><span class="sxs-lookup"><span data-stu-id="15d56-111">In the **Service objects** form, click **Designer** to modify the template BOM so that the BOM represents the objects that you want.</span></span>

8.  <span data-ttu-id="15d56-112">Stäng formulären så att ändringarna sparas.</span><span class="sxs-lookup"><span data-stu-id="15d56-112">Close the forms to save your changes.</span></span>

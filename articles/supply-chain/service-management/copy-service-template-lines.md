# <a name="copy-service-templates-lines"></a>Kopiera servicemallsrader 

[!include [banner](../includes/banner.md)]

Mallinformation kan kopieras antingen till ett serviceavtal eller en serviceorder.

## <a name="copy-service-template-lines-into-a-service-agreement"></a>Kopiera servicemallrader till ett serviceavtal

1. Klicka på **Serviceavtal**.
2. Dubbelklicka på ett serviceavtal för att öppna sidan för **serviceavtal**.
3. Klicka på **mallrader** för att öppna sidan **Kopiera servicemallrader**.
4. På sidan för **trädkontroll** utvidgar du trädfliken för att finna mallen med raderna som du vill kopiera till serviceavtalet. Välj därefter servicemallen.
5. I fliken för **rader** markerar du **kryssrutan** vid de servicemallrader som du vill kopiera.
6. Klicka på **OK**.
7. Stäng sidorna för att spara ändringarna.

Servicemallraderna kopieras nu till serviceavtalet tillsammans med det kopplade serviceobjektet och serviceuppgiftsrelationerna.

## <a name="copy-service-template-lines-into-a-service-order"></a>Kopiera servicemallrader till en serviceorder

1. Klicka på **Serviceorder**.
2. Dubbelklicka på en serviceorder för att öppna sidan för **serviceorder**.
3. Klicka på raderna Copy \> Copy template för att öppna sidan för att **kopiera servicemallrader**.
4. I fliken för **trädkontroll** utvidgar du trädet för att finna mallen med raderna som du vill kopiera till serviceordern. Välj därefter servicemallen.
5. I fliken för **rader** markerar du **kryssrutan** vid de servicemallrader som du vill kopiera.
6. Klicka på **OK**.
7. Stäng sidorna för att spara ändringarna.

Servicemallraderna kopieras nu till den nya serviceordern tillsammans med det kopplade serviceobjektet och serviceuppgiftsrelationerna.

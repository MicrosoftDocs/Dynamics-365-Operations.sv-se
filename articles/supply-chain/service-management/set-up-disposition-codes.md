# <a name="set-up-disposition-codes"></a><span data-ttu-id="433a7-103">Ställ in dispositionskoder</span><span class="sxs-lookup"><span data-stu-id="433a7-103">Set up disposition codes</span></span> 

[!include [banner](../includes/banner.md)]


<span data-ttu-id="433a7-104">Du kan du ställa in dispositionskoder för att ange hur man arbetar med en artikel som returneras av en kund.</span><span class="sxs-lookup"><span data-stu-id="433a7-104">You can set up disposition codes to specify how to process an item that is returned by a customer.</span></span> <span data-ttu-id="433a7-105">Skapa en dispositionskod med namnet **Reparera och returnera** om du vill ange att den returnerade artikeln ska repareras och sedan ska returneras till kunden, till exempel.</span><span class="sxs-lookup"><span data-stu-id="433a7-105">For example, create a disposition code named **Repair and return** to indicate that the returned item will be repaired and then returned to the customer.</span></span> <span data-ttu-id="433a7-106">Mer information om dispositionskoder som normalt används för artiklar som returneras av kunder hittar du i [Ange hur returnerade artiklar ska avyttras](specify-how-to-dispose-of-returned-items.md).</span><span class="sxs-lookup"><span data-stu-id="433a7-106">For more examples of disposition codes that are typically used for items that are returned by customers, see [Specify how to dispose of returned items](specify-how-to-dispose-of-returned-items.md).</span></span>

<span data-ttu-id="433a7-107">Du kan även ange en orsakskod för att förklara varför en artikel har returnerats.</span><span class="sxs-lookup"><span data-stu-id="433a7-107">You can also set up a reason code to help explain why an item was returned.</span></span> <span data-ttu-id="433a7-108">Mer information om orsakskoder finns [skapa returorsakskoder](set-up-return-reason-code.md).</span><span class="sxs-lookup"><span data-stu-id="433a7-108">For more information about reason codes, see [Set up return reason codes](set-up-return-reason-code.md).</span></span>

1.  <span data-ttu-id="433a7-109">Klicka på **Försäljning och marknadsföring** \> **Inställningar** \> **Försäljningsorder** \> **Returer** \> **Dispositionskoder**.</span><span class="sxs-lookup"><span data-stu-id="433a7-109">Go to **Sales and marketing** \> **Setup** \> **Sales orders** \> **Returns** \> **Disposition codes**.</span></span>

2.  <span data-ttu-id="433a7-110">Välj **Nytt** för att skapa en ny dispositionskod.</span><span class="sxs-lookup"><span data-stu-id="433a7-110">Select **New** to create a new disposition code.</span></span>

3.  <span data-ttu-id="433a7-111">Ange ett unikt beskrivande namn, välj en åtgärd och ange en beskrivning för dispositionskoden.</span><span class="sxs-lookup"><span data-stu-id="433a7-111">Enter a unique, descriptive name, select an action, and enter a description for the disposition code.</span></span>

4.  <span data-ttu-id="433a7-112">Om du vill koppla ett eller flera tillägg till den här dispositionskoden klickar du på knappen **Debiteringar** för att öppna formuläret **Ställ in avgifter**.</span><span class="sxs-lookup"><span data-stu-id="433a7-112">If you want to associate any customer charges with this disposition code, select the **Charges** button to open the **Set up charges** form.</span></span>

5.  <span data-ttu-id="433a7-113">Om du vill definiera externa koder som matchar företagets egna dispositionskoder klickar du på knappen **Externa koder** för att öppna formuläret **Externa koder**.</span><span class="sxs-lookup"><span data-stu-id="433a7-113">If you want to define any external codes to match with the company's own disposition codes, select the **External codes** button to open the **External codes** form.</span></span>

# <a name="whats-new-or-changed-in-dynamics-365-for-talent"></a>Nyheter och ändringar i Dynamics 365 for Talent 

[!include [banner](includes/banner.md)]

Den här sidan innehåller information om hur du hittar information om de senaste versionerna av Microsoft Dynamics Talent HR Core.

Varje vecka finns nya versioner för Talent. Dessa versioner visas i innehållsförteckningen på webbplatsen docs.microsoft.com ([docs.microsoft.com/dynamics365/unified-operations](../index.md)). Du kan antingen söka igenom innehållsförteckningen eller använda filterkontrollen ovanför innehållsförteckningen för att hitta de senaste ämnen som du letar efter. Avsnittsrubrikerna börjar med ”Nyheter och ändringar i Dynamics 365 for Talent…”.

Innehållsförteckningen kommer att uppdateras varje vecka, så kontrollera igen för att hitta alla nya funktioner och ändringar i de senaste versionerna av Talent.

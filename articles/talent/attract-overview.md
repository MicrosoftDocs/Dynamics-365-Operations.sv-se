# <a name="attract-overview"></a>Översikt över Attract

[!include[banner](includes/banner.md)]

Attract ger dig möjlighet att hantera processen med att anställa nya användare. Kandidater kan läggas till i öppna jobb och avancera steg för steg tills ett erbjudande förlängs eller ansökan avslås. I varje steg genomgår kandidaterna olika aktiviteter som till exempel utreda samtal, genomföra bedömningar eller schemalägga intervjuer. Efter att ha ansökt får de sökande ett mail där de uppmanas att logga in på Attract, där de kan följa jobbets status, ansökningen svara på alla aktiviteter som har tilldelats dem.

Attract innehåller följande funktioner:

### <a name="attract-dashboard"></a>Attract-instrumentpanel
Det är där som medlemmar i anställningsteamet kan se detaljerad information om de projekt som de deltar i. Det är också där som en rekryteringschef eller rekryterare kan lägga till ett nytt jobb i Attract, antingen genom att skapa ett individuellt eller genom att importera ett antal samtidigt från ett Excel-kalkylblad.

### <a name="jobs"></a>Jobb
På sidan **Jobb** kan information som exempelvis jobbeskrivningar och jobbens plats visas och redigeras. Sidan **Jobb** är platsen där rekryteringsteamet sammanställs och platsen där du kan lägga till eller byta ut teamets rekryterare och intervjuare. Nya kandidater kan också läggas till på sidan **Jobb**.

### <a name="candidates"></a>Kandidater
Kandidater kan läggas till individuellt i ett projekt eller överföras i större antal från ett Excel-kalkylblad. Alla kandidater läggs till automatiskt i den första fasen, Ansök, i samband med rekryteringsprocesserna för samtliga jobb. All ansökningsinformation för en kandidat kan visas och hanteras här. Kompletterande handlingar kan överföras och visas tillsammans med kontaktinformation som exempelvis e-postadresser, telefonnummer och länkar till deras sociala mediakonton. Om den sökande tillhandahållit en länk till sin LinkedIn-profil och LinkedIn-integreringen (som kräver en licens för LinkedIn-rekryterare) har ställts in, visas den sökandes profil.

### <a name="stages"></a>Stadier
Attract innehåller en fördefinierad rekryteringsprocess som innehåller fyra grundläggande steg: ansökan, utredning, intervju och erbjudande. Nya kandidater börjar i ansökningsfasen, där deras CV, personliga brev, portfölj och andra ansökningsdokument granskas. Härifrån kan en sökande vidarebefordras till utredningsfasen, där denne inbjuds att delta i ett utredande telefonsamtal eller en videointervju via Skype. Rekryteringschefen och rekryteraren kan ge feedback om kandidaten och flytta denne till nästa fas. Personliga intervjuer tidsplaneras sedan med tillgängliga medlemmar i rekryteringsteamet, och kan anpassas till att ske mellan fyra ögon eller panelintervjuer av olika längd. Efter intervjuerna registreras och granskas feedback från intervjuteamet. Utvalda kandidater skickas vidare till det sista steget där ett erbjudande skapas och förbereds för granskning och godkännande. 

### <a name="candidate-app"></a>Kandidatapp
Kandidater får ett e-postmeddelande så fort de har lagts till i ett jobb med en länk för att hämta programmet. Detta webbprogram ger de sökande information om jobbet och utgör deras verktyg för att kommunicera med rekryteringsteamet. De sökande kan även registrera sin tillgänglighet för intervjuer.

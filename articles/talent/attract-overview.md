# <a name="attract-features-and-capabilities"></a>Funktioner och möjligheter i Attract

[!include [banner](includes/banner.md)]

Attract ger dig möjlighet att hantera processen med att anställa nya användare. Kandidater kan läggas till i öppna jobb och avancera steg för steg tills antingen ett erbjudande förlängs eller ansökan avslås. Under varje steg genomgår kandidaterna olika aktiviteter som till exempel utreda samtal och intervjuer. Efter att ha ansökt får kandidaterna ett e-postmeddelande där de uppmanas att logga in på Attract, där de kan följa jobbets status och deras ansökning. De kan också svara på aktiviteter som har tilldelats dem.

Det här avsnittet beskriver de funktioner som Attract tillhandahåller.

## <a name="attract-dashboard"></a>Attract-instrumentpanel
På instrumentpanelen kan medlemmar i anställningsteamet visa information om de jobb som de deltar i. Instrumentbrädan är också där en rekryteringschef eller rekryterare kan lägga till nya jobb i Attract, antingen genom att skapa dem individuellt eller genom att importera flera samtidigt från ett Microsoft Excel-kalkylblad.

## <a name="jobs"></a>Jobb
På sidan **Jobb** kan information som exempelvis jobbeskrivningar och jobbens plats visas och redigeras. Sidan **Jobb** är även platsen där rekryteringsteamet sammanställs och platsen där du kan lägga till eller byta ut teamets rekryterare och intervjuare. Nya kandidater kan också läggas till på sidan **Jobb**.

## <a name="candidates"></a>Kandidater
Kandidater kan läggas till individuellt i ett projekt eller de kan överföras i större antal från ett Excel-kalkylblad. Alla kandidater läggs till automatiskt i fasen Ansök, som är den första fasen i rekryteringsprocesserna för alla jobb. Under den här fasen kan alla kandidaters ansökningsinformation visas och hanteras. Kompletterande handlingar kan överföras och visas tillsammans med kontaktinformation som exempelvis e-postadresser, telefonnummer och länkar till deras sociala mediakonton. Om en kandidater tillhandahöll en länk till hans eller hennes LinkedIn-profil och om LinkedIn-integrationen har ställts in kommer kandidatens profil att visas. (LinkedIn-integration kräver en licens för LinkedIn-rekryterare).

## <a name="stages"></a>Stadier
Attract innehåller en fördefinierad rekryteringsprocess som har fyra grundläggande steg: ansökan, utredning, intervju och erbjudande. Nya kandidater börjar i fasen Ansök, där deras CV, personliga brev, portfölj och andra ansökningsdokument granskas. Härifrån kan en kandidater vidarebefordras till fasen Utredning, där denne inbjuds att delta i ett utredande telefonsamtal eller en videointervju via Skype. Rekryteringschefen och rekryteraren kan ge feedback om kandidaten och flytta denne till nästa fas, som är fasen Intervju. Personliga intervjuer tidsplaneras sedan med tillgängliga medlemmar i anställningsteamet. Dessa intervjuer kan anpassas som enskilda intervjuer eller panelintervjuer av olika längd. Efter intervjuerna registreras och granskas feedback från intervjuteamet. Valda kandidater avancerar sedan till den sista fasen Erbjudande. Under den här fasen görs ett erbjudande och förbereds för granskning och godkännande av kandidaten.

## <a name="candidate-app"></a>Kandidatapp
Så fort de har lagts till i ett jobb får kandidaterna ett e-postmeddelande som inkluderar en länk för att hämta webbappen. Appen ger information om jobbet och används för att kommunicera med rekryteringsteamet. De sökande kan även använda det för att registrera sin tillgänglighet för intervjuer.

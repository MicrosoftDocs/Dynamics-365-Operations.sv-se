# <a name="get-support-for-talent"></a>Få support för Talent

[!include [banner](includes/banner.md)]

Det här avsnittet beskrivs hur du kan få hjälp med Talent: Core HR, Attract och Onboard.

<a name="contact-support"></a>Kontakta supporten
---------------

Följ instruktionerna nedan om du vill kontakta Microsoft om du har tekniska frågor om Talent.

### <a name="talent-support---core-hr"></a>Stöd för Talent - Core HR

Om du vill ha stöd för Talent, använd modulen i följande tabell. Resurserna visas i den ordning som du bör följa när du felsöker.

| **Uppgift**                                                | **Mer information**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
|---------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Fråga communityn                                      | Gå till sidan [Fråga communityn](https://community.dynamics.com/365/talent) för att få hjälp med dina frågor och använd alternativet ”Hjälp” för att söka efter avsnitt i Core HR.                                                                                                                                                                                                                                                                                                                                                                                  |
| Använd det molnbaserade supportverktyget                     | I [Lifecycle Services (LCS)](https://lcs.dynamics.com/), är panelen **Support** ett verktyg som hjälper dig att hantera supportincidenter. Om du vill skicka frågor direkt till Microsoft, gå till panelen **Support** i LCS-projektet eller använd **avsnittet supportärenden** i LCS och klicka på **Molnbaserad support**. Härifrån kan skicka du problem på två sätt. 1: När du skickar ett problem vika LCS på fliken **aktivt problem** väljer du ditt problem, och klickar sedan på **skicka till Microsoft**. 2. På fliken **skickade till Microsoft** klickar du på **skicka en ny incident** för att skicka nya problem. |
| Få hjälp med frågor om Talent-licenser         | Kontakta din partner eller Microsoft-återförsäljare.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| Begär nya funktionera och egenskaper i Talent | På huvudsidan i Talent i programmet, klicka på ikonen **?** och välj **idéer**, eller använd länken [idéer](https://powerusers.microsoft.com/t5/Ideas-for-Human-Resources/idb-p/HumanResources) för att visa, söka, rösta eller lägga till nya idéer.                                                                                                                                                                                                                                                                                                                      |

### <a name="talent-support--attract-and-onboard"></a>Talent support – Attract och Onboard

Om du vill ha stöd för Attract och Onboard, använd modulen i följande tabell. Resurserna visas i den ordning som du bör följa när du felsöker. Alla supportalternativ finns under ikonen **?** i området överst till höger i produkten.

| **Uppgift**                                                                | **Mer information**                                                                                                                                                                                                                                                                           |
|-------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Fråga communityn                                                      | Gå till sidan [Fråga communityn](https://community.dynamics.com/365/talent) om du vill få hjälp med frågor.                                                                                                                                                                                |
| Rapportera ett problem                                                        | Välj ikonen **?** i menyn och markera alternativet för att rapportera ett problem. Ange problemet i beskrivningen och inkludera en e-postadress om du vill följa upp problem som har rapporterats.                                                                                           |
| Få hjälp med frågor om Talent-licenser                         | Kontakta din partner eller Microsoft-återförsäljare.                                                                                                                                                                                                                                        |
| Begär nya funktionera och egenskaper i Attract och Onboard | Klicka på ikonen **?**. och välj **Idéer**, eller använd följande länkar för att visa, rösta eller lägga till nya idéer: [Attract](https://powerusers.microsoft.com/t5/Ideas-for-Attract/idb-p/Attract) eller [Onboard](https://powerusers.microsoft.com/t5/Ideas-for-Onboard/idb-p/Onboard). |

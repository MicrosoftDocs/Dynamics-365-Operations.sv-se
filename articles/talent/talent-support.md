# <a name="get-support-for-talent"></a>Få support för Talent

[!include [banner](includes/banner.md)]

Det här avsnittet beskrivs hur du kan få hjälp med Talent: Core HR, Attract och Onboard.

<a name="contact-support"></a>Kontakta supporten
---------------

Följ instruktionerna nedan om du vill kontakta Microsoft om du har tekniska frågor om Talent.

### <a name="talent-support"></a>Support för Talent 

Om du vill ha stöd för Talent, använd modulen i följande tabell. Resurserna visas i den ordning som du bör följa när du felsöker.

| **Uppgift**                                                | **Mer information**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
|---------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Fråga communityn                                      | Gå till sidan [Fråga communityn](https://community.dynamics.com/365/talent) för att få hjälp med dina frågor och använd alternativet ”Hjälp” för att söka efter avsnitt i Core HR.                                                                                                                                                                                                                                                                                                                                                                                  |
| Använd det molnbaserade supportverktyget                     | I [Lifecycle Services (LCS)](https://lcs.dynamics.com/), är panelen **Support** ett verktyg som hjälper dig att hantera supportincidenter. Om du vill skicka frågor direkt till Microsoft, gå till panelen **Support** i LCS-projektet eller använd **avsnittet supportärenden** i LCS och klicka på **Molnbaserad support**. Härifrån kan skicka du problem på två sätt. 1: När du skickar ett problem vika LCS på fliken **aktivt problem** väljer du ditt problem, och klickar sedan på **skicka till Microsoft**. 2. På fliken **skickade till Microsoft** klickar du på **skicka en ny incident** för att skicka nya problem.  |
| Support i program                                 | Välj ikonen **?** ikonen i menyn och välj sedan **Support (Core HR) rapporterar ett problem (Attract, Onboard)**. Problem rapporteras på fliken **aktiva ärenden** i LCS. Observera att om du har köpt den fristående versionen av Onboard kommer rapportera problem att skickas till Microsoft.  |
| Få hjälp med frågor om Talent-licenser         | Kontakta din partner eller Microsoft-återförsäljare.     |
| Begär nya funktionera och egenskaper i Talent | På huvudsidan i Talent i programmet, klicka på ikonen **?** och välj **idéer**, eller använd länken [idéer](https://experience.dynamics.com/ideas/) för att visa, söka, rösta eller lägga till nya idéer.          |

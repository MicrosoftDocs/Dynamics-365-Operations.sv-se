# <a name="whats-new-or-changed-in-dynamics-365-talent-december-17-2019"></a>Nyheter och ändringar i Dynamics 365 Talent (17 december 2019)

Den här artikeln innehåller en beskrivning av nya eller ändrade funktioner i Dynamics 365 Talent.

## <a name="changes-in-attract"></a>Ändringar i Attract

Den här versionen inkluderar felkorrigeringar för Dynamics 365 Talent: Attract.

## <a name="changes-in-onboard"></a>Ändringar i Onboard

Den här versionen inkluderar felkorrigeringar för Dynamics 365 Talent: Onboard.

## <a name="changes-in-core-hr"></a>Ändringar i Core HR

Ändringar som beskrivs i detta avsnitt gäller versionsnummer 8.1.2674. Siffror inom parenteser i vissa rubriker refererar till supportnummer i Microsoft Dynamics Lifecycle Services (LCS).

### <a name="print-performance-reviews"></a>Skriv ut resultatöversyner

Utskriftsprestanda granskningar är nu tillgängliga i alla miljöer. Använd arbetsytan **funktionshantering** om du vill aktivera den här funktionen. Mer information om hur du aktiverar Funktionshantering finns i [Översikt över funktionshantering](https://docs.microsoft.com/dynamics365/fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview). Mer information om hur du skriver ut prestanda granskningar [Prestationsgranskningar](https://docs.microsoft.com/dynamics365/talent/performance-management-overview#performance-reviews).

### <a name="retired-position-with-direct-reports-floats-in-org-diagram-as-an-open-position-385436"></a>Pensionerad position med direktrapporter "flyter" i organisationsdiagram som en öppen position (385436)

Den här veckans utgåva korrigerar hur tillbakadragna befattningar med direktrapporter visas i rapporthierarkin.
 
### <a name="system-unable-to-generate-calendar-days-381195"></a>Systemet kunde inte generera kalenderdagar (381195) 

Veckans ändringar korrigerar ett problem där kalenderdagar inte kunde genereras.

### <a name="current-balance-report-in-leave-pbi-doubles-employee-balance-385581"></a>Aktuell balansrapport i Leave PBI dubblerar medarbetarens saldo (385581)

Den här ändringen korrigerar den aktuella saldorapporten i det här scenariot.

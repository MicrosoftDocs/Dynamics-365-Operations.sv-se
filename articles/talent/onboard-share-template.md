## <a name="share-an-onboarding-guide-with-other-contributors"></a><span data-ttu-id="670eb-105">Dela en integrationsguiden med andra deltagare</span><span class="sxs-lookup"><span data-stu-id="670eb-105">Share an onboarding guide with other contributors</span></span>

    ![[<span data-ttu-id="670eb-108">Dela en integrationsguiden med andra deltagare</span><span class="sxs-lookup"><span data-stu-id="670eb-108">Sharing an onboarding guide with other contributors</span></span>](./media/onboard-share-guide.png)](./media/onboard-share-guide.png)

3. <span data-ttu-id="670eb-109">Under **dela med**, om mottagarna ska kunna revidera guiden, ställ in alternativet **kan dessa personer göra redigeringar?** till **på**.</span><span class="sxs-lookup"><span data-stu-id="670eb-109">Under **Share with**, if recipients should be able to revise the guide, set the **Can these people make edits?** option to **On**.</span></span> <span data-ttu-id="670eb-110">Om mottagarna ska meddelas via e-post som du har delat guiden med dem, anger du alternativet **skicka e-postmeddelande** till **på**.</span><span class="sxs-lookup"><span data-stu-id="670eb-110">If recipients should be notified by email that you've shared the guide with them, set the **Send email notification to recipients** option to **On**.</span></span>

    ![[<span data-ttu-id="670eb-111">Ange åtkomst- och e-postinställningar för en introduktionsguide</span><span class="sxs-lookup"><span data-stu-id="670eb-111">Setting access and email preferences for an onboarding guide</span></span>](./media/onboard-share-guide-with-contributors.png)](./media/onboard-share-guide-with-contributors.png)

4. <span data-ttu-id="670eb-112">Ange ett namn eller en e-postadress och välj önskad mottagare i den nedrullningsbara listan.</span><span class="sxs-lookup"><span data-stu-id="670eb-112">Enter a name or email address, and select the desired recipient in the drop-down list.</span></span>
5. <span data-ttu-id="670eb-113">På fliken **hantera åtkomst** kan du granska eller ändra vad mottagarna kan göra med guiden.</span><span class="sxs-lookup"><span data-stu-id="670eb-113">On the **Manage access** tab, you can review or change what recipients can do with the guide.</span></span> <span data-ttu-id="670eb-114">Om du vill ändra en mottagares åtkomst väljer du **Kan visa**, **Kan redigera** eller **Ta bort**, efter behov.</span><span class="sxs-lookup"><span data-stu-id="670eb-114">To change a recipient's access, in the drop-down list next to the recipient's name, select **Can View**, **Can edit**, or **Remove**, as appropriate.</span></span>
6. <span data-ttu-id="670eb-115">När du är klar med att lägga till eller ändra mottagare väljer du **uppdatera**.</span><span class="sxs-lookup"><span data-stu-id="670eb-115">When you've finished adding or modifying recipients, select **Update**.</span></span>

    <span data-ttu-id="670eb-119">[![Dela en mall med andra deltagare](./media/onboard-share-template.png)](./media/onboard-share-template.png)</span><span class="sxs-lookup"><span data-stu-id="670eb-119">[![Sharing a template with other contributors](./media/onboard-share-template.png)](./media/onboard-share-template.png)</span></span>

    <span data-ttu-id="670eb-122">[Ställa in åtkomst-och e-postinställningar för en mall](./media/onboard-share-template-access-and-email.png)](./media/onboard-share-template-access-and-email.png)</span><span class="sxs-lookup"><span data-stu-id="670eb-122">[!Setting access and email preferences for a template](./media/onboard-share-template-access-and-email.png)](./media/onboard-share-template-access-and-email.png)</span></span>

4. <span data-ttu-id="670eb-123">Ange ett namn eller en e-postadress och välj önskad mottagare i den nedrullningsbara listan.</span><span class="sxs-lookup"><span data-stu-id="670eb-123">Enter a name or email address, and select the desired recipient in the drop-down list.</span></span>
5. <span data-ttu-id="670eb-124">På fliken **hantera åtkomst** kan du granska eller ändra vad mottagarna kan göra med mallen.</span><span class="sxs-lookup"><span data-stu-id="670eb-124">On the **Manage access** tab, you can review or change what recipients can do with the template.</span></span> <span data-ttu-id="670eb-125">Om du vill ändra en mottagares åtkomst väljer du **Kan visa**, **Kan redigera** eller **Ta bort**, efter behov.</span><span class="sxs-lookup"><span data-stu-id="670eb-125">To change a recipient's access, in the drop-down list next to the recipient's name, select **Can View**, **Can edit**, or **Remove**, as appropriate.</span></span>
6. <span data-ttu-id="670eb-126">När du är klar med att lägga till eller ändra mottagare väljer du **uppdatera**.</span><span class="sxs-lookup"><span data-stu-id="670eb-126">When you've finished adding or modifying recipients, select **Update**.</span></span>

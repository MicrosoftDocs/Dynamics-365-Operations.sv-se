# <a name="post-jobs-to-linkedin-from-attract"></a>Publicera jobb på LinkedIn från Attract

[!include [banner](includes/banner.md)]

LinkedIn är världens största online professionella nätverk, vilket ger dig till gång till världens främsta talanger. Microsoft Dynamics 365 Talent: Attract hjälper dig att få den talang du behöver genom att låta dig publicera dina jobb direkt från Attract till LinkedIn.

Attract låter dig skicka begränsade listor till LinkedIn utan extra kostnad. Dessa listor är endast tillgängliga via LinkedIn-programpartners, t.ex. Attract. De visas inte på panelen **karriärer** på ditt företags LinkedIn-sida eftersom bara betalda listor visas där. De visas emellertid när potentiella kandidater visar alla tillgängliga jobb. Begränsade listor visas också på LinkedIn-jobbsökningar.

När du [skapar ett jobb](./creating-jobs-attract.md) i Attract behöver du bara klicka på en knapp för att placera jobbet framför tusentals potentiella jobbkandidater på LinkedIn.

I följande tabell visas de åtgärder som du kan utföra på LinkedIn, beroende på din användarroll.

| Roll | Åtgärder som du kan vidta |
|---|---|
| Admin | Publicera, publicera om och ta bort publiceringen |
| Anställande chef | Skrivskyddat |
| Rekryterare | Publicera, publicera om och ta bort publiceringen |
| Intervjuare | Ingen åtkomst |
| Skrivskydd | Skrivskyddat |

Mer information om användarroller i Attract finns i [Säkerhets- och rollhantering i Attract](./security-attract.md).

Om du är administratör och behöver mer information om hur du konfigurerar LinkedIn-integrationen med Attract, se [konfigurera integration med LinkedIn för Microsoft Dynamics 365 Talent - Attract](./attract-admin-linkedin.md).

Jobb som har publicerats på LinkedIn visas på den publicerade LinkedIn-webbplatsen. LinkedIn har inte testmiljö för att publicera jobb. Se därför till att du inte publicerar några testjobb av misstag.

## <a name="post-jobs-to-linkedin"></a>Publicera jobb till LinkedIn

1. I Attract öppnar du det jobb som du vill publicera i LinkedIn.
2. På fliken **publiceringar** väljer du knappen **bokför nu** som motsvarar LinkedIn.

    [![Attract publicerar jobb på LinkedIn](./media/attract-post-job-to-linkedin.png)](./media/attract-post-job-to-linkedin.png)

3. I dialogrutan **Skapa en webbplats för webbadress** som ska användas för tillfället väljer du ett alternativ under **kandidater kan ansöka med en**. Vi rekommenderar att du väljer **länk i Attract**.
4. Välj **Klar**.
5. I meddelandefältet **Skicka för publicering**, välj **Bekräfta**.

Efter att LinkedIn slutför publiceringen visar avsnittet **Publiceringar** av jobbet i Attract LinkedIn status som **publicerad**. Det kan ta upp till 48 timmar för jobbet att visas på LinkedIn.

När intresserade kandidater väljer **Visa** bredvid din lista, ser de fullständiga jobbdetaljerna tillsammans med din information om hur du ska göra.

Alla jobbinlägg som görs med Attract är begränsade listor. Mer information om begränsade listor på LinkedIn finns i [begränsade listor och Premium jobbplatser för jobbpaketering](https://www.linkedin.com/help/recruiter/answer/79049).

Om du har problem med att logga in eller publicera jobb på LinkedIn, se [Felsök integration med LinkedIn och Microsoft Dynamics 365 Talent - Attract](./attract-troubleshoot-linkedin.md).

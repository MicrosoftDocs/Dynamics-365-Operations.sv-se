# <a name="use-record-template-to-create-a-new-record"></a>Använd postmallar för att skapa en ny post

[!include [task guide banner](../../includes/task-guide-banner.md)]

Den här proceduren visar hur du använder en tidigare definierad postmall för att skapa en ny post. Innan du kan slutföra denna procedur måste du först slutföra proceduren "Skapa en postmall för en förenklad datainmatning".



I den här proceduren används företaget USMF.

1. Gå till anläggningstillgångar > anläggningstillgångar > Fasta tillgångar.
2. Klicka på Ny.
    * Du uppmanas att välja en mall. Välj den mall som motsvarar dina verksamhetsbehov.  
3. Hitta och markera önskad post i listan.
4. Klicka på OK.

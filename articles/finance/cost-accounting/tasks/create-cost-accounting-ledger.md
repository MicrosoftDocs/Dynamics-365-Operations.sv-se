# <a name="create-a-cost-accounting-ledger"></a><span data-ttu-id="b2931-103">Skapa en redovisning för kostnadsredovisning</span><span class="sxs-lookup"><span data-stu-id="b2931-103">Create a cost accounting ledger</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="b2931-104">En huvudbok för kostnadsredovisning utgör den övergripande rapporteringsenheten.</span><span class="sxs-lookup"><span data-stu-id="b2931-104">A cost accounting ledger represents the overall reporting unit.</span></span> <span data-ttu-id="b2931-105">Den definieras av en kostnadselementdimension, statistisk dimension, räkenskapskalender och valuta.</span><span class="sxs-lookup"><span data-stu-id="b2931-105">It is defined by a cost element dimension, statistical dimension, fiscal calendar, and currency.</span></span> <span data-ttu-id="b2931-106">Den är oberoende av begreppet juridiska personer.</span><span class="sxs-lookup"><span data-stu-id="b2931-106">It is agnostic to the concept of legal entities.</span></span> <span data-ttu-id="b2931-107">En juridisk person och dess data kan associeras med många huvudböcker för kostnadsredovisning.</span><span class="sxs-lookup"><span data-stu-id="b2931-107">A legal entity and its data can be associated with many cost accounting ledgers.</span></span> <span data-ttu-id="b2931-108">I den här inspelningen används demonstrationsföretaget USP2.</span><span class="sxs-lookup"><span data-stu-id="b2931-108">This recording uses the USP2 demo data company.</span></span>

1. <span data-ttu-id="b2931-109">Gå till Kostnadsredovisning > Redovisningsinställningar > Huvudböcker för kostnadsredovisning.</span><span class="sxs-lookup"><span data-stu-id="b2931-109">Go to Cost accounting > Ledger setup > Cost accounting ledgers.</span></span>
2. <span data-ttu-id="b2931-110">Klicka på Ny.</span><span class="sxs-lookup"><span data-stu-id="b2931-110">Click New.</span></span>
3. <span data-ttu-id="b2931-111">Skriv ett värde i fältet Namn.</span><span class="sxs-lookup"><span data-stu-id="b2931-111">In the Name field, type a value.</span></span>
4. <span data-ttu-id="b2931-112">Ange eller välj ett värde i fältet Dimension för kostnadselement.</span><span class="sxs-lookup"><span data-stu-id="b2931-112">In the Cost element dimension field, enter or select a value.</span></span>
5. <span data-ttu-id="b2931-113">Ange eller välj ett värde i fältet Räkenskapskalender.</span><span class="sxs-lookup"><span data-stu-id="b2931-113">In the Fiscal calendar field, enter or select a value.</span></span>
6. <span data-ttu-id="b2931-114">Ange eller välj ett värde i fältet Redovisningsvaluta.</span><span class="sxs-lookup"><span data-stu-id="b2931-114">In the Accounting currency field, enter or select a value.</span></span>
7. <span data-ttu-id="b2931-115">Ange eller välj ett värde i fältet Valutakurstyp.</span><span class="sxs-lookup"><span data-stu-id="b2931-115">In the Exchange rate type field, enter or select a value.</span></span>
8. <span data-ttu-id="b2931-116">Ange eller välj ett värde i fältet Statistikdimension.</span><span class="sxs-lookup"><span data-stu-id="b2931-116">In the Statistical dimension field, enter or select a value.</span></span>
9. <span data-ttu-id="b2931-117">Klicka på Spara.</span><span class="sxs-lookup"><span data-stu-id="b2931-117">Click Save.</span></span>

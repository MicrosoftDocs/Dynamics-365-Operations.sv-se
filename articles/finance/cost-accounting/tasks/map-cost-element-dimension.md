# <a name="map-a-cost-element-dimension"></a>Mappa dimension för kostnadselement

[!include [task guide banner](../../includes/task-guide-banner.md)]

En kostnadscontroller kan använda den här proceduren för att mappa en kostnadselementdimension till en kostnadselementdimension i den juridiska personen MXMF. I den här inspelningen används demonstrationsföretaget USP2.

1. Gå till Cost accounting > Dimensions > Cost element dimensions.
2. Hitta och markera önskad post i listan.
    * Välj Kostnadselement för det här exemplet.  
3. Klicka på Dimensionsmappningar.
4. Klicka på Konfigurera mappningar från den här dimensionen.
5. Klicka på Ny.
6. Ange eller välj ett värde i fältet Till dimension.
    * Välj kostnadselement för MXMF för det här exemplet.  
7. Klicka på Ny.
8. Markera vald rad i listan.
9. Ange eller välj ett värde i fältet Från dimensionsmedlem.
    * Välj dimensionsmedlem 606400 Telephone och Fax Expense.  
10. Ange eller välj ett värde i fältet Till dimensionsmedlem.
    * Välj dimensionsmedlem 6001004 Telefono.  
11. Klicka på Spara.

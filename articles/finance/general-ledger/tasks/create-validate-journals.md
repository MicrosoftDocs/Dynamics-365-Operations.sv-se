# <a name="create-and-validate-journals"></a>Skapa och validera journaler

[!include [banner](../../includes/banner.md)]

Den här proceduren skapar och verifierar journaler och journalrader. Du kan testa den här proceduren med demonstrationsföretaget USMF.  

1. Gå till **Redovisning > Journalposter > Allmänna journaler**.
2. Klicka på **Ny**.
3. Öppna sökningen genom att klicka på den nedrullningsbara knappen i fältet **Namn**.
4. Hitta och markera önskad post i listan.
5. Klicka på **Rader**.
6. I fältet **Konto** anger du ett lämpligt konto baserat på kontotyp.
7. I fältet **Beskrivning** anger du ett värde.
8. Ange ett belopp för kontot som antingen **Debet** eller **Kredit**. 
9. I fältet **Motkonto** anger du ett lämpligt konto baserat på typen Motkonto.
10. Klicka på **Validera.**
11. Klicka på **Validera.**
12. Klicka på **Bokför**.
13. Klicka på **Verifikation**.

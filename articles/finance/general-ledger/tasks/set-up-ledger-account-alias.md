# <a name="set-up-a-ledger-account-alias"></a><span data-ttu-id="88c86-103">Ställ in alias för huvudbokskonto</span><span class="sxs-lookup"><span data-stu-id="88c86-103">Set up a ledger account alias</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="88c86-104">Den här proceduren visar hur du skapar ett kontoalias som innehåller en genväg för att ange ett kontonummer.</span><span class="sxs-lookup"><span data-stu-id="88c86-104">This procedure shows how to create an account alias that provides a shortcut for entering an account number.</span></span> <span data-ttu-id="88c86-105">I proceduren används demonstrationsföretag USMF.</span><span class="sxs-lookup"><span data-stu-id="88c86-105">This procedure uses demo data company USMF.</span></span>

1. <span data-ttu-id="88c86-106">Gå till Redovisning > Kontoplan > Konton > Alias för huvudbokskonto.</span><span class="sxs-lookup"><span data-stu-id="88c86-106">Go to General ledger > Chart of accounts > Accounts > Ledger account alias.</span></span>
2. <span data-ttu-id="88c86-107">Klicka på Ny.</span><span class="sxs-lookup"><span data-stu-id="88c86-107">Click New.</span></span>
3. <span data-ttu-id="88c86-108">Skriv ett värde i fältet Alias för huvudbokskonto.</span><span class="sxs-lookup"><span data-stu-id="88c86-108">In the Ledger account alias field, type a value.</span></span>
4. <span data-ttu-id="88c86-109">Välj strukturen som kontot och dimensionerna tillhör i fältet Kontostruktur.</span><span class="sxs-lookup"><span data-stu-id="88c86-109">In the Account structure field, select the structure the account and dimensions belong to.</span></span>
5. <span data-ttu-id="88c86-110">Öppna sökningen genom att klicka på den nedrullningsbara knappen i fältet Företag.</span><span class="sxs-lookup"><span data-stu-id="88c86-110">In the Company field, click the drop-down button to open the lookup.</span></span>
6. <span data-ttu-id="88c86-111">Leta upp och välj det företag som aliaset gäller för i listan.</span><span class="sxs-lookup"><span data-stu-id="88c86-111">In the list, find and select the company that the alias applies to.</span></span>
7. <span data-ttu-id="88c86-112">Klicka på länken på den valda raden i listan.</span><span class="sxs-lookup"><span data-stu-id="88c86-112">In the list, click the link in the selected row.</span></span>
8. <span data-ttu-id="88c86-113">Ange kontot och dimensionerna i fältet Aliasdefinition för huvudbokskonto.</span><span class="sxs-lookup"><span data-stu-id="88c86-113">In the Ledger account alias definition field, specify the account and dimensions.</span></span>
    * <span data-ttu-id="88c86-114">Kontot och dimensionerna fylls i när du använder genvägen.</span><span class="sxs-lookup"><span data-stu-id="88c86-114">The account and dimensions will be populated when using the shortcut.</span></span>  
9. <span data-ttu-id="88c86-115">Välj den dimension som ska ha fokus när aliaset används i fältet Initialt fokus.</span><span class="sxs-lookup"><span data-stu-id="88c86-115">In the Initial focus field, select the dimension that will have focus when the alias is used.</span></span>
    * <span data-ttu-id="88c86-116">När du har skrivit genvägen och kontot och dimensionerna fyllts i är det ursprungliga fokusfältet dit markören eller fokus flyttar.</span><span class="sxs-lookup"><span data-stu-id="88c86-116">After you type the shortcut, and the account and dimensions are populated, the Initial focus field is where the cursor or focus will move to.</span></span>

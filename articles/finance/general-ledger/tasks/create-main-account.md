[!include [banner](../../includes/banner.md)]

1. <span data-ttu-id="823cf-106">Gå till **Navigeringsfönster > Moduler > Redovisning > Kontoplan > Konton > Huvudkonton**.</span><span class="sxs-lookup"><span data-stu-id="823cf-106">Go to **Navigation pane > Modules > General ledger > Chart of accounts > Accounts > Main accounts**.</span></span>
2. <span data-ttu-id="823cf-107">Klicka på **Ny**.</span><span class="sxs-lookup"><span data-stu-id="823cf-107">Click **New**.</span></span>
3. <span data-ttu-id="823cf-108">I fältet **Huvudkonto** anger du ett värde.</span><span class="sxs-lookup"><span data-stu-id="823cf-108">In the **Main account** field, type a value.</span></span>
4. <span data-ttu-id="823cf-109">Skriv ett värde i fältet **Namn**.</span><span class="sxs-lookup"><span data-stu-id="823cf-109">In the **Name** field, type a value.</span></span>
5. <span data-ttu-id="823cf-110">I fältet **Huvudkontotyp** väljer du den typ som bäst representerar kontonas saldo och plats i boksluten.</span><span class="sxs-lookup"><span data-stu-id="823cf-110">In the **Main account type** field, select the type that best represents the accounts balance and location on financial statements.</span></span>
6. <span data-ttu-id="823cf-111">I listan väljer du den kontokategori som huvudkontot tillhör.</span><span class="sxs-lookup"><span data-stu-id="823cf-111">In the list, select the account category the main account belongs to.</span></span> <span data-ttu-id="823cf-112">Kontokategorin används för ekonomiska standardrapporter och innehåll för Power BI-instrumentpaneler.</span><span class="sxs-lookup"><span data-stu-id="823cf-112">Account category is used for default financial reports and Power BI dashboard content.</span></span>  
7. <span data-ttu-id="823cf-113">Klicka på länken på den valda raden i listan.</span><span class="sxs-lookup"><span data-stu-id="823cf-113">In the list, click the link in the selected row.</span></span> <span data-ttu-id="823cf-114">Ändra förvalt debet- eller kreditsaldo.</span><span class="sxs-lookup"><span data-stu-id="823cf-114">Change the default debit or credit balance.</span></span>  
8. <span data-ttu-id="823cf-115">I fältet **Standardvaluta** väljer du ett värde i listan med valutor.</span><span class="sxs-lookup"><span data-stu-id="823cf-115">In the **Default currency** field, select a value from the list of currencies.</span></span>
9. <span data-ttu-id="823cf-116">Hitta och markera önskad post i listan.</span><span class="sxs-lookup"><span data-stu-id="823cf-116">In the list, find and select the desired record.</span></span>
10. <span data-ttu-id="823cf-117">Klicka på länken på den valda raden i listan.</span><span class="sxs-lookup"><span data-stu-id="823cf-117">In the list, click the link in the selected row.</span></span>
11. <span data-ttu-id="823cf-118">Växla utökningen av avsnittet **Juridisk person åsidosätter**.</span><span class="sxs-lookup"><span data-stu-id="823cf-118">Toggle the expansion of the **Legal entity overrides** section.</span></span>
12. <span data-ttu-id="823cf-119">Klicka på **Lägg till** för att välja en juridisk person.</span><span class="sxs-lookup"><span data-stu-id="823cf-119">Click **Add** to select a legal entity.</span></span>
13. <span data-ttu-id="823cf-120">Välj den juridiska personen i listan.</span><span class="sxs-lookup"><span data-stu-id="823cf-120">In the list, select the Legal entity.</span></span>
14. <span data-ttu-id="823cf-121">Klicka på **Lägg till**.</span><span class="sxs-lookup"><span data-stu-id="823cf-121">Click **Add**.</span></span>
15. <span data-ttu-id="823cf-122">Markera vald rad i listan.</span><span class="sxs-lookup"><span data-stu-id="823cf-122">In the list, mark the selected row.</span></span>
16. <span data-ttu-id="823cf-123">Markera eller avmarkera kryssrutan **Uppskjutet**.</span><span class="sxs-lookup"><span data-stu-id="823cf-123">Check or uncheck the **Suspended** checkbox.</span></span>
17. <span data-ttu-id="823cf-124">Expandera avsnittet **Ekonomisk rapportering**.</span><span class="sxs-lookup"><span data-stu-id="823cf-124">Expand the **Financial reporting** section.</span></span>
18. <span data-ttu-id="823cf-125">Öppna sökningen genom att klicka på den nedrullningsbara knappen i fältet **Valutakurstyp**.</span><span class="sxs-lookup"><span data-stu-id="823cf-125">In the **Exchange rate type** field, click the drop-down button to open the lookup.</span></span>
19. <span data-ttu-id="823cf-126">I listan väljer du **valutakurstypen för kontot**.</span><span class="sxs-lookup"><span data-stu-id="823cf-126">In the list, select the **Exchange rate type for the account**.</span></span>
20. <span data-ttu-id="823cf-127">Klicka på länken på den valda raden i listan.</span><span class="sxs-lookup"><span data-stu-id="823cf-127">In the list, click the link in the selected row.</span></span>
21. <span data-ttu-id="823cf-128">Välj metoden för att beräkna valutakurser för kontot i fältet **Valutaregistreringstyp**.</span><span class="sxs-lookup"><span data-stu-id="823cf-128">In the **Currency translation type** field, select the method for calculating exchange rates for the account.</span></span>
22. <span data-ttu-id="823cf-129">Stäng sidan.</span><span class="sxs-lookup"><span data-stu-id="823cf-129">Close the page.</span></span>

# <a name="copy-a-billing-schedule"></a>Kopiera ett faktureringsschema

[!include [banner](../includes/banner.md)]

På sidan **Kopiera tidsplanering** kan du skapa ett nytt faktureringsschema från ett befintligt faktureringsschema. Det nya faktureringsschemat har samma inställningar som det ursprungliga faktureringsschemat. Du kan också använda sidan om du vill kopiera artiklar från ett faktureringsschema till ett annat.

## <a name="copy-a-billing-schedule"></a>Kopiera ett faktureringsschema

För att kopiera faktureringstidsplan, följ dessa steg.

1. På listsidan **Alla faktureringsscheman** eller **Aktiva faktureringsscheman**, välj befintligt faktureringsschema.
2. På fliken **Faktureringsschema** välj **Kopiera tidsplan**.
3. I fältet **Kundkonto** markera ett kundkonto.
4. Uppdatera start- och slutdatum för faktureringen efter behov.
5. Uppdatera värdena i följande valfria fält efter behov: **Slutanvändarkonto**, **Partnerkonto**, **Kundreferens**, **Kundrekvisition** och **Decentraliseringsdatum**.
6. Välj **visa förhandsgranskning** och markera raderna i faktureringsplanen som du vill kopiera.
7. Välj **Kopiera**.

När kopieringsprocessen är klar visas det nya faktureringsschemat längst ned på sidan **Alla faktureringsscheman** eller listsidan **För aktiva faktureringsscheman**, och på sidan **Faktureringsscheman**.

## <a name="copy-items-from-one-billing-schedule-to-another"></a>Kopiera artiklar från en faktureringsplan till en annan

1. På listsidan **Alla faktureringsscheman** eller **Aktiva faktureringsscheman**, välj befintligt faktureringsschema.
2. På fliken **Faktureringsschema** välj **Kopiera tidsplan**.
3. I fältet **Kundkonto** markera ett kundkonto.
4. Välj ett faktureringsschemanummer i fältet **Faktureringsschemanummer**.
4. Uppdatera start- och slutdatum för fakturering.
5. Uppdatera värdena i följande fält efter behov: **Slutanvändarkonto**, **Partnerkonto**, **Kundreferens**, **Kundrekvisition** och **Decentraliseringsdatum**.
6. Välj **visa förhandsgranskning** och markera raderna i faktureringsplanen som du vill kopiera till ett befintligt faktureringsschema.
7. Välj **Kopiera**.

När kopieringsprocessen är klar går du igenom faktureringsplanen som du kopierade artiklarna till och ser till att artiklarna lagts till som du förväntat dig.

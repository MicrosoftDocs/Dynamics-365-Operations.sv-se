<span data-ttu-id="879ae-109">Du kan skicka en befintlig projektoffert för bekräftelse via e-post eller en tryckt kopia.</span><span class="sxs-lookup"><span data-stu-id="879ae-109">You can send an existing project quotation for confirmation through email or as a printed copy.</span></span>

6. <span data-ttu-id="879ae-116">På sidan **Inställningar för utskriftsmål** väljer du **OK**.</span><span class="sxs-lookup"><span data-stu-id="879ae-116">On the **Print destination settings** page, select **OK**.</span></span>

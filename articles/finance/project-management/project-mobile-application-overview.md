# <a name="project-mobile-applications-overview"></a>Project för mobila appar – Översikt

[!include [banner](../includes/banner.md)]

Tre mobilprogram som hör till projekttid är tillgängliga för Microsoft Dynamics 365 Project Timesheet, Registrering av projekttid och Mina tidrapporter/Tidrapporter för min granskning (optimerad för mobila enheter).

## <a name="project-timesheet-mobile-app"></a>Mobilappen Project Timesheet

Mobilappen Dynamics 365 Project Timesheet låter användare skicka och godkänna tidrapporter för projekt på sin mobila enhet. Den här mobilappen hämtar funktionerna för tidrapport och projekthantering och redovisningsområde för Dynamics 365 Finance. Den förbättrar användarens produktivitet och effektivitet och hjälper även med punktlig registrering och godkännande av projekttidrapporter.

## <a name="project-time-entry-workspace"></a>Arbetsyta för registrering av projekttid

Arbetsyta för registrering av projekttid låter användarna ange och spara tid för ett projekt. Det låter inte användarna skicka eller godkänna tidrapporter för projekt. Det kan bara användas för tidsregistrering. Denna mobila arbetsyta är avsedd att användas tillsammans med mobilappen Finance and Operations.

## <a name="my-timesheetstimesheets-for-my-review"></a>Mina tidrapporter/Tidrapporter för min granskning

Mina tidrapporter (optimerad för mobil) och tidrapporter för min granskning (optimerad för mobil) ger en lösning som låter medarbetare skapa och godkänna projekttidrapporter på en mobil enhet. Arbetare kan använda funktionen för Finance via URL:en för programmet. Denna lösning kräver inte mobilappen Finance and Operations.

# <a name="learning-catalog-for-dynamics-365-finance-developers"></a>Utbildningskatalog för utvecklare av Dynamics 365 Finance

Behöver du skriva kod för att integrera med andra datakällor, utöka kärnsystemfunktionaliteten eller bygga ett komplext program?

Följande katalog är ordnad efter grundläggande kunskaper i specifika domäner och från de flesta till de mest avancerade. Om det finns innehåll i flera format får vi meddela dig att du kan välja det utbildningsformat som bäst motsvarar dina behov.

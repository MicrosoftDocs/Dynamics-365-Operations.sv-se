# <a name="set-up-withholding-tax-component-groups-for-the-tds-tax-type"></a><span data-ttu-id="c7b34-103">Ställ in källskattekomponentgrupper för TDS-skattetypen</span><span class="sxs-lookup"><span data-stu-id="c7b34-103">Set up withholding tax component groups for the TDS tax type</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="c7b34-104">I det här avsnittet beskrivs hur du ställer in källskattekomponentgrupper, som **Hyra** och **Leverantör**, för skattetypen Skatteavdrag vid källa (TDS).</span><span class="sxs-lookup"><span data-stu-id="c7b34-104">This topic explains how to set up withholding tax component groups, such as **Rent** and **Contractor**, for the Tax Deducted at Source (TDS) tax type.</span></span>

1. <span data-ttu-id="c7b34-105">Gå till **Skatt \> Konfiguration \> Källskatt \> Källskattekomponentgrupper**.</span><span class="sxs-lookup"><span data-stu-id="c7b34-105">Go to **Tax \> Setup \> Withholding tax \> Withholding tax component groups**.</span></span>

    <span data-ttu-id="c7b34-106">[![Sidan Källskattekomponentgrupper](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span><span class="sxs-lookup"><span data-stu-id="c7b34-106">[![Withholding tax component groups page](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span></span>

2. <span data-ttu-id="c7b34-107">I fältet **Skattetyp** väljer du **TDS** för att konfigurera källskattekomponentgrupper för skattetypen TDS.</span><span class="sxs-lookup"><span data-stu-id="c7b34-107">In the **Tax type** field, select **TDS** to set up withholding tax component groups for the TDS tax type.</span></span>
3. <span data-ttu-id="c7b34-108">I åtgärdsfönstret, välj **Ny** för att skapa en rad.</span><span class="sxs-lookup"><span data-stu-id="c7b34-108">On the Action Pane, select **New** to create a line.</span></span>
4. <span data-ttu-id="c7b34-109">I fältet **Källskattekomponentgrupp** anger du ett namn på TDS-komponentgruppen.</span><span class="sxs-lookup"><span data-stu-id="c7b34-109">In the **Withholding tax component group** field, enter a name for the TDS component group.</span></span>
5. <span data-ttu-id="c7b34-110">Ange en beskrivning i fältet **beskrivning**.</span><span class="sxs-lookup"><span data-stu-id="c7b34-110">In the **Description** field, enter a description.</span></span>
6. <span data-ttu-id="c7b34-111">Under snabbfliken **Allmänt**, i fältet **Status**, väljer du bosättningsstatus för TDS-komponentgruppen.</span><span class="sxs-lookup"><span data-stu-id="c7b34-111">On the **General** FastTab, in the **Status** field, select the residential status of the TDS component group.</span></span> <span data-ttu-id="c7b34-112">Alternativen är **Invånare** och **Ej invånare**.</span><span class="sxs-lookup"><span data-stu-id="c7b34-112">The options are **Resident** and **Non-resident**.</span></span>

8. <span data-ttu-id="c7b34-114">Stäng sidan.</span><span class="sxs-lookup"><span data-stu-id="c7b34-114">Close the page.</span></span>

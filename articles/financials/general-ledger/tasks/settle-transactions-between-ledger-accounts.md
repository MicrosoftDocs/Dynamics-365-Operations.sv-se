# <a name="settle-transactions-between-ledger-accounts"></a>Kvitta transaktioner mellan redovisningskonton

[!include [task guide banner](../../includes/task-guide-banner.md)]

Proceduren visar hur det går till att kvitta transaktioner mellan huvudbokskonton och avbryta en redovisningskvittning. I proceduren används demonstrationsföretag USMF.


## <a name="settle-transaction-between-ledger-accounts"></a>Kvitta transaktion mellan redovisningskonton
1. Gå till Redovisning > Periodiska uppgifter > Redovisningskvittningar.
2. Leta reda på transaktionen i listan du vill kvitta.
    * Beloppsaldot måste vara noll.  
3. Klicka på Inkludera.
4. Klicka på Godkänn.

## <a name="cancel-a-ledger-settlement"></a>Avbryt en redovisningskvittning
1. Stäng sidan.
2. Gå till Redovisning > Förfrågningar och rapporter > Råbalans.
3. Klicka på Parametrar för att öppna dialogrutan.
4. Klicka på Uppdatera.
5. Leta reda på kontot i listan där den kvittade transaktionen finns.
6. Klicka på Alla transaktioner.
7. Använd ett filter för att enkelt hitta transaktionen i listan.
8. Klicka på Redovisningskvittningar.
9. Markera vald rad i listan.

# <a name="create-and-assign-advanced-rule-structures"></a>Skapa och tilldela avancerade regelstrukturer

[!include[task guide banner](../../includes/task-guide-banner.md)]

Den här uppgiften beskriver steg för steg hur du skapar och tilldelar en avancerad regelstruktur till en kontostruktur. Den här guiden använder demoföretaget USMF.


## <a name="create-an-advanced-rule-structure"></a>Skapa en avancerad regelstruktur
1. Gå till Redovisning > Kontoplan > Strukturerar > Avancerade regelstrukturer.
2. Klicka på Nytt om du vill öppna dialogrutan.
3. Ange ett namn som beskriver regelstrukturen i fältet Avancerad regelstruktur.
4. Ange ett värde som beskriver strukturen i fältet Beskrivning.
5. Klicka på OK.
6. Klicka på Lägg till segment.
7. Välj en ekonomisk dimension i listan med segment.
    * Till exempel Butik.  
8. Klicka på Lägg till segment.
9. I listan klickar du på länken för den avancerade regelstrukturen för att visa den.
10. Klicka på Aktivera.
11. Klicka på Aktivera.

## <a name="apply-an-advanced-rule-structure-to-an-account-structure"></a>Använd en avancerad regelstruktur för en kontostruktur
1. Stäng formuläret.
2. Stäng sidan.
3. Gå till Redovisning > Kontoplan > Strukturer > Konfigurera kontostrukturer.
4. I listan letar du upp och väljer den kontostruktur som du vill tillämpa den avancerade regeln på.
5. Klicka på namnet på kontostrukturen för att öppna den.
6. Klicka på Redigera.
    * Du kan också klicka på Avancerade regler så uppmanas du att placera kontostrukturen i utkastläge.  
7. Klicka på Avancerade regler.
8. Klicka på Nytt om du vill öppna dialogrutan.
9. Ange ett värde i fältet Avancerad regel.
10. Skriv ett värde i fältet Namn.
11. Klicka på Skapa.
12. Klicka på Lägg till nya kriterier.
13. Välj huvudkonto eller en ekonomisk dimension i fältet Där.
14. Välj ett alternativ i fältet Operator, t.ex. är mellan och inkluderar.
15. Ange ett värde i fältet Värde.
16. Skriv ett värde i fältet via.
17. Klicka på Lägg till för att öppna dialogrutan.
18. I listan letar du upp den avancerade regelstruktur som du vill använda när de villkor som du angett uppfylls.
19. Klicka på Lägg till.
20. Stäng sidan.
21. Klicka på Aktivera.
22. Klicka på Aktivera.

# <a name="dispose-of-a-fixed-asset-using-a-free-text-invoice"></a><span data-ttu-id="92394-103">Avyttra en anläggningstillgång med hjälp av en fritextfaktura</span><span class="sxs-lookup"><span data-stu-id="92394-103">Dispose of a fixed asset using a free text invoice</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="92394-104">Den här proceduren visar hur du avyttrar en anläggningstillgång med hjälp av formuläret Fritextfaktura.</span><span class="sxs-lookup"><span data-stu-id="92394-104">This procedure shows how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="92394-105">Gå till Kundreskontra > Fakturor > Alla fritextfakturor.</span><span class="sxs-lookup"><span data-stu-id="92394-105">Go to Accounts receivable > Invoices > All free text invoices.</span></span>
2. <span data-ttu-id="92394-106">Klicka på Ny.</span><span class="sxs-lookup"><span data-stu-id="92394-106">Click New.</span></span>
3. <span data-ttu-id="92394-107">I fältet Kundkonto, ange eller välj ett värde.</span><span class="sxs-lookup"><span data-stu-id="92394-107">In the Customer account field, enter or select a value.</span></span>
4. <span data-ttu-id="92394-108">Validera standardfakturadatum och redigera om det är tillämpligt.</span><span class="sxs-lookup"><span data-stu-id="92394-108">Validate the default Invoice date and edit if applicable.</span></span>
5. <span data-ttu-id="92394-109">Validera återstående standardhuvudfält, t.ex. Valuta och redigera om det är tillämpligt.</span><span class="sxs-lookup"><span data-stu-id="92394-109">Validate remaining default header fields, such as Currency and edit if applicable.</span></span>
6. <span data-ttu-id="92394-110">Ange en beskrivning av fakturaraden.</span><span class="sxs-lookup"><span data-stu-id="92394-110">Enter a Description into the invoice line.</span></span>
7. <span data-ttu-id="92394-111">Ange eller välj huvudkontot för fakturarraden.</span><span class="sxs-lookup"><span data-stu-id="92394-111">Enter or select the Main account for the invoice line.</span></span>
8. <span data-ttu-id="92394-112">Validera standardmomsgrupp och artikelmomsgrupp.</span><span class="sxs-lookup"><span data-stu-id="92394-112">Validate the default Sales tax group and Item sales tax group.</span></span>
9. <span data-ttu-id="92394-113">Ange Enhetspris eller Belopp för försäljning av anläggningstillgången.</span><span class="sxs-lookup"><span data-stu-id="92394-113">Enter the Unit price or hte Amount of the sale of the fixed asset.</span></span>
10. <span data-ttu-id="92394-114">Klicka på radinformation.</span><span class="sxs-lookup"><span data-stu-id="92394-114">Click Line details.</span></span>  
11. <span data-ttu-id="92394-115">Välj numret på anläggningstillgången som ska säljas.</span><span class="sxs-lookup"><span data-stu-id="92394-115">Select the Fixed asset number to be sold.</span></span>
12. <span data-ttu-id="92394-116">Klicka på Bokför.</span><span class="sxs-lookup"><span data-stu-id="92394-116">Click Post.</span></span>

# <a name="view-an-expense-report"></a><span data-ttu-id="e6fb5-103">Visa en utgiftsrapport</span><span class="sxs-lookup"><span data-stu-id="e6fb5-103">View an expense report</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="e6fb5-104">När du kör en fråga för att visa transaktioner i redovisningen eller anskaffning och källa kan frågan returnera alltför mycket information.</span><span class="sxs-lookup"><span data-stu-id="e6fb5-104">When you run a query to view transactions in General ledger or Procurement and sourcing, the query might return too much information.</span></span> <span data-ttu-id="e6fb5-105">Därför kanske du vill begränsa sökningen.</span><span class="sxs-lookup"><span data-stu-id="e6fb5-105">Therefore, you might want to narrow your search.</span></span> <span data-ttu-id="e6fb5-106">För att få den information du behöver kan du rulla ned till en transaktion för att se utgiftsrapporten den visades på.</span><span class="sxs-lookup"><span data-stu-id="e6fb5-106">To get the information that you require, you can drill down into a transaction to see the expense report that it appeared on.</span></span>

<span data-ttu-id="e6fb5-107">För utgiftsrapporter kan du rulla ned från en transaktion till den ursprungliga rapporten som lagras i Utgiftshantering.</span><span class="sxs-lookup"><span data-stu-id="e6fb5-107">For expense reports, you can drill down from a transaction to the original report that is stored in Expense management.</span></span> <span data-ttu-id="e6fb5-108">Förutom det totala beloppet för utgifter kan du visa de specificerade transaktionerna, transaktionerna som berättigar till återbetalning av moms och de kvitton som är kopplade till utgifterna.</span><span class="sxs-lookup"><span data-stu-id="e6fb5-108">In addition to the total amount of expenses, you can view the itemized transactions, the transactions that are eligible for recovery of value-added tax (VAT), and the receipts that are attached to the expenses.</span></span>

<span data-ttu-id="e6fb5-109">Gå ned till den ursprungliga utgiftsrapporten som en transaktion visades på, välja transaktionen och välj sedan **originaldokument**.</span><span class="sxs-lookup"><span data-stu-id="e6fb5-109">To drill down to the original expense report that a transaction appeared on, select the transaction, and then select **Original document**.</span></span>

<span data-ttu-id="e6fb5-110">När du visar enskilda transaktioner kan du också visa ytterligare data som kopplas till en transaktion.</span><span class="sxs-lookup"><span data-stu-id="e6fb5-110">When you view individual transactions, you can also view any additional data that is attached to a transaction.</span></span> <span data-ttu-id="e6fb5-111">Till exempel för en specificerad hotellkostnad kan du visa en detaljerad lista över utgifter såsom internetåtkomst, telefonsamtal och room service.</span><span class="sxs-lookup"><span data-stu-id="e6fb5-111">For example, for an itemized hotel expense, you can view a detailed list of expenses such as internet access, telephone calls, and room service.</span></span>

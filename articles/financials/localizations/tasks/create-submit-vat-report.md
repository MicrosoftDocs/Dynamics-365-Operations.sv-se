# <a name="create-and-submit-vat-report"></a>Skapa och skicka en momsrapport

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

I den här proceduren får du hjälp att skapa och skicka in en momsrapport till tyska myndigheter.

Proceduren har skapats med demodataföretaget DEMF. 

Den här funktionen är bara tillgänglig för juridiska personer som har en primär adress i Tyskland.



1. Gå till Moms > Deklarationer > Moms > Rapportera moms för kvittningsperiod.
2. Ange eller välj ett värde i fältet Settlement period.
3. Ange ett datum i fältet Från datum.
4. Klicka på OK.
5. Välj Ja i fältet Skapa dokument för elektronisk momsdeklaration.
6. Klicka på OK.

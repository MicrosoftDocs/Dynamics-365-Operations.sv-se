# <a name="create-a-credit-note-on-project-invoices"></a>Skapa en kreditfaktura för projektfakturor

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

1. Gå till Projekthantering och redovisning > Projekt > Alla projekt. 
2. Hitta och markera önskad post i listan. 
3. Välj det projekt-ID som används för kreditfakturan. Klicka på länken på den valda raden i listan. 
4. Klicka på Hantera i åtgärdsfönstret. 
5. Klicka på Fakturajournaler. 
6. Välj transaktionen för kreditfakturan. Markera vald rad i listan. 
7. Klicka på Funktioner. 
8. Klicka på Välj för kreditfaktura. 
9. Välj transaktion för kreditfakturan genom att markera kryssrutan.
10. Klicka på OK. 

Efter detta steg blir kreditfakturan tillgänglig för fakturaförslaget.

# <a name="rename-or-modify-a-project-stage"></a>Ändra namn på eller ändra en projektfas

[!include [task guide banner](../../includes/task-guide-banner.md)]

Projektfaser kan användas som en guide till projektets livscykel. Du kan definiera de projektfaser som du vill inkludera i dina projekt på fliken Projektfas på sidan Projekthantering och redovisningsparametrar. Dynamics 365 for Finance and Operations ger fem fördefinierade faser för projekt och tre användardefinierade faser. Du kan uppdatera namn på en projektfas som du använder för att följa utvecklingen av projekt. Du kan också ange namn på användardefinierade statusnamn. Den här uppgiften visar hur du ändrar namn eller ändrar en projektfas och använder datauppsättningen USSI.

1. Gå till Projektledning och redovisning > Inställningar > Parametrar för projekthantering och redovisning.
2. Klicka på fliken Projektfas.
3. I fältet Språk väljer du det språk som du vill använda för den nya projektfasens namn.
4. Klicka på länken i listan och välj den fördefinierade eller användardefinierade fas du vill ändra. 
5. Ange namnet på den nya projektfasen i fältet Projektfas.
6. Klicka på Spara.

<span data-ttu-id="b6499-104">Följande procedur visar de steg som systemadministratörer kan använda för att uppdatera webbannons- eller logotypbilden som visas för en juridisk person.</span><span class="sxs-lookup"><span data-stu-id="b6499-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="b6499-105">Det demonstrationsdataföretag som används för att skapa den här proceduren är USMF.</span><span class="sxs-lookup"><span data-stu-id="b6499-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="b6499-106">Gå till Organisationsadministration > Organisationer > Juridiska personer.</span><span class="sxs-lookup"><span data-stu-id="b6499-106">Go to Organization administration > Organizations > Legal entities.</span></span>

    * <span data-ttu-id="b6499-114">Tips: den idealiska lösningen för en webbannonsbild är 1 920 × 281 pixlar.</span><span class="sxs-lookup"><span data-stu-id="b6499-114">Tip: The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="b6499-115">Den idealiska bredden för en logotypbild är 350 pixlar.</span><span class="sxs-lookup"><span data-stu-id="b6499-115">The ideal width for a logo image is 350 pixels.</span></span>

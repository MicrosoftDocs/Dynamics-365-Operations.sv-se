# <a name="set-up-i-9-document-types"></a>Ställ in 1-9-dokumenttyper

[!include[task guide banner](../../../includes/task-guide-banner.md)]

I den här proceduren visas hur du visar och ställer in dokumenttyper som används för I-9-verifiering. Innan du ställer in I-9-dokumenttyper måste du även ställa in de utfärdande organisationerna och identifieringstyperna. Det demodataföretag som används för att skapa den här proceduren är USMF, som innehåller exempel på utfärdande myndigheter och identifieringstyper som behövs för att slutföra proceduren.

1. Gå till Personal > Arbetare > I-9 > I-9-dokumenttyper.
2. Klicka på Ny.
3. Skriv ett värde i fältet I-9-dokumenttyp.
    * Exempel: Skol-ID.  
4. Välj identifieringstypen.  Exempel: Skol-ID.
    * Exempel på identifieringstyper inkluderar ett personnummer (Social Security number (SSN)), visumnummer, pass-ID, eller körkort.  
5. Välj I-9-dokumentlistan som används för dokumenttypen.
    * Som en del av processen I-9 måste medarbetare visa upp dokumentation som visar arbetsgivaren hans/hennes identitet och Employment Authorization. Webbplatsen US Citizenship and Immigration Services innehåller information om vilka dokument som är godkända och vilken lista de tillhör.  http://www.uscis.gov  
6. Ange det officiella formulärnumret för dokumenttypen i fältet Formulär. Exempel: Skol-ID.
    * De officiella formulärnumren hittar du på webbplatsen US Citizenship and Immigration Services.  http://www.uscis.gov  
7. Markera eller avmarkera kryssrutan Aktiv.
8. Ange datumet till "2019-10-27" i fältet Upphör.
    * Det är valfritt att ange ett utgångsdatum.  
9. Välj den myndighet som har utfärdat dokumenttypen. Exempel: Region/område
10. Klicka på Spara.

# <a name="create-a-goal"></a>Skapa ett mål

[!include [task guide banner](../../includes/task-guide-banner.md)]

Du kan skapa mål, lägga till aktiviteter för mål, spåra mått och länka målen till en granskning. Denna procedur skapades med demodataföretaget USMF. Denna procedur är avsedd för en funktion som lades till i Dynamics 365 for Operations, version 1611.

1. Gå till All workspaces > Employee self service.
2. Klicka på View all goals
3. Klicka på Ny.
4. Ange "Complete the Contoso project" i namnfältet.
5. Ange "I will finish the Contoso project in 30 days" i fältet Overview.
    * Mål kan gälla för en person, ett team eller till och med ett företag. Standardinställningen för medarbetare är "enskild".  
6. I fältet Goal anger eller väljer du ett värde.
7. Ange visat datum i fältet Start date.
8. Ändra det visade datumet i fältet End date.
    * När du avslutar ett mål, ange då det datum då du slutförde målet här.  
9. Klicka på Spara.
10. Expandera avsnittet Measurements. Om det redan har expanderats, klicka då på pilen Expand två gånger.
11. Klicka på Add measurement för att öppna dialogrutan.
12. Ange eller välj ett värde i fältet Measurement.
13. Ange ett värde i fältet Target amount.
14. Klicka på OK.
15. Expandera valet Activities. Om det redan har expanderats, klicka då på pilen Expand två gånger.
16. Klicka på Lägg till.
17. Ange ett värde i fältet Titel.
18. Ange ett värde i fältet Beskrivning.
19. Klicka på Spara.
20. Klicka på Show sections för att öppna dialogrutan.
21. Välj No i fältet Show measurements.
22. Klicka på Spara.
23. Klicka på Show sections för att öppna dialogrutan.
24. Välj Yes i fältet Show measurements.
25. Klicka på Spara.
    * Det finns ofta mål som är ofta desamma eller liknande för teammedlemmar eller för alla medarbetare.     Din personaladministratör eller chef kan skapa målmallar för dessa mål. Du kan lägga till ett mål från en mall och sedan ändra den så att den matchar dina tidslinjer.  
    * Det finns ofta ett flertal mål som är ofta desamma eller liknande för teammedlemmar eller för alla medarbetare.     Din personaladministratör eller chef kan skapa målgrupper som innehåller en eller flera målmallar. Du kan lägga till mål från en målgrupp och sedan ändra dem så att de matchar dina tidslinjer.  
    * När du skapar en granskning, kan du lägga till mål till den granskningen. Se uppgiftsguiden för granskningen för att se hur du kan lägga till mål till dina granskningar.

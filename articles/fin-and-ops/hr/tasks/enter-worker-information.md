# <a name="enter-worker-information"></a>Ange arbetarinformation

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

När en rekryterare har skapat ett massanställningsprojekt och genererat befattningar, är nästa steg att ange arbetarinformation för de personer som anställs. Det demonstrationsdataföretag som används för att skapa den här proceduren är USMF. Demonstrationsdata innehåller ett öppet massanställningsprojekt. Gå till Personal > Rekrytering > Massanställningsprojekt för att starta proceduren.

1. Markera vald rad i listan.
2. Klicka på Befattningsinformation.
    * Redigera startdatumet för arbetaren.  
    * Redigera slutdatumet för arbetaren.  
    * Ändra arbetartypen för arbetare, om det behövs.  
3. I fältet Förnamn, skriv ett värde.
4. Skriv ett värde i fältet Mellannamn.
5. I fältet Efternamn, skriv ett värde.
    * Ändra jobbet för arbetaren, om det behövs.  
    * Ändra avdelningen för arbetaren, om det behövs.  
    * Ändra heltidslönestatusen för arbetaren, om det behövs.  
6. Klicka på OK.
7. Hitta och markera önskad post i listan.
8. Avmarkera vald rad i listan.
    * Välj nästa projektbefattning och fortsätt ange information om arbetare för varje person som du anställer.

# <a name="create-a-legal-entity"></a><span data-ttu-id="dc726-103">Skapa en juridisk person</span><span class="sxs-lookup"><span data-stu-id="dc726-103">Create a legal entity</span></span>

<span data-ttu-id="dc726-104">En juridisk person är en organisation som identifieras genom registrering med en juridisk myndighet.</span><span class="sxs-lookup"><span data-stu-id="dc726-104">A legal entity is an organization that is identified through registration with a legal authority.</span></span> <span data-ttu-id="dc726-105">Juridiska personer kan ingå kontrakt och behövs för att förbereda utdrag som rapporterar på deras resultat.</span><span class="sxs-lookup"><span data-stu-id="dc726-105">Legal entities can enter into contracts and are required to prepare statements that report on their performance.</span></span> <span data-ttu-id="dc726-106">Följande procedur förklarar hur du skapar en juridisk person.</span><span class="sxs-lookup"><span data-stu-id="dc726-106">The following procedure explains how to create a legal entity.</span></span> <span data-ttu-id="dc726-107">Det demonstrationsdataföretag som används för att skapa den här proceduren är USMF.</span><span class="sxs-lookup"><span data-stu-id="dc726-107">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="dc726-108">Gå till Organisationsadministration > Organisationer > Juridiska personer.</span><span class="sxs-lookup"><span data-stu-id="dc726-108">Go to Organization administration > Organizations > Legal entities.</span></span>
2. <span data-ttu-id="dc726-109">Klicka på Ny.</span><span class="sxs-lookup"><span data-stu-id="dc726-109">Click New.</span></span>
3. <span data-ttu-id="dc726-110">Skriv ett värde i fältet Namn.</span><span class="sxs-lookup"><span data-stu-id="dc726-110">In the Name field, type a value.</span></span>
4. <span data-ttu-id="dc726-111">Skriv ett värde i fältet Företag.</span><span class="sxs-lookup"><span data-stu-id="dc726-111">In the Company field, type a value.</span></span>
5. <span data-ttu-id="dc726-112">Ange eller välj ett värde i fältet Land/region.</span><span class="sxs-lookup"><span data-stu-id="dc726-112">In the Country/region field, enter or select a value.</span></span>
6. <span data-ttu-id="dc726-113">Klicka på OK.</span><span class="sxs-lookup"><span data-stu-id="dc726-113">Click OK.</span></span>
    * <span data-ttu-id="dc726-114">I avsnittet Allmänt ger du följande allmän information om den juridiska personen: Ange ett söknamn, om ett söknamn krävs.</span><span class="sxs-lookup"><span data-stu-id="dc726-114">In the General section, provide the following general information about the legal entity: Enter a search name, if a search name is required.</span></span> <span data-ttu-id="dc726-115">En söknamn är ett alternativt namn som kan användas för att söka efter den här juridisk person.</span><span class="sxs-lookup"><span data-stu-id="dc726-115">A search name is an alternate name that can be used to search for this legal entity.</span></span> <span data-ttu-id="dc726-116">Markera om den här juridisk person används som ett konsolideringsföretag.</span><span class="sxs-lookup"><span data-stu-id="dc726-116">Select whether this legal entity is being used as a consolidation company.</span></span> <span data-ttu-id="dc726-117">Markera om den här juridisk person används som ett elimineringsföretag.</span><span class="sxs-lookup"><span data-stu-id="dc726-117">Select whether this legal entity is being used as an elimination company.</span></span>  
7. <span data-ttu-id="dc726-118">Expandera avsnittet Adresser.</span><span class="sxs-lookup"><span data-stu-id="dc726-118">Expand the Addresses section.</span></span>
    * <span data-ttu-id="dc726-119">Ange adressinformation, till exempel gatunamn och nummer, postnummer och ort i avsnittet Adresser.</span><span class="sxs-lookup"><span data-stu-id="dc726-119">In the Addresses section, enter address information, such as the street name and number, postal code, and city.</span></span>  
8. <span data-ttu-id="dc726-120">Expandera avsnittet Kontaktinformation.</span><span class="sxs-lookup"><span data-stu-id="dc726-120">Expand the Contact information section.</span></span>
    * <span data-ttu-id="dc726-121">Ange information om metoder för kommunikationen till exempel e-postadresser, telefonnummer, webbadress (URL), i avsnittet Kontaktinformation.</span><span class="sxs-lookup"><span data-stu-id="dc726-121">In the Contact information section, enter information about methods of communication, such as email addresses, URLs, and telephone numbers.</span></span>  
9. <span data-ttu-id="dc726-122">Expandera avsnittet Rapportering enligt lag.</span><span class="sxs-lookup"><span data-stu-id="dc726-122">Expand the Statutory reporting section.</span></span>
    * <span data-ttu-id="dc726-123">Ange registreringsnumren som används för lagstadgad rapportering i avsnittet Rapportering enligt lag.</span><span class="sxs-lookup"><span data-stu-id="dc726-123">In the Statutory reporting section, enter the registration numbers that are used for statutory reporting.</span></span>

    * <span data-ttu-id="dc726-125">I avsnittet Registreringsnummer, ange vilken information som krävs av den juridiska personen.</span><span class="sxs-lookup"><span data-stu-id="dc726-125">In the Registration numbers section, enter any information required by the legal entity.</span></span>  
11. <span data-ttu-id="dc726-126">Expandera avsnittet Bankkontoinformation.</span><span class="sxs-lookup"><span data-stu-id="dc726-126">Expand the Bank account information section.</span></span>
    * <span data-ttu-id="dc726-127">Ange bankkonton och organisationsnummer för juridiska personen i avsnittet Bankkontoinformation.</span><span class="sxs-lookup"><span data-stu-id="dc726-127">In the Bank account information section, enter bank accounts and routing numbers for the legal entity.</span></span>  
12. <span data-ttu-id="dc726-128">Visa avsnittet Utländsk handel och logistik</span><span class="sxs-lookup"><span data-stu-id="dc726-128">Expand the Foreign trade and logistics section.</span></span>
    * <span data-ttu-id="dc726-129">Ange leveransinformation för juridiska personen i avsnittet Utrikeshandel- och logistik.</span><span class="sxs-lookup"><span data-stu-id="dc726-129">In the Foreign trade and logistics section, enter shipping information for the legal entity.</span></span>  
13. <span data-ttu-id="dc726-130">Visa avsnittet nummersekvenser.</span><span class="sxs-lookup"><span data-stu-id="dc726-130">Expand the Number sequences section.</span></span>
    * <span data-ttu-id="dc726-131">I avsnittet Nummerserie, kan du visa de nummerserier som är kopplade till juridiska personen.</span><span class="sxs-lookup"><span data-stu-id="dc726-131">In the Number sequences section, you can view the number sequences that are associated with the legal entity.</span></span>  
14. <span data-ttu-id="dc726-132">Expandera avsnittet Bilder.</span><span class="sxs-lookup"><span data-stu-id="dc726-132">Expand the Images section.</span></span>
    * <span data-ttu-id="dc726-133">I avsnittet Bilder kan du visa eller ändra logotyp- och/eller instrumentpanelbilden som associeras med juridiska personen.</span><span class="sxs-lookup"><span data-stu-id="dc726-133">In the Images section, view or change the logo and/or dashboard image that are associated with the legal entity.</span></span>  
15. <span data-ttu-id="dc726-134">Expandera momsregistreringsavsnittet.</span><span class="sxs-lookup"><span data-stu-id="dc726-134">Expand the Tax registration section.</span></span>
    * <span data-ttu-id="dc726-135">Ange registreringsnumren som används för att rapportera till skattemyndigheten i avsnittet Momsregistrering.</span><span class="sxs-lookup"><span data-stu-id="dc726-135">In the Tax registration section, enter the registration numbers that are used to report to tax authorities.</span></span>  
16. <span data-ttu-id="dc726-136">Expandera avsnittet Moms 1099.</span><span class="sxs-lookup"><span data-stu-id="dc726-136">Expand the Tax 1099 section.</span></span>
    * <span data-ttu-id="dc726-137">Ange 1099-information för juridiska personen i avsnittet Moms 1099.</span><span class="sxs-lookup"><span data-stu-id="dc726-137">In the Tax 1099 section, enter 1099 information for the legal entity.</span></span>  
17. <span data-ttu-id="dc726-138">Klicka på Spara.</span><span class="sxs-lookup"><span data-stu-id="dc726-138">Click Save.</span></span>

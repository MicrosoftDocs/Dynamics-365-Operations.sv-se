# <a name="change-the-date-for-a-session"></a>Ändra datumet för en session

[!include [task guide banner](../../includes/task-guide-banner.md)]

I det här avsnittet beskrivs hur du ändrar datumet för en session i Dynamics 365 for Finance and Operations. Som standard används aktuellt datum vid registrering och bokföring av journalposter eller källdokument. Du kan ändra det datum som används för din aktuella session. Använd denna funktion för att antedatera journalposter eller källdokument vid behov.

1. I navigeringsfönstret, gå till **Moduler > Allmänt > Allmänt > Sessionsdatum och -tid**.
2. Ange ett datum i fältet **Datum.**
3. Välj **OK**.

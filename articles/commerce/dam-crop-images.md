<span data-ttu-id="06063-104">I det här avsnittet beskrivs hur du beskär bilder i Microsoft Dynamics 365 Commerce webbplatsskapare.</span><span class="sxs-lookup"><span data-stu-id="06063-104">This topic describes how to crop images in Microsoft Dynamics 365 Commerce site builder.</span></span>

## <a name="overview"></a><span data-ttu-id="06063-105">Översikt</span><span class="sxs-lookup"><span data-stu-id="06063-105">Overview</span></span>

<span data-ttu-id="06063-106">Med mediebiblioteket i Commerce webbplatsskaparen kan du beskära bilder för att optimera dem för olika typer av moduler och visningsformat.</span><span class="sxs-lookup"><span data-stu-id="06063-106">The Commerce site builder Media Library allows you to crop images to optimize them for different module types and viewports.</span></span>

## <a name="crop-an-image"></a><span data-ttu-id="06063-107">Beskära en bild</span><span class="sxs-lookup"><span data-stu-id="06063-107">Crop an image</span></span>

<span data-ttu-id="06063-108">Så här beskär du en bild i webbplatsskapare.</span><span class="sxs-lookup"><span data-stu-id="06063-108">To crop an image in site builder, follow these steps.</span></span>

1. <span data-ttu-id="06063-109">I vänster navigeringsfönster av Commerce webbplatsskapare, välj **mediebiblioteket**.</span><span class="sxs-lookup"><span data-stu-id="06063-109">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="06063-110">Markera den bild som du vill ändra i huvudfönstret.</span><span class="sxs-lookup"><span data-stu-id="06063-110">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="06063-111">Klicka på **Redigera** i kommandofältet.</span><span class="sxs-lookup"><span data-stu-id="06063-111">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="06063-112">Markera bilden som ska övergå **redigeringsläge**.</span><span class="sxs-lookup"><span data-stu-id="06063-112">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="06063-113">Under **redigeringsläge**, välj **redigera vy efter modul**.</span><span class="sxs-lookup"><span data-stu-id="06063-113">Under **Edit Mode**, select **Edit View by Module**.</span></span>
1. <span data-ttu-id="06063-114">Från nedrullningsbara menyn **Modul** välj modultyp.</span><span class="sxs-lookup"><span data-stu-id="06063-114">From the **Module** drop-down menu, select the module type.</span></span>
1. <span data-ttu-id="06063-115">Från nedrullningsbara menyn **Visningstyp** välj visningstyp.</span><span class="sxs-lookup"><span data-stu-id="06063-115">From the **View type** drop-down menu, select the view type.</span></span>
1. <span data-ttu-id="06063-116">Från nedrullningsbara menyn **Placering** välj bildplacering.</span><span class="sxs-lookup"><span data-stu-id="06063-116">From the **Placement** drop-down menu, select the image placement.</span></span>
1. <span data-ttu-id="06063-117">Från nedrullningsbara menyn **visningsområde** välj visningsområdets format.</span><span class="sxs-lookup"><span data-stu-id="06063-117">From the **Viewport** drop-down menu, select the viewport size.</span></span>
1. <span data-ttu-id="06063-118">Bilden är överskriven med området som representerar beskärningsregionen.</span><span class="sxs-lookup"><span data-stu-id="06063-118">The image is overlaid with the area representing the crop region.</span></span> <span data-ttu-id="06063-119">Flytta och ändra storlek på beskärningsregionen efter behov.</span><span class="sxs-lookup"><span data-stu-id="06063-119">Move and resize the crop region as needed.</span></span> <span data-ttu-id="06063-120">Förhållandet mellan bredd och höjd kommer att hanteras automatiskt.</span><span class="sxs-lookup"><span data-stu-id="06063-120">The aspect ratio will be maintained automatically.</span></span>
1. <span data-ttu-id="06063-121">När du är klar väljer du på kommandofältet **Spara** och välj sen **Slutför redigering**.</span><span class="sxs-lookup"><span data-stu-id="06063-121">When you're done, on the command bar, select **Save**, and then select **Finish editing**.</span></span> 

<span data-ttu-id="06063-122">När den anpassade beskärningen har slutförts kommer bildändringarna att börja gälla nästan omedelbart.</span><span class="sxs-lookup"><span data-stu-id="06063-122">After custom cropping is completed, image modifications will take effect almost immediately.</span></span>

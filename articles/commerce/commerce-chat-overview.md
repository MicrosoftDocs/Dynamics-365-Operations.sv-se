# <a name="commerce-chat-features-overview"></a>Funktioner för Commerce-chatt – översikt

[!include [banner](includes/banner.md)]

Genom att integrera Microsoft Dynamics 365 Flerkanal för Customer Service och Power Virtual Agents program med Dynamics 365 Commerce e-handelswebbplatser kan företag stärka sina onlineshoppare genom chattfunktioner. Den här artikeln ger en översikt över Commerce-chattfunktionerna som är tillgängliga för onlineshoppare när 365 Flerkanal för Customer Service och Power Virtual Agents program är integrerade med Dynamics 365 Commerce e-handelswebbplatser.

## <a name="omnichannel-for-customer-service"></a>Flerkanal för Customer Service

Flerkanal för Customer Service är ett robust program som utökar effekten av Dynamics 365 Customer Service. Med den kan organisationer ansluta och kommunicera med kunderna via kanaler som live-chatt, chatt och kort meddelande (SMS). För mer information om Flerkanal för Customer Service, se [Introduktion till Flerkanal för Customer Service](/dynamics365/customer-service/introduction-omnichannel). För information om hur man integrerar Flerkanal för Customer Service med Dynamics 365 Commerce en e-handelswebbplats [finns i Commerce-chatten med modulen Flerkanal för Customer Service](commerce-chat-module.md).

## <a name="power-virtual-agents"></a>Power Virtual Agents

Power Virtual Agents låter dig skapa kraftfulla AI-drivna chattrobotar för en rad förfrågningar. Mer information om Power Virtual Agents, se [Power Virtual Agents översikt](/power-virtual-agents/fundamentals-what-is-power-virtual-agents). För information om hur man integrerar Power Virtual Agents med Dynamics 365 Commerce e-handelswebbplats finns i [Commerce-chatt med Microsoft Power Virtual Agents](chat-module-pva.md).

> [!NOTE]
> Power Virtual Agents har redan inkluderats i Flerkanal för Customer Service. Om du behöver både virtuell agent (chattjänst) och funktioner för livehandläggare, ska du använda Flerkanal för Customer Service.

<span data-ttu-id="bce91-109">Följ dessa steg för att skapa en ny butiksfunktionsprofil.</span><span class="sxs-lookup"><span data-stu-id="bce91-109">To create a retail functionality profile, follow these steps.</span></span>

1. <span data-ttu-id="bce91-110">I navigeringsfönstret, gå till **Moduler \> Kanalinställningar \> Kassaprofiler \> Funktionsprofiler**.</span><span class="sxs-lookup"><span data-stu-id="bce91-110">In the navigation pane, go to **Modules \> Channel setup \> POS profiles \> Functionality profiles**.</span></span>

1. <span data-ttu-id="bce91-116">I avsnittet **Allmänt**, välj **Kvittoprofil-ID** för e-postkvitton.</span><span class="sxs-lookup"><span data-stu-id="bce91-116">In the **General** section, select a **Receipt profile ID** for email receipts.</span></span>
1. <span data-ttu-id="bce91-117">I avsnittet **Funktioner** ändrar du inställningar efter behov.</span><span class="sxs-lookup"><span data-stu-id="bce91-117">In the **Functions** section, modify settings, as needed.</span></span>
1. <span data-ttu-id="bce91-118">I avsnittet **Belopp** ändrar du inställningar efter behov.</span><span class="sxs-lookup"><span data-stu-id="bce91-118">In the **Amount** section, modify settings as, needed.</span></span>
1. <span data-ttu-id="bce91-119">I avsnittet **Informationskoder** ändrar du inställningar efter behov.</span><span class="sxs-lookup"><span data-stu-id="bce91-119">In the **Info Codes** section, modify settings, as needed.</span></span>

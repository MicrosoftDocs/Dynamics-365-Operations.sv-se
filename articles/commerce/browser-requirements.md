# <a name="browser-requirements-for-dynamics-365-commerce-sites"></a>Krav för webbläsare för Dynamics 365 Commerce-webbplatser

[!include [banner](includes/banner.md)]

Detta ämne behandlar webbläsarkraven för Microsoft Dynamics 365 Commerce-webbplatser.

Dynamics 365 Commerce-webbplatser stöds bäst av följande webbläsare. Gå till programtillverkarens webbplats för att hitta den senaste versionen för respektive webbläsare.

- [Microsoft Edge](https://www.microsoft.com/edge) (upp till aktuell version minus 1)
- [Mozilla Firefox](https://www.mozilla.org/firefox/new) (upp till aktuell version minus 1)
- [Google Chrome](https://www.google.com/chrome) (upp till aktuell version minus 1)
- [Apple Safari](https://support.apple.com/downloads/safari) (upp till aktuell version minus 1)

> [!NOTE]
> Internet Explorer stöds inte. Om du använder Internet Explorer kanske vissa funktioner inte fungerar, och användarna kan därför få en undermålig upplevelse.

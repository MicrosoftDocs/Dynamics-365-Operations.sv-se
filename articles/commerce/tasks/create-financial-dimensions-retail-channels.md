# <a name="create-financial-dimensions-for-retail-channels-and-configure-dimension-values-on-stores"></a>Skapa ekonomiska dimensioner för butikskanaler och konfigurera dimensionsvärden för butiker

[!include [banner](../includes/banner.md)]

Den här proceduren beskriver hur du skapar en ekonomisk dimension för handelskanal med dimensionsvärden och stegen för att konfigurera värden för ekonomisk dimension på butiker. Artikeln inkluderar inte andra relaterade steg, till exempel att skapa dimensionsuppsättningar och kontostrukturer. I proceduren används demonstrationsföretaget USRT.

1. Gå till huvudboken > kontoplan > mått > finansiella mått.
2. Klicka på Ny.
3. I fältet Använd värden från, välj Commerceskanaler.
4. I fältet Dimensionsnamn, skriv ett värde.
5. Klicka på Aktivera.
6. Klicka på Stäng.
7. Klicka på Aktivera.
8. Klicka på Dimensionsvärden.
9. Stäng sidan.
10. Klicka på Spara.
11. Stäng sidan.
12. Gå till Butik och handel > Kanaler > Butiker >Alla butiker.
13. Klicka på länken på den valda raden i listan.
14. Växla expansionen av avsnittet Ekonomiska dimensioner.
15. Klicka på Redigera.
16. Öppna sökningen genom att klicka på den nedrullningsbara knappen i fältet Commerceskanal.
17. I listan du söker efter och väljer dimensionsvärdet för butiken som uppdateras.
18. Klicka på länken på den valda raden i listan.
19. Öppna sökningen genom att klicka på den nedrullningsbara knappen i fältet CostCenter.
20. Hitta och markera önskad post i listan.
21. Klicka på länken på den valda raden i listan.
22. Öppna sökningen genom att klicka på den nedrullningsbara knappen i fältet Avdelning.
23. Hitta och markera önskad post i listan.
24. Klicka på länken på den valda raden i listan.
25. Klicka på Spara.

# <a name="configure-credit-card-processing"></a> Konfigurera kreditkortsbearbetning

[!include [banner](../includes/banner.md)]

Den här proceduren går igenom hur du visar en lista med betalningsförmedlare och hur du konfigurerar ett betalningskonto för kundreskontra. I den här proceduren används i USRT-företaget med demodata och är avsedda för administratörer och IT-ansvariga.


## <a name="view-a-list-of-payment-providers"></a>Visa en lista med betalningsförmedlare
1. Gå till kundfordringar > Betalningar inställning > betaltjänster.
2. Klicka på Visa tillgängliga leverantörer.

## <a name="configure-payment-account"></a>Konfigurera betalningskonto
1. Klicka på Ny.
2. Skriv ett värde i fältet Betalningstjänst.
3. Välj ett alternativ i fältet Betalningskoppling.
4. Växla utökningen av avsnittet Konto för betaltjänster.
5. I fältet Miljö, skriv PROD.
6. Klicka på Kreditkortstyper.
7. Öppna sökningen genom att klicka på den nedrullningsbara knappen i fältet Betalningsjournal.
8. Klicka på länken på den valda raden i listan.
9. Klicka på Lägg till.
10. Ange ett värde i fältet Valuta.
11. Hitta och markera önskad post i listan.
12. Öppna sökningen genom att klicka på den nedrullningsbara knappen i fältet Betalningsjournal.
13. Klicka på länken på den valda raden i listan.
14. Klicka på Lägg till.
15. Ange ett värde i fältet Valuta.
16. Hitta och markera önskad post i listan.
    * Du kan upprepa dessa steg för så många korttyper som du behöver.  
17. Öppna sökningen genom att klicka på den nedrullningsbara knappen i fältet Betalningsjournal.
18. Klicka på länken på den valda raden i listan.
19. Klicka på Lägg till.
20. Ange ett värde i fältet Valuta.
21. Klicka på Spara.
22. Stäng sidan.
23. Klicka på Validera.
24. Klicka på kryssrutan Standardföretag för nya kreditkort.
25. Klicka på Spara.

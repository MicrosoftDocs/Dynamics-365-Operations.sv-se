# <a name="customize-image-focal-points"></a><span data-ttu-id="44144-103">Anpassa bildens fokuspunkter</span><span class="sxs-lookup"><span data-stu-id="44144-103">Customize image focal points</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="44144-104">I det här avsnittet beskrivs hur du anpassar bildens fokuspunkter i Microsoft Dynamics 365 Commerce webbplatsskapare.</span><span class="sxs-lookup"><span data-stu-id="44144-104">This topic describes how to customize image focal points in Microsoft Dynamics 365 Commerce site builder.</span></span>

<span data-ttu-id="44144-105">När en bild överförs till mediebiblioteket för Commerce-webbplatsskapare försöker systemet fastställa bildens fokuspunkt.</span><span class="sxs-lookup"><span data-stu-id="44144-105">When an image is uploaded to the Commerce site builder Media Library, the system attempts to determine the focal point of the image.</span></span> <span data-ttu-id="44144-106">Om bilden till exempel har en person på den, ställer systemet in fokuspunkten på personen som standard.</span><span class="sxs-lookup"><span data-stu-id="44144-106">For example, if the image has a person on it, the system will set the focal point to the face of the person by default.</span></span> <span data-ttu-id="44144-107">I de flesta fall fungerar fokuspunkten automatiskt bra för alla visningslägen, men ibland kan du behöva justera fokuspunkten för att se till att en viss del av bilden alltid är synlig.</span><span class="sxs-lookup"><span data-stu-id="44144-107">In most cases the automatically set focal point works well for all viewports, but sometimes you may want to adjust the focal point to ensure that a specific part of the image is always visible.</span></span>

### <a name="define-a-custom-focal-point-for-an-image"></a><span data-ttu-id="44144-108">Definiera en anpassad fokuspunkt för en bild</span><span class="sxs-lookup"><span data-stu-id="44144-108">Define a custom focal point for an image</span></span>

<span data-ttu-id="44144-109">Om du vill definiera en egen fokuspunkt för en bild följer du stegen nedan.</span><span class="sxs-lookup"><span data-stu-id="44144-109">To define a custom focal point for an image, follow these steps.</span></span>

1. <span data-ttu-id="44144-110">I vänster navigeringsfönster av Commerce webbplatsskapare, välj **mediebiblioteket**.</span><span class="sxs-lookup"><span data-stu-id="44144-110">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="44144-111">Markera den bild som du vill ändra i huvudfönstret.</span><span class="sxs-lookup"><span data-stu-id="44144-111">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="44144-112">Klicka på **Redigera** i kommandofältet.</span><span class="sxs-lookup"><span data-stu-id="44144-112">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="44144-113">Markera bilden som ska övergå **redigeringsläge**.</span><span class="sxs-lookup"><span data-stu-id="44144-113">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="44144-114">Under **Redigera läge**, välj **Ändra fokuspunkt**.</span><span class="sxs-lookup"><span data-stu-id="44144-114">Under **Edit Mode**, select **Change Focal Point**.</span></span> <span data-ttu-id="44144-115">En cirkelformad fokuspunktskontroll visas över bilden.</span><span class="sxs-lookup"><span data-stu-id="44144-115">A circular focal point control appears over the image.</span></span>
1. <span data-ttu-id="44144-116">Markera fokuspunktkontrollen för att flytta den över önskad fokuspunkt.</span><span class="sxs-lookup"><span data-stu-id="44144-116">Select the focal point control to move it over the desired focal point.</span></span>
1. <span data-ttu-id="44144-117">När du är klar väljer du på kommandofältet **Spara** och välj sen **Slutför redigering**.</span><span class="sxs-lookup"><span data-stu-id="44144-117">When you're done, on the command bar select **Save**, and then select **Finish editing**.</span></span>

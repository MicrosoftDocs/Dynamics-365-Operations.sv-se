# <a name="ways-to-add-content"></a>Sätt att lägga till innehåll

[!include [banner](includes/preview-banner.md)]
[!include [banner](includes/banner.md)]

Det här avsnittet innehåller information om hur du lägger till och hanterar innehåll på din Microsoft Dynamics 365 Commerce-webbplats.

## <a name="overview"></a>Översikt

Det finns många sätt att ändra utseende, känsla och innehåll på din webbplats. Beroende på vilken anpassningsnivå som krävs kan många av dessa ändringar genomföras av andra än utvecklare. Till exempel behöver ingen kod skrivas för att skapa mallar, välja teman och välja och konfigurera moduler. Utvecklingskunskaper krävs däremot för att skapa ett nytt tema eller en ny modul, eftersom programutvecklingskit (SDK) frö e-handel och Microsoft Dynamics Lifecycle Services (LCS) måste användas för distribution arbetsflöde.

Följande ämne ger detaljerad information om hur du lägger till och hanterar innehåll på din webbplats. De fokuserar på områden på din webbplats som inte behöver en utvecklaren. Vid behov pekar de ut uppgifter som kräver SDK-arbete.

- Om du vill ändra text, bilder eller video på en befintlig webbplatssida, se [arbeta med moduler](work-with-modules.md).
- För att garantera en idiotsäker, varumärkesredigeringsupplevelse för författare till webbinnehåll, se [arbeta med mallar](work-with-templates.md).
- Information om hur du ordnar om avsnitt på en webbplatssida finns på [arbeta med layouter](work-with-layouts.md).
- Om du vill ändra teckensnitt, färger och utseende på webbplatssidorna se [Välj ett webbplatstema](select-site-theme.md).

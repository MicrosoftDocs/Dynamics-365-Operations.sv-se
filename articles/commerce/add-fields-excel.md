# <a name="add-fields-to-an-excel-workbook-to-edit-retail-transactions"></a>Lägga till fält i en Excel-arbetsbok för att redigera butikstransaktioner

[!include [banner](../includes/banner.md)]

I det här avsnittet beskrivs hur du lägger till fält i en Microsoft Excel-arbetsbok så att du kan redigera butikstransaktioner i Microsoft Dynamics 365 Commerce.

## <a name="overview"></a>Översikt

När du genererar en Excel-fil så att du kan redigera butikstransaktioner fylls filen med några standardfält. Om ett fält som måste uppdateras inte syns som standard i den genererade Excel-filen kan du lägga till det.

## <a name="add-fields-to-a-worksheet-in-an-excel-workbook"></a>Lägga till fält i ett kalkylblad i en Excel-arbetsbok

Följ de här stegen om du vill lägga till fält i en Excel-arbetsbok så att du kan redigera butikstransaktioner.

1. Ladda ned och öppna Excel-filen från sidan **Utdrag**, sidan **Butikstransaktioner** eller panelen **Transaktionsvalideringsfel** i arbetsytan **Butiksekonomi**.
1. Välj **Design**.
1. Välj pennsymbolen för önskad tabell, och sök sedan i listan över tillgängliga fält och markera det fält som du vill lägga till.
1. Välj **Lägg till** och sedan **Uppdatera**. Du kan ändra ordning på fälten.
1. När uppdateringen är klar ska du välja **Uppdatera** för att hämta data för den nya kolumnen.

Det nya fältet och data för det ska nu vara tillgängliga för redigering i Excel.

# <a name="gender"></a>Kön


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

I den här artikeln beskrivs alternativet Kön som ställts in i Dynamics 365 Human Resources.

Fysiskt namn: mshr_hcmpersongender

Denna uppräkning utgör alternativuppsättningen för kön för kandidaten. Den är tillgänglig i alternativuppsättningen mshr_hcmpersongender.

| Värde | Etikett | beskrivning |
| --- | --- | --- |
| 200000000 | None | Inget kön har angetts. |
| 200000001 | Man | Man. |
| 200000002 | Kvinna | Kvinna. |
| 200000003 | Icke-specifikt | Val för ett icke-specifikt kön. |

# <a name="payroll---gender-option-set"></a>Lön - Alternativuppsättning för kön

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

I detta ämne beskrivs alternativuppsättningen Kön för Dynamics 365 Human Resources.

## <a name="option-set-gender-option"></a>Alternativuppsättning: Alternativet Kön

Denna uppräkning utgör alternativuppsättningen för kön för medarbetaren. Den är tillgänglig i alternativuppsättningen mshr_hcmpersongender.

| Värde | Etikett | beskrivning |
| --- | --- | --- |
| 200000000 | None | Inget kön har angetts. |
| 200000001 | Man | Man. |
| 200000002 | Kvinna | Kvinna. |
| 200000003 | Icke-specifikt | Val för ett icke-specifikt kön. |

# <a name="set-up-rounding-rules"></a>Ställa in avrundningsregler

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Du kan ange avrundningsregler i Dynamics 365 Human Resources så att de uppfyller behoven hos din organisation.

1. I arbetsytan **Förmånshantering** under **inställningar**, välj **avrundningsregler**.

2. Välj **Ny**.

3. Ange värden för **Avrunda till närmaste**, **Scenario** och **Avrundningsformulär**.

4. Välj **Spara**.

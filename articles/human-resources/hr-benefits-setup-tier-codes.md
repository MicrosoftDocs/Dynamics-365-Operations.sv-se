# <a name="set-up-tier-codes"></a>Ställa in skiktkoder

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Dynamics 365 Human Resources använder nivåkoder med förmånspriser som betalas av medarbetare och arbetsgivaren. Du kan definiera ett obegränsat antal nivåkoder och ställa in kostnadsberäkningar för att uppfylla företagets behov.

1. I arbetsytan **Förmånshantering** under **inställningar**, välj **nivåkoder**.

2. Välj **Ny**.

3. Ange värden för de följande fälten:

   | Fält | Beskrivning |
   | --- | --- |
   | **Nivåkod** | En unik nivåkod för förmån eller nummer som identifierar nivåkoden. |
   | **Beskrivning** | En beskrivning av nivåkod för förmån. |

4. Välj **Spara**.

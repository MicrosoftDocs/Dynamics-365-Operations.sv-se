# <a name="create-a-goal"></a>Skapa ett mål


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Du kan skapa mål, lägga till aktiviteter för mål, spåra mått och länka målen till en granskning. Denna procedur skapades med demodataföretaget USMF. Denna procedur är avsedd för en funktion som lades till i Dynamics 365 for Operations, version 1611.

1. Gå till **Alla arbetsytor** > **Självbetjäning för medarbetare**.
2. Klicka på **Visa alla mål**.
3. Klicka på **Ny**.
4. I fältet **Namn** ange **Slutför Contoso -projektet**.
5. I fältet **Översikt** ange **Jag kommer att avsluta Contoso-projektet om 30 dagar**.
    * Mål kan gälla för en person, ett team eller till och med ett företag. Standardinställningen för medarbetare är "enskild".  
6. I fältet **målkategori** anger eller väljer du ett värde.
7. Ange visat datum i fältet **Startdatum**.
8. Ändra det visade datumet i fältet **Slutdatum**.
    * När du avslutar ett mål, ange då det datum då du slutförde målet här.  
9. Klicka på **Spara**.
10. Expandera avsnittet **Mått**. Om det redan har expanderats, klicka då på pilen **Expandera** två gånger.
11. Klicka på **Lägg till mått** för att öppna dialogrutan.
12. Ange eller välj ett värde i fältet **Mått**.
13. Ange en siffra i fältet **Målbelopp**.
14. Klicka på **OK**.
15. Expandera valet **Aktiviteter**. Om det redan har expanderats, klicka då på pilen **Expandera** två gånger.
16. Klicka på **Lägg till**.
17. Ange ett värde i fältet **Rubrik**.
18. I fältet **Beskrivning** anger du ett värde.
19. Klicka på **Spara**.
20. Klicka på **Visa avsnitt** för att öppna dialogrutan.
21. Välj **Nej** i fältet **Visa mått**.
22. Klicka på **Spara**.
23. Klicka på **Visa avsnitt** för att öppna dialogrutan.
24. Välj **Ja** i fältet **Visa mått**.
25. Klicka på **Spara**.
    * Det finns mål som är ofta desamma eller liknande för teammedlemmar eller för alla medarbetare. Din personaladministratör eller chef kan skapa målmallar för dessa mål. Du kan lägga till ett mål från en mall och sedan ändra den så att den matchar dina tidslinjer.  
    * Det finns ofta ett flertal mål som är ofta desamma eller liknande för teammedlemmar eller för alla medarbetare. Din personaladministratör eller chef kan skapa målgrupper som innehåller en eller flera målmallar. Du kan lägga till mål från en målgrupp och sedan ändra dem så att de matchar dina tidslinjer.  
    * När du skapar en granskning, kan du lägga till mål till den granskningen. Se uppgiftsguiden för granskningen för att se hur du kan lägga till mål till dina granskningar.

# <a name="education-credit-basis"></a>Kreditbas för Education

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

I detta ämne beskrivs alternativuppsättningen Kreditbas för Education för Dynamics 365 Human Resources.

Fysiskt namn: mshr_hcmeducationcreditbasis

Uppräkningen utgör alternativuppsättningen av värden för utbildningskreditbas för kandidatens utbildningspost.

| Värde | Etikett | beskrivning |
| --- | --- | --- |
| 200000000 | Tom | Inget värde har valts. |
| 200000001 | Period | Termin. |
| 200000002 | Kvartal | Kvartal. |
| 200000003 | Tertial | Trimester. |
| 200000004 | Villkor | Termin. |
| 200000005 | Annat | Övrigt. |

# <a name="change-employee-self-service-workspace-name"></a>Ändra namn på arbetsytan för självbetjäning för medarbetare

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Om du har frivilliga eller andra icke-anställda, kanske du vill ändra namnet på arbetsytan **Självbetjäning för medarbetare**. Du kan ändra arbetsytan till **självbetjäning** i stället.

> [!NOTE]
> Ändra namnet på arbetsytan **Självbetjäning för medarbetare** ändras också menyalternativet som används internt av Dynamics 365 Human Resources. Om du tidigare har använt säkerhetsanpassningar på menyalternativet **HcmEmployeeSelfServiceWorkspace** rekommenderar vi att du tillämpar samma ändringar på **HcmSelfServiceWorkspace** för att behålla pariteten.

1. I personal, välj **personalhantering**, välj **länkar** och välj sedan **personalparametrar**.

2. Välj fliken **medarbetarens självbetjäning**.

3. Under **Displaynamn**, välj **Självbetjäning**.

   ![Ändra namn på arbetsytan för självbetjäning för självbetjäning](./media/hr-employee-self-service-workspace-name.png)

4. Välj **Spara**.

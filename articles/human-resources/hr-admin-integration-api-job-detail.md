Detta ämne tillhandahåller information och en exempelfrågeställning för entiteten för jobbinformation i Dynamics 365 Human Resources.

## <a name="properties"></a>Egenskaper

| Egenskap<br>**Fysiskt namn**<br>**_Typ_** | Använd | beskrivning |
| --- | --- | --- |
| **Jobb-ID**<br>mshr_jobid<br>*Sträng* | Skrivskydd<br>Obligatoriskt | Unikt ID för ett jobb. |
| **Lägre tröskelvärde för marknadspris**<br>mshr_marketpricelowthreshold<br>*Decimal* | | Ett systemgenererat GUID-värde som unikt identifierar positionen.  |

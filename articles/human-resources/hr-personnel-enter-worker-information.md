# <a name="enter-worker-information"></a>Ange arbetarinformation

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]



När en rekryterare har skapat ett massanställningsprojekt och genererat befattningar, är nästa steg att ange arbetarinformation för de personer som anställs. Det demonstrationsdataföretag som används för att skapa den här proceduren är USMF. Demonstrationsdata innehåller ett öppet massanställningsprojekt. Gå till **Personal** > **Rekrytering** > **Massanställningsprojekt** för att inleda denna procedur.

1. Markera vald rad i listan.
2. Klicka på **Befattningsinformation**.
    * Redigera **Startdatum** för medarbetaren.  
    * Redigera **Slutdatum** för medarbetaren.  
    * Ändra vid behov **Medarbetartyp** för medarbetaren.  
3. Ange ett värde i fältet **Förnamn**.
4. Ange ett värde i fältet **Mellannamn**.
5. Ange ett värde i fältet **Efternamn**.
    * Ändra jobbet för arbetaren, om det behövs.  
    * Ändra avdelningen för arbetaren, om det behövs.  
    * Ändra heltidslönestatusen för arbetaren, om det behövs.  
6. Klicka på **OK**.
7. Hitta och markera önskad post i listan.
8. Avmarkera vald rad i listan.
    * Välj nästa projektbefattning och fortsätt ange information om arbetare för varje person som du anställer.

# <a name="payroll---no-yes-option-set"></a>Löm - Alternativuppsättning Nej Ja


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

I denna artikel beskrivs alternativuppsättningen Nej Ja för Dynamics 365 Human Resources.

## <a name="option-set-no-yes"></a>Alternativuppsättning: Nej Ja

Fysiskt namn: mshr_noyes

Denna uppräkning ger en alternativuppsättning för typiska booleska egenskaper i virtuella entiteter för Personal.

| Värde | Etikett | beskrivning |
| --- | --- | --- |
| 200000000 | Nej | Nej. |
| 200000001 | Ja | Ja. |

# <a name="contact-type"></a>Kontakttyp


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

I denna artikel beskrivs alternativuppsättningen Kontakttyp för Dynamics 365 Human Resources.

Fysiskt namn: mshr_logisticselectronicaddressmethodtype

Denna uppräkning utgör alternativuppsättningen med kontakttypsvärden för kandidater. 

| Värde | Etikett | beskrivning |
| --- | --- | --- |
| 200000000 | None | Ingen typ har valts. |
| 200000001 | Telefon | Telefonnummer. |
| 200000002 | E-postadress | E-postadress. |
| 200000003 | URL | Webbplatsens URL. |
| 200000004 | Telex | Telexnummer. |
| 200000005 | Fax | Faxnummer. |
| 200000006 | Facebook | Facebook-konto. Identifieras via användar-ID. |
| 200000007 | Twitter | Twitter-konto. Identifieras via @username. |
| 200000008 | LinkedIn | LinkedIn-konto. Identifieras via användarnamn. |

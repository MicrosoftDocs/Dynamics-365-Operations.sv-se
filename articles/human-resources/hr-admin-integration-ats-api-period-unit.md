# <a name="period-unit"></a>Periodenhet

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

I detta ämne beskrivs alternativuppsättningen Periodisk enhet för Dynamics 365 Human Resources.

Fysiskt namn: mshr_periodunit

Uppräkningen ger alternativuppsättningen värden för måttenheter för tidsperioder.

| Värde | Etikett | beskrivning |
| --- | --- | --- |
| 200000000 | Dagar | Dag. |
| 200000001 | Månader | Månad. |
| 200000002 | År | År. |

# <a name="custom-fields"></a>​Anpassade fält

[!include [rename-banner](~/includes/cc-data-platform-banner.md)]

Du kan lägga till anpassade fält till sidor i Dynamics 365 Human Resources. Mer information om hur du skapar anpassade fält finns i [Skapa och arbeta med anpassade fält](https://docs.microsoft.com/dynamics365/unified-operations/fin-and-ops/get-started/user-defined-fields).

De flesta Common Data Service-entiteter ger stöd för anpassade fält. En fullständig lista över Common Data Service-entiteter finns i [Common Data Service-entiteter](https://docs.microsoft.com/dynamics365/human-resources/hr-developer-entities). 

Se till att du aktiverar alla anpassade fält som du skapar så att du kan använda dem i Common Data Service. Mer information finns i [tillhandahålla anpassade fält i dataenheter](https://docs.microsoft.com/dynamics365/unified-operations/fin-and-ops/get-started/user-defined-fields#exposing-custom-fields-on-data-entities).

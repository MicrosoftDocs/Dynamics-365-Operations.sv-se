# <a name="custom-fields"></a>​Anpassade fält

Du kan lägga till anpassade fält till sidor i Dynamics 365 Human Resources. Mer information om hur du skapar anpassade fält på en sida finns i [Skapa och arbeta med anpassade fält](https://docs.microsoft.com/dynamics365/unified-operations/fin-and-ops/get-started/user-defined-fields).

Följande entiteter i Common Data Service stöder anpassade fält som skapats i Personal:

- Arbetare
- Etniskt ursprung
- Veteranstatus
- Språkkod
- Befattning
- Jobbtyp
- Jobbfunktion
- Befattning
- Befattningstyp

Se till att du aktiverar alla anpassade fält som du skapar så att du kan använda dem i Common Data Service. Mer information finns i [tillhandahålla anpassade fält i dataenheter](https://docs.microsoft.com/dynamics365/unified-operations/fin-and-ops/get-started/user-defined-fields#exposing-custom-fields-on-data-entities).

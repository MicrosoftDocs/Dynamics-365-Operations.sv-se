# <a name="custom-fields"></a>​Anpassade fält

[!include [rename-banner](~/includes/cc-data-platform-banner.md)]

Du kan lägga till anpassade fält till sidor i Dynamics 365 Human Resources. Mer information om hur du skapar anpassade fält finns i [Skapa och arbeta med anpassade fält](https://docs.microsoft.com/dynamics365/unified-operations/fin-and-ops/get-started/user-defined-fields).

De flesta Dataverse-register stöder anpassade fält. En fullständig lista över Dataverse-register finns i [Dataverse-register](https://docs.microsoft.com/dynamics365/human-resources/hr-developer-entities). 

Se till att du aktiverar alla anpassade fält som du skapar så att du kan använda dem i Dataverse. Mer information finns i [tillhandahålla anpassade fält i dataenheter](https://docs.microsoft.com/dynamics365/unified-operations/fin-and-ops/get-started/user-defined-fields#exposing-custom-fields-on-data-entities).

## <a name="option-set-process-type"></a>Alternativuppsättning: Processtyp

Fysiskt namn: mshr_hrmCompProcessType

Denna uppräkning ger en alternativuppsättning för processtyp i virtuella entiteter för Personal.

| Värde | Etikett | beskrivning |
| --- | --- | --- |
| 200000000 | None | Ingen. |
| 200000001 | Årlig | Årlig. |
| 200000002 | Jubileumsdag | Jubileumsdag. |
| 200000003 | Icke-cyklisk | Icke-cyklisk. |
| 200000004 | Halvår | Halvårsvis. |
| 200000005 | Varje kvartal | Varje kvartal. |
| 200000006 | Månatlig | Månatlig. |

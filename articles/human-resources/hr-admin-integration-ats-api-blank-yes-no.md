# <a name="blank-yes-no"></a>Tomt Ja Nej


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

I den här artikeln beskrivs alternativet Tomt Ja Nej för Dynamics 365 Human Resources.

Fysiskt namn: mshr_hrmblankyesno

Denna uppräkning ger en alternativuppsättning för Ja/Nej-egenskaper som även tillåter tomma svar.

| Värde | Etikett | beskrivning |
| --- | --- | --- |
| 200000000 | Tom | Inget värde har valts. |
| 200000001 | Ja | Ja. |
| 200000002 | Nej | Nej |

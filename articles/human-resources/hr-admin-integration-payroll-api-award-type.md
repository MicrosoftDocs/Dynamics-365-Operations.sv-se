# <a name="award-type"></a>Belöningstyp


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

I detta ämne beskrivs alternativuppsättningen Kompensation för Dynamics 365 Human Resources.

## <a name="option-set-award-type"></a>Alternativuppsättning: Kompensationstyp

Fysiskt namn: mshr_HrmCompVarAwardEmplType

Denna uppräkning ger en alternativuppsättning för kompensationstypen variabel kompensation i virtuella entiteter för Personal.

| Värde | Etikett | beskrivning |
| --- | --- | --- |
| 200000000 | Allmän belöning | Allmän kompensation. |
| 200000001 | Fast lön i klumpsumma | Fast lön i klumpsumma. |

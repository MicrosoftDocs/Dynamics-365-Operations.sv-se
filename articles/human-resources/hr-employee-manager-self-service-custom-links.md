# <a name="create-custom-links-in-manager-self-service"></a>Skapa anpassade länkar i Självbetjäning för chef

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Du kan lägga till anpassade länkar på fliken **Mitt team** i Självbetjäning för chef. Den här funktionen ger dig snabb åtkomst till viktig information. Det är ungefär som att lägga till anpassade länkar på fliken **Min information** i Självbetjäning för chef.

## <a name="enable-the--feature"></a>Aktivera funktionen

Om du vill använda den här funktionen aktiverar du **anpassade länkar i Självbetjäning för chef** i arbetsytan **Funktionshantering**. Mer information om att aktivera förhandsfunktioner finns i [Hantera funktioner](hr-admin-manage-features.md).

## <a name="set-up-custom-links"></a>Ställa in anpassade länkar

1. I **Personal-parametrar**, välj **Självbetjäning för chef**.

2. Under **Ange länkar för chefer** kan du lägga till, redigera eller ta bort en länk. Du kan också gruppera länkarna så att de visas i en grupp i Självbetjäning för chef.

   ![Ange anpassade länkar i Självbetjäning för chef](./media/hr-employee-manager-self-service-custom-links-setup.png)

3. Om du vill visa länkarna går du till fliken **Mitt team** i Självbetjäning för medarbetare.

   ![Visa anpassade länkar i Självbetjäning för chef](./media/hr-employee-manager-self-service-custom-links-view.png)

# <a name="screening-frequency"></a>Gallringsfrekvens

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Detta ämne beskriver alternativuppsättningen för gallringsfrekvens för Dynamics 365 Human Resources.

Fysiskt namn: mshr_hcmfrequencyunit

Denna uppräkning utgör alternativuppsättningen med värden för gallringsfrekvensen. 

| Värde | Etikett | beskrivning |
| --- | --- | --- |
| 200000000 Endast en gång | Gallringen krävs bara en gång. |
| 200000001 Dagligen | Gallringsfrekvensen beräknas i dagar. |
| 200000002 Veckovis | Gallringsfrekvensen beräknas i veckor. |
| 200000003 Månadsvis | Gallringsfrekvensen beräknas i månader. |
| 200000004 Årligen | Gallringsfrekvensen beräknas i år. |

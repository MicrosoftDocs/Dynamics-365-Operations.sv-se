# <a name="regulatory-job-category"></a>Kategori för reglerjobb

I detta ämne beskrivs alternativuppsättningen Reglerjobb för Dynamics 365 Human Resources.

Fysiskt namn: mshr_hcmregulatoryjobcategory

Denna uppräkning anger alternativuppsättningen för värden för reglerjobbskategorin. Denna information finns i befintlig alternativuppsättning mshr_hcmregulatoryjobcategory.

| Värde | Etikett | beskrivning |
| --- | --- | --- |
| 200000000 | None | Ingen. |
| 200000001 | Chef | Tjänstemän på direktörs-/ledningsnivå, samt chefer. |
| 200000002 | Ansvarig | Tjänstemän och chefer på lägre/mellannivå. |
| 200000003 | Yrkesmässig | Yrkespersoner. |
| 200000004 | Tekniker | Tekniker. |
| 200000005 | Säljare | Säljare. |
| 200000006 | Administrativ | Administrativa supportmedarbetare. |
| 200000007 | Hantverkare | Hantverkare. |
| 200000008 | Industriarbetare | Industriarbetare. |
| 200000009 | Arbetare | Arbetare/assistenter. |
| 200000010 | Servicepersonal | Servicepersonal. |

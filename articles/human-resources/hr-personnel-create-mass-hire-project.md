# <a name="create-a-mass-hire-project"></a>Skapa ett massanställningsprojekt


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]



I den här proceduren beskrivs processen för att ställa in ett massanställningsprojekt. En rekryterare kan använda massanställningsprojekt för att skapa flera befattningar och anställa ett antal arbetare till dessa befattningar på ett enkelt sätt. Gå till **Personal** > **Rekrytering** > **Massanställningsprojekt** för att inleda denna procedur. Det demonstrationsdataföretag som används för att skapa den här proceduren är USMF.

1. Klicka på **Ny**.
2. I fältet **Massanställningsprojekt** anger du ett värde.
3. I fältet **Beskrivning** anger du ett värde.
4. Ange ett värde i fältet **Projektstart**.
5. I fältet **Projektslut** anger du ett datum.
6. Klicka på **Öppna projekt**.
7. Klicka på **Ja**.
8. Klicka på **Skapa befattningar**.
9. I fältet **Kvantitet** anger du antalet befattningar som du vill skapa
    * Startdatumet kommer att bli **anställningsdatumet** för de nya medarbetarna.  
    * Slutdatumet kommer att bli **uppsägningsdatumet** för de nya medarbetarna.  
    * Ange om de nya medarbetarna ska vara **anställda** eller **underleverantörer**.  
10. Klicka på knappen i listrutan i fältet **Jobb** för att välja det jobb som du vill skapa befattningarna för.
11. Hitta och markera önskad post i listan.
12. Klicka på länken på den valda raden i listan.
    * Heltidsvärdet ska hämtas från det jobb som har valts. Du kan ändra detta om det behövs.  
    * Du kan välja avdelning för de nya befattningarna.  
13. Klicka på **OK**.

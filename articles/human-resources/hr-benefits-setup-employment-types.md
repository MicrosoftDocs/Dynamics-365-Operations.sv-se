# <a name="set-up-employment-types"></a>Ställ in anställningstyper

[!include [banner](includes/preview-feature.md)]

Ställ in medarbetartyper i Dynamics 365 Human Resources för att stämma överens med dina arbetare.

1. I arbetsytan **Förmånshantering** under **inställningar**, välj **Anställningstyper**.

2. Välj **Ny**.

3. Ange värden för de följande fälten:

   | Fält | Beskrivning |
   | --- | --- |
   | Anställningstyp | Unik identifiering av anställningstypen. |
   | Beskrivning | En beskrivning av anställningstyp. |

4. Välj **Spara**.

# <a name="ScreenDestinationType">Skärmmål</a>

[!include [banner](../includes/banner.md)]

Du kan konfigurera en skärmdestination för varje mapp- eller fil-komponent i ett ER-format (elektronisk rapportering) som har konfigurerats för att generera utgående dokument. Baserat på destinationsinställningen kan ett genererat dokument öppnas för förhandsgranskning i en separat webbläsarflik.

Om du ställer in **Aktiverad** som **Ja** skapas en förhandsgranskning av resultatet. Du kan visa vissa filtyper, till exempel XML, TXT eller PDF, direkt i ett webbläsarfönster. För andra filtyper som exempelvis Microsoft Excel eller Word, används tjänsten Microsoft Office.

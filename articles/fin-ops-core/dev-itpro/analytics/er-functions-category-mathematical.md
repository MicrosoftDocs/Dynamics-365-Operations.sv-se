# <a name="list-of-er-functions-in-the-mathematical-category"></a>Lista över ER-funktioner i matematiska kategorin

[!include [banner](../includes/banner.md)]

De matematiska funktionerna för elektronisk rapportering (ER) kan användas för att göra många vanliga matematiska beräkningar. Det här avsnittet innehåller en sammanfattning av dessa funktioner.

## <a name="list-of-supported-functions"></a>Lista över funktioner som stöds

| Funktion | Beskrivning |
|----------|-------------|
| [Abs](er-functions-mathematical-abs.md)             | Den här funktionen returnerar absolutvärdet (modul) för det angivna talet som ett *verkligt* värde. Med andra ord returnera tal utan tecken. |
| [Effekt](er-functions-mathematical-power.md)         | Den här funktionen returnerar ett *verkligt* värde som representerar resultatet av att höja det angivna positiva talet till den angivna effekten. |
| [Avrundning](er-functions-mathematical-round.md)         | Denna funktion returnerar det specificerade numret som *verkligt* värde efter det avrundats till det angivna antalet decimaler. |
| [RoundDown](er-functions-mathematical-rounddown.md) | Denna funktion returnerar det specificerade numret som *verkligt* värde efter det avrundats ned till det angivna antalet decimaler. |
| [RoundUp](er-functions-mathematical-roundup.md)     | Denna funktion returnerar det specificerade numret som *verkligt* värde efter det avrundats upp till det angivna antalet decimaler. |

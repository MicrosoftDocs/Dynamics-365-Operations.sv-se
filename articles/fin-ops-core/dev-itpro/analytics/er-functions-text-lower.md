# <a name="lower-er-function"></a>LOWER ER-funktion

[!include [banner](../includes/banner.md)]

`LOWER`-funktionen returnerar den angivna textsträngen som ett *sträng* värde efter att den har konverterats till gemener.

## <a name="syntax"></a>Syntax

```vb
LOWER (text)
```

## <a name="arguments"></a>Argument

`text`: *Sträng*

Ett *sträng*-värde som anger texten.

## <a name="return-values"></a>Returvärden

*Sträng*

Det resulterande textvärdet.

## <a name="example"></a>Exempel

`LOWER ("Sample")` returnerar **"exempel"**.

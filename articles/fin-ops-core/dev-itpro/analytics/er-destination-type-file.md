# <a name="file-destination"></a>Filmål

[!include [banner](../includes/banner.md)]

Du kan konfigurera en fildestination för varje MAPP- eller FIL-komponent i ett ER-format (elektronisk rapportering) som har konfigurerats för att generera utgående dokument. Baserat på destinationsinställningen kan ett genererat dokument hämtas från webbläsaren.

På sidan **Destinationsinställningar** om du anger **Aktiverad** till **Ja** öppnas en dialogruta för Öppna eller Spara när konfigurationen är slutförd.

[![Sidan Destinationsinställningar.](./media/ER_Destinations-EnableFileDestination.png)](./media/ER_Destinations-EnableFileDestination.png)

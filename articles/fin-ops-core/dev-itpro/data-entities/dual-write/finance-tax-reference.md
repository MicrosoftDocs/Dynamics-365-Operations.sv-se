# <a name="access-to-finance-and-tax-reference-data"></a>Åtkomst till ekonomi- och momsreferensdata

[!include [banner](../../includes/banner.md)]



Alla företag arbetar med en grundläggande uppsättning ekonomiska data, t.ex. räkenskapskalenderår, valutan som företaget samarbetar i, de konton som pengarna ska köra verksamheten med i eller går ur, momssatser och remittering. Dessa data finns i appar för ekonomi och drift. Den är dock exponerad för Dataverse så att kundengagemangsappen kan ha en enda källa för ekonomi- och momsdata. På så sätt kan data vara enhetliga i hela affärsekosystemet.

Ekonomi- och momsdata integreras med hjälp av följande mappningar:

+ [Integrerad redovisning](ledger-mapping.md)
+ [Integrerat skattehuvud](tax-mapping.md)

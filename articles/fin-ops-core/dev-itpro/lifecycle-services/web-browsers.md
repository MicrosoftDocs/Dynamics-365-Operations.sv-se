# <a name="supported-web-browsers-for-lifecycle-services"></a>Få support för webbläsare för Lifecycle Services 

[!include[banner](../includes/banner.md)]

Användare kan komma åt Microsoft Dynamics Lifecyle Services (LCS) genom att använda de senaste versionerna av dessa populära webbläsare: 

- Microsoft Edge (rekommenderas: [krombaserad Edge](https://support.microsoft.com/microsoft-edge/download-the-new-microsoft-edge-based-on-chromium-0f4a3dd7-55df-60f5-739f-00010dba52cf))
- Google Chrome
- Apple Safari

Gå till programtillverkarens webbplats för att hitta den senaste versionen för respektive webbläsare. För optimal prestanda och optimal erfarenhet rekommenderar vi att du använder den senaste versionen av en modern webbläsare, speciellt Microsoft Edge.

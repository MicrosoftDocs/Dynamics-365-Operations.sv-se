# <a name="set-a-users-preferred-time-zone"></a>Ställa in en användares prioriterade tidszon

[!include [banner](../../includes/banner.md)]

Följande avsnitt beskriver hur en användare med rollen systemadministratör kan ange tidszon för en användare.

1. Gå till **Navigeringsfönster > Moduler > Systemadministration > Användare > Användare**.
2. Hitta och markera önskad post i listan.
3. Välj **Användaralternativ**.
4. Välj fliken **Inställningar**.
5. I fältet **Tidszon** väljer du ett alternativ iden nedrullningsbara listan.
6. Välj **Spara**.

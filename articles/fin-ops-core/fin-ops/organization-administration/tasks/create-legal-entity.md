# <a name="create-a-legal-entity"></a>Skapa en juridisk person

[!include [banner](../../includes/banner.md)]

En juridisk person är en organisation som identifieras genom registrering med en juridisk myndighet. Juridiska personer kan ingå kontrakt och behövs för att förbereda utdrag som rapporterar på deras resultat. Följande procedur förklarar hur du skapar en juridisk person. Det demonstrationsdataföretag som används för att skapa den här proceduren är USMF.

1. Gå till **Navigeringsfönster > Moduler > Organisationsadministration > Organizationer > Juridiska personer**.
2. Klicka på **Ny**.
3. Skriv ett värde i fältet **Namn**.
4. Skriv ett värde i fältet **Företag**.
5. Ange eller välj ett värde i fältet **Land/region**.
6. Klicka på **OK**. I avsnittet **Allmänt** ger du följande allmän information om den juridiska personen: Ange ett söknamn, om ett söknamn krävs. En söknamn är ett alternativt namn som kan användas för att söka efter den här juridisk person. Markera om den här juridisk person används som ett konsolideringsföretag. Markera om den här juridisk person används som ett elimineringsföretag. 
7. Expandera avsnittet **Adresser**. I avsnittet **Adresser**, klicka på **Redigera** för att ange adressinformation, till exempel gatunamn och nummer, postnummer och ort.
8. Expandera avsnittet **Kontaktinformation.** Ange information om metoder för kommunikationen till exempel e-postadresser, telefonnummer, webbadress, i avsnittet **Kontaktinformation**. 
9. Expandera avsnittet **Rapportering enligt lag**. Ange registreringsnumren som används för lagstadgad rapportering i avsnittet **Rapportering enligt lag**.
10. Visa avsnittet **Registreringsnummer**. I avsnittet **Registreringsnummer**, ange vilken information som krävs av den juridiska personen.  
11. Expandera avsnittet **Bankkontoinformation**. Ange bankkonton och organisationsnummer för juridiska personen i avsnittet **Bankkontoinformation**.
12. Visa avsnittet **Utländsk handel och logistik** Ange leveransinformation för juridiska personen i avsnittet **Utrikeshandel- och logistik**.  
13. Visa avsnittet **nummersekvenser**. I avsnittet **Nummerserie**, kan du visa de nummerserier som är kopplade till juridiska personen.  
14. Expandera avsnittet **Bilder**. I avsnittet **Bilder** kan du visa eller ändra logotyp- och/eller instrumentpanelbilden som associeras med juridiska personen.  
15. Expandera avsnittet **momsregistrering**. Ange registreringsnumren som används för att rapportera till skattemyndigheten i avsnittet **Momsregistrering**.
16. Expandera avsnittet **Moms 1099**. Ange 1099-information för juridiska personen i avsnittet **Skatt 1099**.  
17. Klicka på **Spara**.

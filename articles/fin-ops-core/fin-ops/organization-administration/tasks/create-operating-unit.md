[!include [banner](../../includes/banner.md)]

1. <span data-ttu-id="6486f-109">Gå till **Navigeringsfönster > Moduler > Organisationsadministration > Organisationer > Driftenheter**.</span><span class="sxs-lookup"><span data-stu-id="6486f-109">Go to **Navigation pane > Modules > Organization administration > Organizations > Operating units.**</span></span>

3. <span data-ttu-id="6486f-111">Hitta och markera önskad post i listan.</span><span class="sxs-lookup"><span data-stu-id="6486f-111">In the list, find and select the desired record.</span></span> <span data-ttu-id="6486f-112">Välj den typ av driftenhet som du vill skapa.</span><span class="sxs-lookup"><span data-stu-id="6486f-112">Select the type of operating unit you want to create.</span></span>  
4. <span data-ttu-id="6486f-113">Klicka på länken på den valda raden i listan.</span><span class="sxs-lookup"><span data-stu-id="6486f-113">In the list, click the link in the selected row.</span></span>

    + <span data-ttu-id="6486f-115">Expandera avsnittet **Allmänt**, om det behövs.</span><span class="sxs-lookup"><span data-stu-id="6486f-115">Expand the **General** section, if necessary.</span></span>

    + <span data-ttu-id="6486f-117">Expandera avsnittet **Adresser**, om det behövs.</span><span class="sxs-lookup"><span data-stu-id="6486f-117">Expand the **Addresses** section, if necessary.</span></span>

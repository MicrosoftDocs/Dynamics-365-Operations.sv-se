[!include [banner](../../includes/banner.md)]

<span data-ttu-id="3a246-104">Följande procedur visar de steg som systemadministratörer kan använda för att uppdatera webbannons- eller logotypbilden som visas för en juridisk person.</span><span class="sxs-lookup"><span data-stu-id="3a246-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="3a246-105">Det demonstrationsdataföretag som används för att skapa den här proceduren är USMF.</span><span class="sxs-lookup"><span data-stu-id="3a246-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="3a246-106">Gå till **Navigeringsfönster > Moduler > Organisationsadministration > Organizationer > Juridiska personer**.</span><span class="sxs-lookup"><span data-stu-id="3a246-106">Go to **Navigation pane > Modules > Organization administration > Organizations > Legal entities**.</span></span>
2. <span data-ttu-id="3a246-107">I listan till vänster, välj den juridiska person du vill uppdatera webbannonsen eller logotypen för.</span><span class="sxs-lookup"><span data-stu-id="3a246-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="3a246-108">Om den redan har markerats, gå till nästa steg.</span><span class="sxs-lookup"><span data-stu-id="3a246-108">If it is already selected, go to the next step.</span></span>

4. <span data-ttu-id="3a246-110">Välj **Ändra**.</span><span class="sxs-lookup"><span data-stu-id="3a246-110">Select **Change**.</span></span>
    
    <span data-ttu-id="3a246-111">Den idealiska lösningen för en webbannonsbild är 1 920 × 281 pixlar.</span><span class="sxs-lookup"><span data-stu-id="3a246-111">The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="3a246-112">Den idealiska bredden för en logotypbild är 350 pixlar.</span><span class="sxs-lookup"><span data-stu-id="3a246-112">The ideal width for a logo image is 350 pixels.</span></span>
    
5. <span data-ttu-id="3a246-113">Välj **Spara**.</span><span class="sxs-lookup"><span data-stu-id="3a246-113">Select **Save**.</span></span>
6. <span data-ttu-id="3a246-114">Gå till **Navigeringsfönster > Moduler > Vanliga > Vanliga > Standardinstrumentpanel**.</span><span class="sxs-lookup"><span data-stu-id="3a246-114">Go to **Navigation pane > Modules > Common > Common > Default dashboard**.</span></span> <span data-ttu-id="3a246-115">Du bör se den nya webbannonsen eller logotypbilden på instrumentpanelen.</span><span class="sxs-lookup"><span data-stu-id="3a246-115">You should see the new banner or logo image on the dashboard.</span></span>  
    
    <span data-ttu-id="3a246-116">Du kan behöva ändra ditt företag, med hjälp av företagsväljaren, till det som du överför webbannonsen för.</span><span class="sxs-lookup"><span data-stu-id="3a246-116">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>

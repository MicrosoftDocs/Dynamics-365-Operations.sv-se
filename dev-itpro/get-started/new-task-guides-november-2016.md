# <a name="new-or-updated-task-guides-november-2016"></a>Ny eller uppdaterad uppgift får (2016 November)

Det här avsnittet beskrivs de nya eller uppdaterade uppgiften riktlinjer som publicerades för Microsoft (November 2016 - företagsprofiler) APQC enhetliga bibliotek för Microsoft Dynamics 365 för operationer. Inge nytt Komma igång-biblioteket gavs ut.

Den (November 2016--företagsprofiler) APQC Unified biblioteket innehåller guiderna för uppgiften som ska användas om du behöver hjälp med Microsoft Dynamics 365 för operationer, Microsoft Dynamics 365 för operationer plattformsuppdateringen Microsoft Dynamics AX-plattformen uppdatering 2, 3 Dynamics AX maj 2016 uppdateringen och Microsoft Dynamics AX februari 2016. Mer information om dessa versioner finns [vad är nya eller ändrade](whats-new-changed.md). Lokaliserade versioner av uppgiften finns i biblioteket. Mer information om hur du ansluter hjälpsystemet till ett bibliotek av aktivitet guide finns [Kom igång med](help-overview.md). **Anmärkning:** asterisker (**\***) bredvid rubriken för en process Affärsmodelleraren (BPM) raden anger handbok för den affärsprocessen uppgiften skapades för Dynamics 365 för November 2016 uppdatera och stöds inte i tidigare versioner av Microsoft Dynamics AX.

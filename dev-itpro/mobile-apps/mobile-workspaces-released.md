# <a name="mobile-workspaces"></a>Mobila arbetsytor

[!include[banner](../includes/banner.md)]

Detta avsnitt visar de mobila arbetsytor som är tillgängliga för användning med mobilappen Dynamics 365 for Unified Operations.

> [!NOTE]
> Mobilappen kallades tidigare *Microsoft Dynamics 365 for Finance and Operations*.

| Mobila arbetsytor     | beskrivning   | Tillgänglighet   |
|----------------------|---------------|--------------|
|[Företagskatalog](company-directory-mobile-workspace.md)| Låter dig visa och kontakta andra medarbetare i din organisation.| 2017 juni |    
|[Mitt team](manager-self-service-mobile-workspace.md)| Du kan visa dina underställda och utökade personal, samt även skicka beröm som berör personer i din rapporteringskedja.|2017 juni |     
|[Godkännande av faktura](invoice-approval-mobile-workspace.md)| Innehåller en lista med fakturor som har tilldelats till dig via arbetsflödet för leverantörsfakturahuvud.| 2017 juni   |
| [Utläggshantering](/dynamics365/unified-operations/financials/expense-management/expense-management-mobile-workspace) | Du kan hämta och överföra ett kvitto så att du kan bifoga dessa till en senare utgiftsrapport. Den mobila arbetsytan låter även dig snabbt skapa en utgiftsrad med en kopplad inleverans. | 2017 april |
| [Godkännande av inköpsorder](/dynamics365/unified-operations/supply-chain/procurement/purchase-order-mobile-workspace) | Visa och svara på inköpsorder med åtgärder såsom "godkänn" eller "avvisa". | 2017 april |
| [Post för projekttid](/dynamics365/unified-operations/financials/project-management/project-time-entry-mobile-workspace) | Du kan ange och spara tid för ett projekt genom att använda din mobila enhet. | 2017 mars |
| [Kostnadskontroll](/dynamics365/unified-operations/financials/cost-accounting/cost-controlling-mobile-workspace)     | Chefer för kostnadsställen kan se kostnadsställets effektivitet.                                                                                               |  2017 januari        |
| [Lagerbehållning ](/dynamics365/unified-operations/supply-chain/inventory/inventory-on-hand-mobile-workspace)    | Få insyn i reserverades och tillgängliga lager.                                                                                                    |   2017 januari       |
| [Försäljningsorder](/dynamics365/unified-operations/supply-chain/sales-marketing/sales-orders-mobile-workspace)         | Håll dig uppdaterad om dina försäljningsorder.                                                                                                                          |  2017 januari                  |
| [Leverantörssamarbete](/dynamics365/unified-operations/supply-chain/procurement/vendor-collaboration-mobile-workspace) | Leverantörer kan hållas uppdaterade om de inköpsorder som har skickats till dem för godkännande. De kan också visa information om nya och uppdaterade inköpsorder och kontakter. |2017 januari    |

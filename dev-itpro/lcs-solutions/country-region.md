# <a name="localization-and-regulatory-features"></a>Lokalisering och andra funktioner

Microsoft Dynamics-356 för operationer innehåller funktioner för vissa bestämmelser om lands-/ regionspecifika och processer. Den här funktionen aktiveras utifrån den primära adressen för den juridiska personen som aktiv. Det här avsnittet innehåller listor över resurser som kan hjälpa dig att lära dig mer om lands-/ regionspecifika funktioner.

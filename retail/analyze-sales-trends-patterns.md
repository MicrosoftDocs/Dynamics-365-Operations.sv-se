# <a name="analyze-sales-trends-and-patterns"></a>Analysera försäljningstrender och försäljningsmönster

Du kan undersöka försäljningstrender och mönster i realtid i Microsoft Dynamics 365 för verksamhet – Retail.

Som en del av Dynamics 365 för operationer, användarna kan undersöka försäljningstrender och mönster i realtid över olika nivåer för organisationshierarkin under ett antal år med out-of-box **kanaler försäljning per år** rapporten. Du kan öppna den här rapporten från någon av följande platser:

-   **Retail store management** Dynamics 365 arbetsytan för operationer &gt;**butik och handel**&gt;**kanaler**&gt;**Retail store management**&gt;**rapporter**&gt;**kanaler försäljning efter år**
-   **Retail store ekonomi** Dynamics 365 arbetsytan för operationer &gt;**butik och handel**&gt;**kanaler**&gt;**Retail store ekonomi**&gt;**rapporter**&gt;**kanaler försäljning efter år**
-   **Förfrågningar och rapporter** Dynamics 365 för avsnittet &gt;**butik och handel**&gt;**förfrågningar och rapporter**&gt;**försäljningsrapporter**&gt;**kanaler försäljning efter år**

Användare kan även studera försäljningstrender och mönster på timnivå i organisationshierarkin för en vald period genom att öppna den inbyggda rapporten **Kanalförsäljning per timme**. Du kan öppna den här rapporten från någon av följande platser:

-   **Retail store management** Dynamics 365 arbetsytan för operationer &gt;**butik och handel**&gt;**kanaler**&gt;**Retail store management**&gt;**rapporter**&gt;**kanaler försäljning per timme rapport**
-   **Retail store ekonomi** Dynamics 365 arbetsytan för operationer &gt;**butik och handel**&gt;**kanaler**&gt;**Retail store ekonomi**&gt;**rapporter**&gt;**kanaler försäljning per timme rapport**
-   **Förfrågningar och rapporter** Dynamics 365 för avsnittet &gt;**butik och handel**&gt;**förfrågningar och rapporter**&gt;**försäljningsrapporter**&gt;**kanaler försäljning per timme rapport**

# <a name="commerce-essentials-overview"></a>Commerce essentials: översikt

Den här artikeln innehåller information om Commerce essentials, vilket är en retail försäljning (PO) och konfigurationsalternativ för e-handel för Microsoft Dynamics 365 för operationer. Grundläggande information om handel innehåller en inbyggd återförsäljararbetsbelastning som kan hjälpa återförsäljare att snabbt komma igång med en Microsoft Dynamics-baserad butikslösning och som ger dem möjlighet att fokusera på butiksspecifika funktioner. 

<a name="overview"></a>Översikt
--------

Affärsprogrammiljöer innehåller ofta många system som presterar mycket specifika funktioner. Ofta återförsäljare vill utnyttja retail management aspekter av Microsoft Dynamics 365 för operationer och vill inte ersätta de befintliga ekonomiska eller personal (HR) system. För dessa återförsäljare, en arbetsbelastning som är specifik för butik krävs. Grundläggande om handel ger dessa butiksspecifika funktioner som ett konfigurationsalternativ för Dynamics 365 för operationer. När konfigurationen är slutförd, alla icke-butiksaspekter av lösningen är dolda, så att fokus kan läggas på det som är viktigt: att hantera en butiksorganisation. Grundläggande om handel introducerades som ett konfigurationsalternativ i Microsoft Dynamics AX 2012 R3 CU8. Sedan dess har många förbättringar har gjorts för att göra den till en lösning som fungerar för omni-återförsäljare, inte bara för riktiga organisationer.

## <a name="configuration"></a>Inställningar
Konfigurera grundläggande om handel genom att gå till **systemadministration**&gt;**inställningar**&gt;**licenskonfigurationen**, och rulla ned till **Retail**. Expandera **noden för butik** , och markera kryssrutan för **Grundläggande om handel**. Klicka på **Spara**, granska ändringarna och sedan uppdatera sidan. Om du vill växla tillbaka till fullständiga lösningen går du till **essentials Commerce**&gt;**Headquarters installation**&gt;**licenskonfigurationen**, och rulla ned till **Retail**. Expandera **noden för butik** och avmarkera kryssrutan för **Grundläggande om handel**. Klicka på **Spara**, granska ändringarna och sedan uppdatera sidan.

## <a name="what-is-included"></a>Vad ingår?
Grundläggande om handel innehåller alla retail back office-funktioner som ingår i grundläggande Dynamics 365 för operationer lösningen, utom call center funktioner. Alla kanaler, arbetstagare, produkt, lagra, registrera, och enhetshanteringsfunktioner ingår i en enda navigationsnod. Dessutom, alla fullständiga butiksarbetsytor ingår för att ge rollspecifika landningssidor. Grundläggande om handel riktar sig till återförsäljare som har ett bankkontor och andra eller rörliga scenario som kan dra nytta av Retail POS (MOPS Modern) och e-handel kanaler. Genom att använda Grundläggande om handel, återförsäljare kan hantera dessa kanaler i en enda plats och därmed få riktig omni-kanal kapaciteter.

## <a name="how-is-commerce-essentials-different"></a>Hur är Grundläggande om handel annorlunda?
Förutom att tillhandahålla enhetlig navigering och butikssidor bara, Grundläggande om handel döljer alla icke-butiks element, såsom fält och kontroller. Resultatet är ett strömlinjeformat användargränssnitt som hjälper detaljister fokusera på arbetsuppgiften i handen. Återförsäljare som har en fullständig produktlicens kan använda grundläggande om handel som utgångspunkt för flera användare som retail. Andra element, till exempel avancerade funktioner för personal kan läggas till grundläggande om handel som krävs enligt villkoren i licensavtalet är på plats. Som en del av rationalisering och förenkling som Grundläggande om handel tillhandahåller, bokföringsexportkapacitet låter återförsäljare mappa summor som uppstått i Grundläggande om handel på kontonummer i deras befintliga redovisningssystem. När butikssystemet genererar ekonomiska data, data kan enkelt exporteras från Grundläggande om handel i detaljhandlarens redovisningssystem. Om det krävs mer avancerade funktioner för integration, grundläggande om handel även fullständig Import/Export-ramverket som ingår i Dynamics 365 för operationer. Hundratals data entiteter som ingår som standard ger Grundläggande om handel den flexibilitet som passar bra in i en multi-nivå miljö där flera specialiserade system används.
